# 撓度積分常數邊界條件（DEFLECTION-BOUNDARY）

★★ 中頻陷阱

## 陷阱描述
雙積分法求梁撓度時，積分常數 C1 和 C2 需由邊界條件決定，常因邊界條件選錯而導致整題錯誤。

## 邊界條件彙整
| 支承類型 | 條件 |
|---------|------|
| 簡支端（pin/roller） | v = 0（撓度 = 0） |
| 固定端（fixed） | v = 0 且 v' = 0（撓度 = 0，轉角 = 0） |
| 自由端（free） | M = 0 且 V = 0（彎矩 = 0，剪力 = 0） |
| 內部鉸接 | v 連續，M = 0（彎矩不連續允許） |

## 常見錯誤
1. 固定端只用 v = 0 忘記 v' = 0 → 少一個方程式
2. 對分段梁不用連續條件（v 和 v' 在接合點需連續）
3. 單位不一致（E 用 GPa 但 I 用 cm⁴）

## 出現題目
MM-U3-2 梁撓度題（積分法）
