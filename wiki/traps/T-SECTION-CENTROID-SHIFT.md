# T 形截面形心偏移（T-SECTION-CENTROID-SHIFT）

★★★ 高頻陷阱

## 陷阱描述
直覺認為 T 形截面的形心在幾何中間 → **錯誤**

## 正確觀念
T 形截面（翼板 + 腹板）的形心偏向翼板側（面積較大的一側）：
- ȳ = (A_f · y_f + A_w · y_w) / (A_f + A_w)（從底端量起）
- 形心偏上 → c_bot > c_top（底纖維距形心更遠）
- 最大拉應力 ≠ 最大壓應力（不對稱）

## 計算步驟
1. 分解為翼板矩形 + 腹板矩形
2. 各部分面積 A 和形心位置 y（從統一基準量起）
3. ȳ = ΣAy / ΣA
4. c_bot = ȳ；c_top = h - ȳ（h = 截面全高）

## 出現題目
幾乎所有 T 形截面題（MM-U1-1 形心計算）
