# 不靜定桿件相容條件符號（INDETERMINATE-COMPATIBILITY）

★★ 中頻陷阱

## 陷阱描述
不靜定桿件（軸力或梁）建立相容方程式時，各段變形量的符號（伸長正/縮短負）需一致，否則解出的多餘力符號相反。

## 正確建立方式
1. 定義正向：通常以伸長為正、縮短為負（或明確標示）
2. 寫出每段 δᵢ = PᵢLᵢ/(AᵢEᵢ)（Pᵢ 受拉為正）
3. 相容條件：Σδᵢ = 已知總位移（通常為 0 或給定值）
4. 代入解多餘力

## 常見錯誤
- 忘記熱膨脹量 δ_T = αΔTL 也要計入
- 在解含熱膨脹的不靜定時，ΔT > 0 使桿件膨脹，受壓縮拘束時產生壓應力（負值）

## 出現題目
MM-U3-1 不靜定軸力題、含溫度載重題
