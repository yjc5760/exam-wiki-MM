# 廣義虎克定律泊松效應（GENERALIZED-HOOKE-POISSON）

★★ 中頻陷阱

## 陷阱描述
多軸應力狀態下，各方向應變受其他方向應力的泊松效應影響，不能只用單軸虎克定律 ε = σ/E。

## 正確廣義虎克定律（等向性材料）
εₓ = (σₓ - ν(σᵧ + σ_z)) / E
εᵧ = (σᵧ - ν(σₓ + σ_z)) / E
ε_z = (σ_z - ν(σₓ + σᵧ)) / E

剪應變：γ_xy = τ_xy / G（G = E/(2(1+ν))）

## 常見錯誤
平面應力（σ_z = 0）時，z 方向仍有應變：ε_z = -ν(σₓ + σᵧ)/E ≠ 0
（這是平面應力而非平面應變，ε_z 不為零）

MM-U1-2 廣義虎克定律題、MM-U1-3 應力分析題
