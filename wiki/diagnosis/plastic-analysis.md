# 塑性分析診斷（MM-U4-1）

## 決策樹

求塑性矩 Mp？
  → Mp = σᵧ · Z（Z = 塑性斷面模數）
  → 矩形：Z = bh²/4；形狀因子 f = Z/S = 1.5

求極限載重？
  → 機構法：設定破壞機構 → 虛功原理：P_u·δ = ΣMp·θ
  → 確認破壞機構為最低極限載重的機構

求降伏載重？
  → 最外緣達 σᵧ 時的載重：M_y = σᵧ·I/c = σᵧ·S

## 常見陷阱

1. **Z vs S 混淆**：Z（塑性）= bh²/4；S（彈性）= bh²/6；Z > S 永遠成立
2. **塑性鉸的位置**：固定端、集中力作用點、彎矩最大處（靜定梁）；不靜定梁需確認
3. **虛功方向**：外力虛功（正）= 內力虛功（正）；注意 θ 的方向與 Mp 的方向是否一致
