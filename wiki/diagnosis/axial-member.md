# 軸力桿件診斷（MM-U2-1 + MM-U3-1）

## 決策樹

靜定桿件？
  → σ = P/A；δ = PL/(AE)
  → 分段加載：各段分開計算，取代數和

不靜定桿件？
  → 靜力方程 + 相容條件（δ 值限制）
  → 相容條件：各段的伸長量相加為指定值（通常為0或已知位移）

溫度變化？
  → δ_T = αΔTL（自由熱變形）
  → 受拘束：σ_T = -EαΔT（全約束）；部分約束需聯立

## 常見陷阱

1. **拉正壓負**：軸力符號規定要一致
2. **不靜定次數**：支承反力數 - 平衡方程數 = 不靜定次數
3. **熱膨脹方向**：ΔT > 0 → 桿件伸長 → 若受壓縮拘束 → 產生壓應力
