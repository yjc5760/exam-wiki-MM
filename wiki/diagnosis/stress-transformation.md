# 應力轉換與 Mohr's Circle 診斷（MM-U1-3）

## 決策樹

已知 σₓ, σᵧ, τₓᵧ，求主應力？
  → σ₁,₂ = C ± R，C = (σₓ+σᵧ)/2，R = √[(Δσ/2)² + τ²]

求最大剪應力？
  → τ_max = R（平面內最大剪應力）
  → 絕對最大剪應力考慮三維（σ₃=0 時）

求特定角度截面應力？
  → σ_θ = C + R·cos2α，τ_θ = R·sin2α（用 Mohr's circle 轉換）

已知主應力，求原始應力狀態？
  → 反向推導：σₓ + σᵧ = σ₁ + σ₂（不變量）

## 常見陷阱

1. **τ 方向**：Mohr's circle 上 τ 正方向向下（與一般座標相反）
2. **θ vs 2θ**：物理上轉 θ，在 Mohr's circle 上轉 2θ
3. **絕對最大剪應力**：平面應力時，若 σ₁、σ₂ 同號，絕對最大剪應力 = max(σ₁, σ₂)/2（不是 R）
