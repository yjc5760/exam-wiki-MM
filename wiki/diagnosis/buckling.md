# 柱挫屈診斷（MM-U3-4）

## 決策樹

求臨界載重？
  → Pₑ = π²EI/(KL)²
  → 確認 K（端部條件）：兩端鉸=1.0；兩端固定=0.5；一固一鉸=0.7；懸臂=2.0
  → I 取最小值（最弱軸先挫屈）

求臨界應力？
  → σₑ = Pₑ/A = π²E/(KL/r)²（r = √(I/A)）
  → 需確認是否超過降伏應力（若 σₑ > σᵧ → 短柱，用降伏控制）

多柱同時承載？
  → 各柱臨界載重可能不同，系統失穩由最弱柱決定

## 常見陷阱

1. **K 值選擇錯誤**：邊界條件是「理想」的，實際工程常修正 K
2. **I 要取最小值**：對不對稱截面，挫屈沿 I_min 軸發生
3. **Euler 公式的適用範圍**：細長比需夠大（σₑ < σᵧ），否則 Euler 公式高估
4. **有效長度 KL 的物理意義**：等效於兩端鉸接柱的長度
