# 梁桿件斷面應力診斷（MM-U2-2）

## 決策樹

問的是正向應力？
  → 計算 σ = My/I
    → 先求形心、慣性矩（平行軸定理）
    → 確認 y（距中性軸）；最外緣用 c
    → T 形截面：c_bot ≠ c_top，分別計算

問的是剪應力？
  → 使用 Jourawski 公式：τ = VQ/(Ib)
    → Q = 截面一部分的面積一次矩（剪力截面外側）
    → 矩形截面：最大在形心軸（τ_max = 1.5V/A）
    → T 形截面：最大在腹板形心軸附近

問的是最大值？
  → σ_max：|M|_max 截面，距中性軸最遠的點
  → τ_max：|V|_max 截面，Q/b 最大的點（通常是形心軸）

## 常見陷阱

1. **T 形截面形心偏移**：形心不在幾何中心，c_bot ≠ c_top，最大拉應力與最大壓應力不等
2. **最大剪應力位置**：不在最外緣（那裡 Q=0），而在形心軸（Q 最大且 b 最小）
3. **彎矩剪力圖要先畫**：找 M_max 和 V_max 的位置才能知道在哪個截面算應力
4. **正負號規定**：正彎矩使下方受拉，需確認 y 的方向定義是否一致
