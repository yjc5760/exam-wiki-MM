# 梁桿件變位診斷（MM-U3-2）

## 決策樹

靜定梁，標準載重形式？
  → 疊加法（查表）：δ_max = PL³/(48EI)（集中，簡支中點）等

靜定梁，非標準或需找斜率？
  → 積分法：EI·v'' = M(x)，積分兩次，代入邊界條件

靜不定梁？
  → 選擇多餘反力，釋放成靜定梁
  → 用撓度相容條件（δ_released = 0）求多餘反力
  → 回代求所有反力和彎矩

需要任意點撓度？
  → Mohr's 積分法（共軛梁法）：使用虛功原理

## 常見陷阱

1. **積分常數**：雙積分法需要兩個邊界條件（固定端 v=v'=0；簡支端 v=0）
2. **EI 單位**：若 E 用 GPa，I 用 mm⁴，δ 會是 kN·mm³/GPa → 需換算
3. **靜不定次數**：n 次靜不定需 n 個相容方程式
4. **疊加順序**：先算各載重分別的撓度，再相加（線性疊加只在彈性範圍有效）
