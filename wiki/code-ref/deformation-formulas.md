# 變位公式彙整

## 軸向變形
δ = PL/(AE)（均勻截面）
熱變形：δ_T = αΔTL

## 梁撓度（標準案例）
簡支梁，集中力 P 在中點：δ_max = PL³/(48EI)
簡支梁，均布載重 w：δ_max = 5wL⁴/(384EI)
懸臂梁，端點集中力 P：δ_max = PL³/(3EI)（端點）；θ = PL²/(2EI)
懸臂梁，均布載重 w：δ_max = wL⁴/(8EI)

## 扭轉角
φ = TL/(GJ)（均勻截面圓桿）
φ = ∫T dx/(GJ)（變截面或變扭矩）
薄壁封閉：φ = TL/(4GA_m²)·∮(ds/t)

## 雙積分法
EI·v'' = M(x)
邊界條件：
  簡支端：v=0
  固定端：v=0, v'=0
  內部鉸接：v 連續，M=0
