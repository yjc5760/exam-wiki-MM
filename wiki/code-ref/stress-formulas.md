# 應力公式彙整

## 正向應力
軸力：σ = P/A
彎曲：σ = My/I（y 從中性軸量起）

## 剪應力
梁剪力（Jourawski）：τ = VQ/(Ib)
  Q = ∫y dA（截面一部分的面積一次矩）
圓截面最大剪應力：τ_max = 4V/(3πr²) = 4V/(3A)
矩形截面最大剪應力：τ_max = 3V/(2bh) = 1.5 V/A

## 扭轉剪應力
圓截面：τ = Tr/J（r 從圓心量起）；τ_max = Tc/J
薄壁封閉截面（剪力流）：q = T/(2Aₘ)；τ = q/t

## Mohr's Circle（應力轉換）
σ₁,₂ = C ± R；C = (σₓ+σᵧ)/2；R = √[(Δσ/2)² + τ²]
τ_max = R；主應力面角度：tan(2θ_p) = -τ_xy/((σx-σy)/2)
