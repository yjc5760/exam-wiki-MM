# 塑性分析公式彙整

## 塑性矩 Mp
Mp = σᵧ · Z（Z = 塑性斷面模數）

常用截面塑性斷面模數 Z：
| 截面 | Z | 形狀因子 f = Z/S |
|------|---|------------------|
| 矩形 b×h | bh²/4 | 1.5 |
| 圓形 d | d³/6 | 16/(3π) ≈ 1.70 |
| 空心圓 | π(d_o³-d_i³)/6 | 視比例而定 |

## 降伏載重 Py（梁）
M_y = σᵧ · S = σᵧ · I/c；Py 由 M_y 反算

## 極限載重（機構法）
虛功原理：P_u·δ = ΣMpᵢ·θᵢ
破壞機構條件：形成足夠塑性鉸使結構成機構（靜不定次數 + 1 個塑性鉸）

## 殘留應力
σ_res = σ_loaded - σ_springback
σ_springback = M_applied · y / I（彈性）
