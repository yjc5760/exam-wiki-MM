<a id="1-moduleid"></a>
## 1　題目編號（moduleId）

```
MM-YYYY-N
```

| 欄位 | 說明 | 規則 |
|------|------|------|
| `MM` | 科目代碼（Mechanics of Materials） | 固定，大寫 |
| `YYYY` | 西元年 | 4 位數，如 `2016` |
| `N` | 該年第幾題 | 阿拉伯數字，無前導零，如 `1`、`2`、`3`、`4` |

**範例：**

| moduleId | 說明 |
|----------|------|
| `MM-2016-1` | 2016 年第 1 題 |
| `MM-2020-3` | 2020 年第 3 題 |
| `MM-2025-4` | 2025 年第 4 題 |

> 年份用西元（不用民國）。題號從 `1` 起算，無前導零（不可寫成 `MM-2016-01`）。

## 2　考卷 PDF 命名（`raw/exams/`）

```
MM-YYYY_材料力學.pdf
```

| 欄位 | 說明 |
|------|------|
| `MM-YYYY` | 科目代碼 + 西元年，底線 `_` 隔開後半 |
| `材料力學` | 固定字串，科目全名 |
| `.pdf` | 副檔名，小寫 |

**範例：**
```
MM-2016_材料力學.pdf
MM-2024_材料力學.pdf
命題大綱.pdf              ← 命題大綱（不含年份）
```

> 檔名開頭必須是 `MM-YYYY`，Cowork 的 SOLVE 指令依此定位考卷。

<a id="4-solution-md"></a>
## 4　解析 .md 主檔格式

### 4.1 開頭標籤區塊（每份解析必須包含）

```markdown
# 考題編號：MM-YYYY-N

**主分類：** `MM-UN-n` 分類名稱
**副分類：** `MM-UN-n` 分類名稱（無副分類則省略）
**分析法：** 彈性分析 / 塑性分析 / 概念題 / 混合
**標籤：** `標籤1` `標籤2` `標籤3` ...
```

### 4.2 數學公式（強制 LaTeX）

- 行內：`` $\sigma = M y / I$ ``
- 獨立：`$$\tau_{max} = \frac{VQ}{Ib}$$`
- **禁止純文字公式**（不可寫 σ = My/I 而不加 $ 符號）

### 4.3 圖片引用規範（雙重保險原則）

每張圖片在 .md 中必須包含 **alt text + 圖說** 兩部分：

```markdown
![精確描述圖片工程內容的 alt text](MM-YYYY-N-fig-1.png)

*圖說：關鍵數值、條件、結論的完整文字化說明。*
```

| 圖片類型 | Alt Text 要求 | 圖說要求 |
|---------|-------------|---------|
| `fig`（題目附圖） | 結構型式、截面尺寸、載重位置、材料說明 | 關鍵幾何數值、支撐條件、E/G 值 |
| `chart`（設計圖表） | 圖表類型、座標軸範圍 | Mohr's circle 控制點座標、應力轉換結論 |
| `eqn`（參考公式） | 公式類型與數量 | **所有公式完整以 LaTeX 文字化**（最重要） |
| `hand`（手寫補充） | 方法名稱與推導目標 | 步驟摘要、關鍵中間結果、最終公式 |
| `section`（斷面圖） | 斷面型式、主要尺寸、形心位置 | 寬、高、面積、慣性矩等數值 |

<a id="4-4-vha"></a>
### 4.4 變數層次分析（Variable Hierarchy Analysis）

> **目的：** 幫助考生在複習時精確定位「卡在哪裡」，做到針對性補強而非重新看整題。

每道題的解析 .md 中，在 `## 3. 解題戰略地圖` 之後、`## 4. 步驟化詳細計算` 之前，**必須加入一個 `## 3.5 變數層次分析` 區塊**。

#### 格式模板

```markdown
## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`[最終求解目標，如：計算最大正向應力 σ_max 與最大剪應力 τ_max]`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: } \bar{y} = \frac{\sum A_i y_i}{\sum A_i}$$

$$\text{Step 2: } I_z = \sum\!\left(\frac{b_i h_i^3}{12} + A_i d_i^2\right)$$

$$\text{Step 3: } \sigma_{max} = \frac{M \cdot c_{max}}{\boxed{I_z}}$$

`[依題型替換為實際公式鏈，保留 \boxed{} 標記推導變數]`

### L1：題目直接給定
_看到題目就能讀出的數字，不需要任何公式。_

| 符號 | 數值 | 說明 |
|------|------|------|
| $b$ | 100 mm | 矩形截面寬度 |
| $h$ | 200 mm | 矩形截面高度 |
| ... | ... | ... |

### L2：需知識點推導
_需要知道公式名稱與適用條件，套入 L1 即可算出。_

**Step 1：[步驟名稱]**

| 符號 | 公式/來源 | 卡關? |
|------|----------|:-----:|
| $A$ | $b \times h$ | |
| $\bar{y}$ | $\sum A_i y_i / \sum A_i$（平行軸定理） | |

**Step 2：[步驟名稱]**

| 符號 | 公式/來源 | 卡關? |
|------|----------|:-----:|
| $I_z$ | $bh^3/12$（矩形）或平行軸定理 | |
| $\sigma_{max}$ | $Mc/I_z$ | |

### L3：深層知識（不懂就卡住）
_L2 中某些公式本身需要背景概念才能正確應用的知識點。_

| 知識點 | 說明 | 卡關? |
|--------|------|:-----:|
| 最大剪應力位置 | 為何 τ_max 在形心軸而非最外緣？ | |
| T 形截面形心偏移 | 翼板大面積使形心偏上，c_bot > c_top | |
```

#### 層次定義

| 層次 | 定義 | 判斷標準 |
|------|------|---------|
| **L1** | 題目直接給的數字 | 不看任何公式就能讀出 |
| **L2** | 需要套公式推導 | 知道公式名稱 + 套 L1 即可算 |
| **L3** | 深層概念（控制 L2 的應用） | 不懂這個概念就會用錯 L2 公式 |

## 5　PNG 靜態截圖規範

### 格式

```
MM-YYYY-N-[類型碼]-[序號].png
```

### 類型碼對照表

| 類型碼 | 內容 | 誰負責 | 範例 |
|--------|------|:------:|------|
| `fig` | 題目附圖（結構示意、幾何圖、截面圖） | 使用者 | `MM-2016-1-fig-1.png` |
| `chart` | 設計圖表（Mohr's circle、應力分布圖） | 使用者 | `MM-2016-1-chart-1.png` |
| `eqn` | 題目提供的參考公式截圖 | 使用者 | `MM-2016-1-eqn-1.png` |
| `hand` | 手寫補充推導 | 使用者 | `MM-2016-1-hand-1.png` |
| `section` | 斷面示意圖（幾何與形心） | 使用者 | `MM-2016-1-section-1.png` |

### 命名規則

| 規則 | 說明 |
|------|------|
| 序號從 `1` 起 | 單張也要寫 `-1`，不可省略 |
| 全部小寫 | 類型碼和副檔名均小寫 |
| 連字號 `-` 分隔 | 不用底線 `_` |

<a id="8-tags"></a>
## 8　標籤分類系統

每道題目有四個分類維度，全部記錄在 `raw/json/question_index.json`：

| 欄位 | 說明 | 範例 |
|------|------|------|
| `primaryTopicId` | 命題大綱主分類（唯一） | `"MM-U2-2"` |
| `primaryTopicName` | 主分類名稱（直接引用命題大綱子項名稱） | `"梁桿件斷面應力計算"` |
| `secondaryTopicIds` | 命題大綱副分類（跨子項時用） | `["MM-U1-1"]` |
| `analysisMethod` | 分析法 | `"彈性分析"` / `"塑性分析"` / `"概念題"` / `"混合"` |
| `tags` | 自由標籤（核心考點，3–8 個） | `["T形截面","形心","慣性矩","最大正向應力"]` |

### 命題大綱分類對照

> **最新鮮的官方考點分類，請直接查閱：`raw/json/syllabus_taxonomy.json` 中 `id: "MM"` 的段落。**
> 所有 `primaryTopicId` 與主分類名稱，一律以該檔案為唯一準則。
> topicId 格式：`MM-UN-n`（U=單元號，n=子項號）

### 標準標籤詞彙

| 類別 | 標準標籤 |
|------|---------|
| **分析法** | 彈性分析、塑性分析、概念題 |
| **斷面性質** | 形心、慣性矩、極慣性矩、斷面模數、平行軸定理、組合截面、T形截面、I形截面、空心圓截面 |
| **虎克定律** | 正向應力、正向應變、剪應力、剪應變、泊松比、彈性模數、剪力模數、體積模數、廣義虎克定律 |
| **應力應變分析** | 主應力、主應變、最大剪應力、Mohr's circle、應力轉換、應變轉換、雙軸應力、平面應力 |
| **軸力桿件** | 軸向應力、軸向變形、不靜定桿件、熱應力、材料不均質、變截面桿件 |
| **梁斷面應力** | 彎曲正向應力、Jourawski剪應力公式、最大剪應力位置、T形截面、矩形截面、複合梁 |
| **扭力桿件** | 扭轉剪應力、扭轉角、圓形截面、空心圓截面、剪力流、薄壁截面 |
| **梁變位** | 撓度、斜率、積分法、疊加法、Mohr's積分法、共軛梁法、靜不定梁、固定端梁 |
| **挫屈** | Euler臨界載重、有效長度、細長比、各種端部條件、臨界應力 |
| **塑性分析** | 塑性鉸、塑性矩、完全塑化、降伏載重、極限載重、殘留應力 |

## 9　`question_index.json` 欄位規範

### 各欄位允許值

| 欄位 | 允許值 | 說明 |
|------|--------|------|
| `moduleId` | `MM-YYYY-N` | 題目唯一識別碼 |
| `year` | 整數（西元年，如 `2016`） | 西元年 |
| `rocYear` | 整數（民國年，如 `105`） | 民國年 |
| `primaryTopicId` | `raw/json/syllabus_taxonomy.json` 中的 `id` | 命題大綱主分類，唯一 |
| `primaryTopicName` | `raw/json/syllabus_taxonomy.json` 中的 `name` | 主分類名稱（直接引用命題大綱子項） |
| `secondaryTopicIds` | `[]` 或 `["MM-UN-n"]` | 跨子項時填入，可多個 |
| `analysisMethod` | `彈性分析` / `塑性分析` / `概念題` / `混合` | 分析方法 |
| `verificationStatus` | `verified` / `unverified` / `needs-review` | 驗證狀態 |
| `hasSolution` | `true` / `false` | 是否已有解析 `.md` |
| `hasViz` | `true` / `false` | 是否有互動圖 |
| `tags` | 字串陣列（中文，3–8 個） | 核心考點標籤 |

### `verificationStatus` 說明

| 狀態 | 說明 | ingest 是否允許 |
|------|------|:--------------:|
| `verified` | 人工驗算確認正確 | ✅ |
| `unverified` | 尚未驗算 | ❌ |
| `needs-review` | 發現錯誤，待修正 | ❌ |

## 10　完成標準（Definition of Done）

### 一道題「解題完成」的標準

| 項目 | 檢查方式 |
|------|---------|
| `raw/solutions/MM-YYYY-N/MM-YYYY-N.md` 存在 | 檔案系統確認 |
| 開頭標籤區塊完整（編號、主分類、分析法、標籤） | 讀取 .md 前 20 行 |
| 所有獨立公式使用 LaTeX `$$...$$` | grep `\$\$` |
| 每張 PNG 圖片有對應 `*圖說：*` | grep `圖說：` |
| `question_index.json` 中 `hasSolution: true` | JSON 確認 |
| `tags` ≥ 3 個 | JSON 確認 |

### 一道題「ingest 完成」的標準

| 項目 | 檢查方式 |
|------|---------|
| `wiki/problems/MM-YYYY-N.md` 存在且有完整標籤 | 檔案系統確認 |
| `wiki/index.md` 在對應分類下有此題連結 | grep moduleId |
| `wiki/by-year.md` 在對應年份有此題 | grep moduleId |
| `wiki/log.md` 有 ingest 紀錄 | grep moduleId |

## 11　常見錯誤對照表

| 類別 | ❌ 錯誤 | ✅ 正確 | 原因 |
|------|--------|--------|------|
| moduleId | `MM-105-1` | `MM-2016-1` | 年份用民國而非西元 |
| moduleId | `MM-2016-01` | `MM-2016-1` | 題號有前導零 |
| moduleId | `mm-2016-1` | `MM-2016-1` | 科目代碼小寫 |
| 考卷 PDF | `MM-2016材料力學.pdf` | `MM-2016_材料力學.pdf` | 年份後缺底線 |
| PNG | `MM-2016-1-fig1.png` | `MM-2016-1-fig-1.png` | 類型碼與序號間缺連字號 |
| PNG | `MM-2016-1-eqn.png` | `MM-2016-1-eqn-1.png` | 缺序號（單張也要寫 `-1`） |
| viz HTML | `MM-2016-1-beam.html` | `MM-2016-1-beam-viz.html` | 缺 `-viz` 後綴 |
| analysisMethod | `elastic` | `彈性分析` | 分析法值用中文 |
| tags | `T梁` | `T形截面` | 標籤應含完整中文說明 |
| 公式 | `σ = My/I` | `$\sigma = My/I$` | 禁止純文字公式 |
| verificationStatus | `Verified` | `verified` | 狀態值全小寫 |
