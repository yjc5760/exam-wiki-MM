**主分類：** `MM-U1-3` 應力及應變分析原理與應用

**分析法：** 彈性分析

## 1. 原始題目重述 (Problem Restatement)

如圖三所示，一平面應力元素承受：

$$\sigma_x = 16{,}000\ \text{psi（拉）}, \quad \sigma_y = 6{,}000\ \text{psi（拉）}, \quad \tau_{xy} = +4{,}000\ \text{psi}$$

（正剪應力慣例：在 +x 面上，$\tau_{xy}$ 指向 +y 方向；在 +y 面上，$\tau_{yx}$ 指向 +x 方向）

**求：**
1. 主應力 $\sigma_1$、$\sigma_2$
2. 最大剪應力 $\tau_{max}$
3. 主應力軸 $x'$ 與 $x$ 軸之夾角 $\theta_p$

*圖說：應力元素中 $\sigma_x = 16{,}000\ \text{psi}$（右面向外）、$\sigma_y = 6{,}000\ \text{psi}$（上面向上）、$\tau = 4{,}000\ \text{psi}$（右面向上、上面向右為正方向）。*

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 平面應力轉換公式（或莫爾圓）的直接應用——給定 $(\sigma_x, \sigma_y, \tau_{xy})$，求主應力、最大剪應力及主應力方向。

**出題者意圖：**
- 測試考生能否正確套用主應力公式，並區分「平面內最大剪應力」與「絕對最大剪應力」
- 考核主應力方向角 $\theta_p$ 的計算（注意雙角公式 $\tan 2\theta_p$）
- 此類題型標準化程度高，是高分基本題，但符號錯誤是常見失分點

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 計算平均應力 $\sigma_{avg}$ 與莫爾圓半徑 $R$
2. 代入公式求 $\sigma_1$、$\sigma_2$
3. 由 $\tan 2\theta_p$ 求主應力方向角 $\theta_p$
4. 討論平面內最大剪應力 $\tau_{max,in}$ 與絕對最大剪應力 $\tau_{max,abs}$

**關鍵陷阱：**

| # | 陷阱 | 應對 |
|---|------|------|
| 1 | **$\tau_{max}$ 的定義混淆**：「平面內最大剪應力」$= R$，「絕對最大剪應力」（考慮第三主應力 $\sigma_3=0$）可能更大 | 本題兩個主應力均為正，絕對最大剪應力 $= \sigma_1/2$，大於平面內最大剪應力 |
| 2 | **$2\theta_p$ vs. $\theta_p$ 混淆**：$\tan(2\theta_p) = 2\tau/(\sigma_x-\sigma_y)$，除以 2 後才是 $\theta_p$ | 計算 $\arctan$ 後記得除以 2 |
| 3 | **主應力對應方向**：兩個主應力方向相差 90°；$\theta_{p1}$ 對應 $\sigma_1$（最大），$\theta_{p2} = \theta_{p1}+90°$ 對應 $\sigma_2$ | 代入驗算哪個角度給 $\sigma_1$，哪個給 $\sigma_2$ |
| 4 | **最大剪應力面 vs. 主應力面**：兩者差 45°；最大剪應力面不是主應力面 | 記「剪最大在主應力面轉 45°」 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標

主應力 $\sigma_1$、$\sigma_2$，最大剪應力 $\tau_{max}$，以及主應力方向角 $\theta_p$。

### 本題關鍵公式（依計算順序）

$$\text{Step 1: } \sigma_{avg} = \frac{\sigma_x + \sigma_y}{2}$$

$$\text{Step 2: } R = \sqrt{\left(\frac{\sigma_x - \sigma_y}{2}\right)^2 + \tau_{xy}^2}$$

$$\text{Step 3: } \sigma_{1,2} = \boxed{\sigma_{avg}} \pm \boxed{R}$$

$$\text{Step 4: } \tau_{max,\text{in-plane}} = \boxed{R}$$

$$\text{Step 5: } \tan(2\theta_p) = \frac{2\tau_{xy}}{\sigma_x - \sigma_y} \quad \Rightarrow \quad \theta_p = \frac{1}{2}\arctan\!\left(\frac{2\tau_{xy}}{\sigma_x-\sigma_y}\right)$$

$$\text{Step 6（進階）: } \tau_{max,abs} = \frac{\boxed{\sigma_1}}{2} \quad \text{（當 }\sigma_1>0>\sigma_3=0\text{，取平面外）}$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $\sigma_x$ | 16,000 psi | x 方向正應力（拉） |
| $\sigma_y$ | 6,000 psi | y 方向正應力（拉） |
| $\tau_{xy}$ | +4,000 psi | 剪應力（正方向：右面↑、上面→） |

### L2：需知識點推導

**【莫爾圓中心與半徑】**

| 符號 | 公式 | 數值 | 卡關? |
|------|------|------|-------|
| $\sigma_{avg}$ | $(\sigma_x+\sigma_y)/2$ | 11,000 psi | |
| $(\sigma_x-\sigma_y)/2$ | 應力差之半 | 5,000 psi | |
| $R$ | $\sqrt{5000^2+4000^2}$ | $1000\sqrt{41} \approx 6{,}403\ \text{psi}$ | |

**【主應力與最大剪應力】**

| 符號 | 公式 | 數值 | 卡關? |
|------|------|------|-------|
| $\sigma_1$ | $\sigma_{avg}+R$ | ≈ 17,403 psi | |
| $\sigma_2$ | $\sigma_{avg}-R$ | ≈ 4,597 psi | |
| $\tau_{max,\text{in}}$ | $R$ | ≈ 6,403 psi | |
| $\tau_{max,abs}$ | $\sigma_1/2$（因 $\sigma_3=0<\sigma_1$） | ≈ 8,702 psi | |

**【主應力方向角】**

| 符號 | 公式 | 數值 | 卡關? |
|------|------|------|-------|
| $2\theta_p$ | $\arctan(2\tau_{xy}/(\sigma_x-\sigma_y))=\arctan(0.8)$ | 38.66° | |
| $\theta_p$ | $2\theta_p / 2$ | **19.33°（CCW from x-axis）** | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| 莫爾圓幾何關係 | 圓心 = $(\sigma_{avg}, 0)$，半徑 = $R$；主應力 = 圓與 $\sigma$-軸交點；最大剪應力 = 圓的最高/最低點 | |
| 絕對最大剪應力與平面內最大剪應力的差異 | 平面應力有三個主應力（$\sigma_1$, $\sigma_2$, $\sigma_3=0$）；絕對最大剪應力 = 最大與最小主應力之差的一半 | |
| $\theta_p$ 方向的物理意義 | $\theta_p$ 為從 x 軸逆時針旋轉到 $\sigma_1$ 主應力面法線的夾角；確認方向需代入驗算 | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：計算平均應力（莫爾圓圓心）

$$\sigma_{avg} = \frac{\sigma_x + \sigma_y}{2} = \frac{16{,}000 + 6{,}000}{2} = \boxed{11{,}000\ \text{psi}}$$

### Step 2：計算莫爾圓半徑 $R$

$$R = \sqrt{\left(\frac{\sigma_x - \sigma_y}{2}\right)^2 + \tau_{xy}^2} = \sqrt{\left(\frac{16{,}000 - 6{,}000}{2}\right)^2 + (4{,}000)^2}$$

$$= \sqrt{(5{,}000)^2 + (4{,}000)^2} = \sqrt{25{,}000{,}000 + 16{,}000{,}000} = \sqrt{41{,}000{,}000}$$

$$= 1{,}000\sqrt{41} \approx \boxed{6{,}403\ \text{psi}}$$

### Step 3：主應力

$$\sigma_{1,2} = \sigma_{avg} \pm R = 11{,}000 \pm 1{,}000\sqrt{41}$$

$$\boxed{\sigma_1 = 11{,}000 + 1{,}000\sqrt{41} \approx 17{,}403\ \text{psi（拉）}}$$

$$\boxed{\sigma_2 = 11{,}000 - 1{,}000\sqrt{41} \approx 4{,}597\ \text{psi（拉）}}$$

> **策略註解：** 兩個主應力均為正值（拉力），表示在此應力狀態下不存在壓應力主軸。

### Step 4：最大剪應力

**平面內最大剪應力：**

$$\tau_{max,\text{in-plane}} = R = 1{,}000\sqrt{41} \approx \boxed{6{,}403\ \text{psi}}$$

（發生在與主應力面成 45° 的截面上，此時正應力 = $\sigma_{avg} = 11{,}000\ \text{psi}$）

**絕對最大剪應力（含第三主應力方向）：**

平面應力狀態下，第三主應力 $\sigma_3 = 0$（垂直紙面方向無應力）。

由於 $\sigma_1 > \sigma_2 > \sigma_3 = 0$，絕對最大剪應力發生在 $\sigma_1$–$\sigma_3$ 平面（即含 $\sigma_1$ 主方向、垂直於紙面方向的 45° 斜面）：

$$\tau_{max,abs} = \frac{\sigma_1 - \sigma_3}{2} = \frac{17{,}403 - 0}{2} \approx \boxed{8{,}702\ \text{psi}}$$

> **考場策略：** 若題目只說「最大剪應力 $\tau_{max}$」而不指定「平面內」，應同時給出兩個值並說明哪個更大；本題絕對最大剪應力（8,702 psi）> 平面內最大剪應力（6,403 psi）。

### Step 5：主應力方向角 $\theta_p$

$$\tan(2\theta_p) = \frac{2\tau_{xy}}{\sigma_x - \sigma_y} = \frac{2 \times 4{,}000}{16{,}000 - 6{,}000} = \frac{8{,}000}{10{,}000} = 0.8$$

$$2\theta_p = \arctan(0.8) = 38.66°$$

$$\boxed{\theta_p = 19.33° \approx 19.3°\text{（逆時針，x 軸轉至 }\sigma_1\text{ 主應力面法線方向）}}$$

### Step 6：驗算主應力方向

代入應力轉換公式，在 $\theta = 19.33°$ 截面：

$$\sigma_{\theta} = \frac{\sigma_x+\sigma_y}{2} + \frac{\sigma_x-\sigma_y}{2}\cos(2\theta) + \tau_{xy}\sin(2\theta)$$

$$= 11{,}000 + 5{,}000\cos(38.66°) + 4{,}000\sin(38.66°)$$

$$= 11{,}000 + 5{,}000 \times 0.781 + 4{,}000 \times 0.625$$

$$= 11{,}000 + 3{,}904 + 2{,}499 = 17{,}403\ \text{psi} \approx \sigma_1 \checkmark$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 「平面內最大剪應力」vs.「絕對最大剪應力」

| | 平面內 | 絕對最大 |
|---|--------|---------|
| 值 | $R = 6{,}403\ \text{psi}$ | $\sigma_1/2 = 8{,}702\ \text{psi}$ |
| 發生截面 | 與主應力面成 45°（同一平面內） | 包含 $\sigma_1$ 方向且垂直紙面的 45° 斜面 |
| 正應力 | $\sigma_{avg} = 11{,}000\ \text{psi}$ | $\sigma_1/2 = 8{,}702\ \text{psi}$ |

**判斷規則（背起來）：**
- 若 $\sigma_1$ 與 $\sigma_2$ **同號**：絕對最大剪應力 = 較大主應力/2（本題適用）
- 若 $\sigma_1$ 與 $\sigma_2$ **異號**：絕對最大剪應力 = $(\sigma_1 - \sigma_2)/2$（等於平面內最大剪應力）

### 考場安全策略

1. 先求 $\sigma_{avg}$ 和 $R$，再求 $\sigma_1$、$\sigma_2$：這三個數字是莫爾圓的「圓心」和「半徑」
2. $\theta_p$ 的計算：$\arctan(0.8) = 38.66°$ → 除以 2 → **19.33°**（不要忘記除以 2！）
3. 最後比較 $\sigma_1$、$\sigma_2$、$\sigma_3=0$ 的大小，判斷絕對最大剪應力平面

### 精確值表達

$$\sigma_1 = (11{,}000 + 1{,}000\sqrt{41})\ \text{psi}, \quad \sigma_2 = (11{,}000 - 1{,}000\sqrt{41})\ \text{psi}$$

$$\tau_{max,\text{in}} = 1{,}000\sqrt{41}\ \text{psi}, \quad \theta_p = \frac{1}{2}\arctan(0.8)$$
