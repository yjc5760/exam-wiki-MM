# 考題編號：MM-2010-4

**主分類：** `MM-U4-1` 軸力桿件、扭力桿件與梁之塑性分析
**副分類：** `MM-U4-2` 殘留應力與應變
**分析法：** 塑性分析
**標籤：** `梁彈塑性分析` `降伏彎矩My` `曲率半徑` `彈塑性彎曲` `殘留曲率` `彈性卸載` `矩形斷面` `完全彈塑性材料`

---

## 1. 原始題目重述 (Problem Restatement)

本題為矩形斷面梁的彈塑性彎曲分析，共四個子問題：

**(一)** 彎曲理論中 $\varepsilon = y/R$ 是基於什麼假設而成立？此公式是否適用於塑性範圍？為什麼？

**(二)** 一桿件其斷面為寬 50 mm、高 120 mm 的矩形斷面，理想彈塑性材料，$\sigma_y = 240\,\text{MPa}$，$E = 200\,\text{GPa}$，求降伏彎矩 $M_y$ 及達到 $M_y$ 時對應之曲率半徑。

**(三)** 若 $M = 35\,\text{kN·m}$ 作用於該斷面，求曲率半徑。

**(四)** 其後又把 $M = 35\,\text{kN·m}$ 移除，求最後沒有彎矩作用下之曲率半徑。

*（本題無附圖，幾何與材料條件均以文字給定）*

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
1. 平截面假設（Bernoulli-Euler）→ 幾何（動力學）關係，不依賴材料本構
2. 理想彈塑性材料的彎曲分析：降伏彎矩 $M_y$ → 部分塑性（$M > M_y$）→ 彈性卸載 → 殘留曲率
3. 彈性卸載的疊加原理（卸載過程視為彈性回彈）

**出題者意圖：**
- 觀念題(一)：澄清「幾何假設」與「材料假設」的區別
- 計算題(二)~(四)：完整走一遍「加載到降伏→繼續加載進入塑性→卸載殘留」的全流程

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

### 作戰計畫

```
(一) 平截面假設說明（2個假設）→ 塑性範圍適用性分析

(二) 斷面性質 I, c → My = σy·I/c → κy = My/(EI) → Ry

(三) M=35>My → 部分塑性 → 解 yY（彈性核心半高）→ 曲率 κ = σy/(E·yY) → R

(四) 彈性卸載 Δκ = M/(EI) → κ_res = κ_load - Δκ → R_res
```

### 關鍵陷阱

| # | 陷阱 | 正確做法 |
|---|------|---------|
| 1 | 誤以為 $\varepsilon=y/R$ 僅適用彈性 | 此為幾何關係，塑性範圍亦成立（只要平截面假設成立） |
| 2 | 部分塑性彎矩公式寫錯 | $M = \sigma_y b(h^2/4 - y_Y^2/3)$，注意係數 |
| 3 | 曲率公式混淆 | 彈性段：$\kappa = M/(EI)$；部分塑性：$\kappa = \sigma_y/(E\cdot y_Y)$ |
| 4 | 卸載用 $\Delta\kappa = M/(EI)$ | 卸載視為彈性回彈，故 $\Delta\kappa = M_{unload}/(EI)$（代入原本加載量 M） |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標
求矩形斷面在 $M_y$、$M=35\,\text{kN·m}$、及卸載後三種狀態下的曲率半徑 $R$，以及理解 $\varepsilon = y/R$ 的適用性。

### 本題關鍵公式（依計算順序）

$$\text{Step 1: } I = \frac{bh^3}{12}, \quad c = \frac{h}{2}$$

$$\text{Step 2: } M_y = \sigma_y \cdot \frac{I}{c} = \sigma_y \cdot S \quad (S=\text{彈性斷面模數})$$

$$\text{Step 3: } \kappa_y = \frac{M_y}{\boxed{EI}}, \quad R_y = \frac{1}{\boxed{\kappa_y}}$$

$$\text{Step 4（部分塑性彎矩）: } M = \sigma_y \cdot b\left(\frac{h^2}{4} - \frac{y_Y^2}{3}\right) \Rightarrow \text{解} \; y_Y$$

$$\text{Step 5（部分塑性曲率）: } \kappa = \frac{\sigma_y}{E \cdot \boxed{y_Y}}, \quad R = \frac{1}{\boxed{\kappa}}$$

$$\text{Step 6（彈性卸載）: } \Delta\kappa = \frac{M}{\boxed{EI}}, \quad \kappa_{res} = \boxed{\kappa} - \Delta\kappa, \quad R_{res} = \frac{1}{\boxed{\kappa_{res}}}$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $b$ | $50\,\text{mm}$ | 矩形斷面寬度 |
| $h$ | $120\,\text{mm}$ | 矩形斷面高度 |
| $\sigma_y$ | $240\,\text{MPa} = 240\,\text{N/mm}^2$ | 降伏應力（理想彈塑性） |
| $E$ | $200\,\text{GPa} = 200{,}000\,\text{N/mm}^2$ | 彈性模數 |
| $M$ | $35\,\text{kN·m} = 35\times10^6\,\text{N·mm}$ | 外加彎矩（子問題三） |

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### (一) $\varepsilon = y/R$ 的假設與塑性範圍適用性

**成立所需的假設：**

1. **平截面假設（Bernoulli-Euler 假設）：** 梁彎曲前為平截面，彎曲後仍保持平截面，即截面內各點的軸向應變 $\varepsilon$ 沿截面高度呈線性分布。

2. **小變形假設：** 曲率半徑 $R$ 遠大於梁的尺寸，使曲率近似 $\kappa \approx v'' = M/EI$（線性曲率）。

**是否適用於塑性範圍？**

**可以。** $\varepsilon = y/R$ 是純幾何（運動學）關係，只依賴「平截面保持平截面」這一幾何假設，**不涉及材料的應力-應變關係（本構律）**。

- 在彈性範圍：$\varepsilon = y/R$，再由虎克定律 $\sigma = E\varepsilon = Ey/R$ → 線性應力分布
- 在塑性範圍：$\varepsilon = y/R$ **仍成立**（幾何關係不變），但應力分布不再線性（受材料本構律約束，如理想彈塑性材料，超過 $\varepsilon_y = \sigma_y/E$ 後應力不再增加，維持在 $\pm\sigma_y$）

只要材料的塑性變形仍維持「截面無翹曲」（即各點應變沿截面高度線性分布），此假設即告成立。

---

### (二) 降伏彎矩 $M_y$ 與對應曲率半徑

#### 斷面性質計算

$$I = \frac{bh^3}{12} = \frac{50 \times 120^3}{12} = \frac{50 \times 1{,}728{,}000}{12} = 7{,}200{,}000\,\text{mm}^4 = 7.2 \times 10^6\,\text{mm}^4$$

$$c = \frac{h}{2} = 60\,\text{mm}$$

$$EI = 200{,}000 \times 7{,}200{,}000 = 1.44 \times 10^{12}\,\text{N·mm}^2 = 1{,}440\,\text{kN·m}^2$$

#### 降伏彎矩

$$M_y = \sigma_y \cdot \frac{I}{c} = 240 \times \frac{7.2\times10^6}{60} = 240 \times 120{,}000 = 28{,}800{,}000\,\text{N·mm}$$

$$\boxed{M_y = 28.8\,\text{kN·m}}$$

#### 對應曲率半徑

$$\kappa_y = \frac{M_y}{EI} = \frac{28{,}800{,}000}{1.44\times10^{12}} = 2.0 \times 10^{-5}\,\text{mm}^{-1}$$

$$\boxed{R_y = \frac{1}{\kappa_y} = 50{,}000\,\text{mm} = 50\,\text{m}}$$

（驗證：$\varepsilon_y = \sigma_y/E = 240/200{,}000 = 1.2\times10^{-3}$；$\kappa_y = \varepsilon_y/c = 1.2\times10^{-3}/60 = 2.0\times10^{-5}\,\text{mm}^{-1}$ ✓）

---

### (三) $M = 35\,\text{kN·m}$ 時之曲率半徑

#### 確認為部分塑性狀態

$$M = 35\,\text{kN·m} > M_y = 28.8\,\text{kN·m}$$

故截面已進入**部分塑性**狀態：外側塑性區（$|y| > y_Y$，應力 $= \pm\sigma_y$）+ 內側彈性核心（$|y| \leq y_Y$，應力線性分布）。

#### 部分塑性彎矩公式推導

設彈性核心半高為 $y_Y$（彈性區：$-y_Y \leq y \leq y_Y$）：

$$M = 2\int_0^{y_Y} \frac{\sigma_y}{y_Y} y \cdot (by)\,dy + 2\int_{y_Y}^{h/2} \sigma_y \cdot by\,dy$$

$$= \frac{2b\sigma_y}{y_Y}\cdot\frac{y_Y^3}{3} + 2b\sigma_y\left[\frac{(h/2)^2 - y_Y^2}{2}\right]$$

$$= \frac{2b\sigma_y y_Y^2}{3} + b\sigma_y\left(\frac{h^2}{4} - y_Y^2\right)$$

$$= b\sigma_y\left(\frac{h^2}{4} - \frac{y_Y^2}{3}\right) \tag{部分塑性彎矩公式}$$

*策略註解：此公式可記憶為「全塑性弦矩 $\sigma_y b h^2/4$ 減去彈性核心少算的部分」*

#### 代入數值，解 $y_Y$

$$35 \times 10^6 = 50 \times 240 \left(\frac{120^2}{4} - \frac{y_Y^2}{3}\right) = 12{,}000\left(3600 - \frac{y_Y^2}{3}\right)$$

$$\frac{35\times10^6}{12{,}000} = 3600 - \frac{y_Y^2}{3}$$

$$2916.67 = 3600 - \frac{y_Y^2}{3}$$

$$\frac{y_Y^2}{3} = 3600 - 2916.67 = 683.33\,\text{mm}^2$$

$$y_Y^2 = 2050\,\text{mm}^2 \Rightarrow \boxed{y_Y = 45.28\,\text{mm}}$$

（驗證：$y_Y = 45.28 < c = 60\,\text{mm}$ ✓，確實為部分塑性）

#### 計算曲率半徑

在彈性核心邊界 $y = y_Y$ 處，應力恰為 $\sigma_y$，由 $\varepsilon = y/R$ 與虎克定律（彈性核心內有效）：

$$\varepsilon_y = \frac{\sigma_y}{E} = \frac{y_Y}{R} \Rightarrow \kappa = \frac{1}{R} = \frac{\sigma_y}{E \cdot y_Y}$$

$$\kappa = \frac{240}{200{,}000 \times 45.28} = \frac{240}{9{,}056{,}000} = 2.650 \times 10^{-5}\,\text{mm}^{-1}$$

$$\boxed{R = \frac{1}{\kappa} \approx 37{,}736\,\text{mm} \approx 37.7\,\text{m}}$$

---

### (四) 移除 $M = 35\,\text{kN·m}$ 後之殘留曲率半徑

#### 彈性卸載原理

將 $M = 35\,\text{kN·m}$ 移除，等效於疊加一個 **反向彈性彎矩** $-35\,\text{kN·m}$。

卸載過程假設為**彈性**（即使材料已進入塑性，卸載路徑沿初始彈性斜率進行），曲率變化：

$$\Delta\kappa_{unload} = \frac{M_{unload}}{EI} = \frac{35\times10^6}{1.44\times10^{12}} = 2.431\times10^{-5}\,\text{mm}^{-1}$$

#### 殘留曲率

$$\kappa_{res} = \kappa_{load} - \Delta\kappa_{unload} = 2.650\times10^{-5} - 2.431\times10^{-5} = 0.219\times10^{-5}\,\text{mm}^{-1}$$

#### 殘留曲率半徑

$$\boxed{R_{res} = \frac{1}{\kappa_{res}} = \frac{1}{2.19\times10^{-6}} \approx 456{,}621\,\text{mm} \approx 457\,\text{m}}$$

#### 物理意義

卸載後雖然外力為零，截面仍保有**殘留曲率** $\kappa_{res} \approx 2.19\times10^{-6}\,\text{mm}^{-1}$（**永久彎曲**）。殘留曲率半徑高達約 457 m，說明塑性變形量雖小，但對精密構件（如彈簧、精密梁）的幾何形狀仍有顯著影響。

#### 計算流程彙整

| 狀態 | 彎矩 $M$ | 曲率 $\kappa$ | 曲率半徑 $R$ |
|------|---------|-------------|------------|
| 初次降伏 | $M_y = 28.8\,\text{kN·m}$ | $2.000\times10^{-5}\,\text{mm}^{-1}$ | $50\,\text{m}$ |
| 部分塑性加載 | $M = 35\,\text{kN·m}$ | $2.650\times10^{-5}\,\text{mm}^{-1}$ | $37.7\,\text{m}$ |
| 彈性卸載量 | $\Delta M = 35\,\text{kN·m}$ | $-2.431\times10^{-5}\,\text{mm}^{-1}$ | — |
| 卸載後殘留 | $0$ | $0.219\times10^{-5}\,\text{mm}^{-1}$ | **$\approx 457\,\text{m}$** |

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 彈性核心寬度 $y_Y$ 與塑性程度

本題中 $y_Y = 45.28\,\text{mm}$，$c = 60\,\text{mm}$，故塑性化比例為：

$$\text{塑性化深度} = c - y_Y = 14.72\,\text{mm} \quad \Rightarrow \quad \text{佔半高比例} = \frac{14.72}{60} \approx 24.5\%$$

截面尚未完全塑性化（完全塑性化需 $y_Y \to 0$，此時 $M_p = \sigma_y b h^2/4 = 43.2\,\text{kN·m}$）。

### 形狀係數（Shape Factor）

$$f = \frac{M_p}{M_y} = \frac{43.2}{28.8} = 1.5$$

矩形截面的形狀係數為 **1.5**（理論值），意味著截面在完全塑性狀態下能承受的彎矩比初始降伏時多50%。

### 卸載殘留應力分布（進階）

移除 $M = 35\,\text{kN·m}$ 後：
- **加載應力分布**：$|y| \leq y_Y$ 線性（最大 $\pm\sigma_y$），$|y| > y_Y$ 均為 $\pm\sigma_y$
- **彈性回彈應力**（$-35\,\text{kN·m}$ 的彈性分布）：在截面頂底為 $\mp M\cdot c/I = \mp240\times(35/28.8) = \mp291.7\,\text{MPa}$
- **殘留應力** = 加載 $-$ 彈性回彈：截面外側出現反號殘留應力

此殘留應力分布決定了構件再次受力時的行為（「彈性安定」現象）。
