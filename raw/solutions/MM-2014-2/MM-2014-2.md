## 1. 原始題目重述 (Problem Restatement)

彈性模數為 $E$ 的直梁，斷面承受合成彎矩 $M_\xi$、$M_\eta$，及合成軸力 $N$ 之作用。其斷面正向纖維應力（fiber stress）分布假設為：

$$\sigma = a\xi + b\eta + d$$

其中 $\xi$、$\eta$ 為以斷面形心 $C$ 為原點之**主慣性軸**（principal centroidal axes），如圖 2 所示。

題目定義斷面性質：

$$I_\xi = \iint_A \eta^2 \, dA, \qquad I_\eta = \iint_A \xi^2 \, dA$$

**請求**：係數 $a$、$b$、$d$ 與合成彎矩 $M_\xi$、$M_\eta$ 及合成軸力 $N$ 之關係。

![MM-2014-2 題目附圖：斷面形心 C 為原點，ξ 為水平主慣性軸，η 為垂直主慣性軸，斷面積 A 位於形心下方](MM-2014-2-fig-1.png)

*圖說：η 為縱軸（垂直），ξ 為橫軸（水平），形心 C 為原點，斷面面積 A 以積分表示*

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 從平衡積分（截面合力 = 內力結果）推導廣義撓曲公式的係數，確認考生理解「應力分布是假設（線性分布），內力平衡是約束（給出係數）」的邏輯。

**出題者意圖：**
- 測驗考生能否用形心主軸的正交性質（$\iint \xi\,dA = 0$，$\iint \eta\,dA = 0$，$\iint \xi\eta\,dA = 0$）解耦三個係數
- 確認考生清楚 $I_\xi = \iint \eta^2\,dA$（對 $\xi$ 軸的慣性矩，積分變數是 $\eta$）的定義
- 驗證考生知道廣義撓曲公式 $\sigma = \frac{N}{A} + \frac{M_\eta}{I_\eta}\xi + \frac{M_\xi}{I_\xi}\eta$ 的理論根源

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**步驟化作戰計畫：**
1. 寫出三個截面合力/合力矩的積分定義（$N$、$M_\xi$、$M_\eta$）
2. 將 $\sigma = a\xi + b\eta + d$ 代入各積分
3. 利用形心主軸的正交性質化簡，分別解出 $d$、$b$、$a$
4. 整理成 $\sigma$ 的最終表達式

**關鍵陷阱：**

| # | 陷阱 | 正確處理 |
|---|------|---------|
| 1 | $I_\xi$ 與 $I_\eta$ 的定義搞混 | $I_\xi = \iint \eta^2\,dA$（以 $\eta$ 距離對 $\xi$ 軸），$I_\eta = \iint \xi^2\,dA$ |
| 2 | 誤認為 $M_\xi$ 的應力對應 $\xi$（水平方向） | $M_\xi$ 是對 $\xi$ 軸的力矩，造成的應力隨 $\eta$ 變化，故 $b = M_\xi/I_\xi$ |
| 3 | 忘記主軸條件 $\iint \xi\eta\,dA = 0$，導致耦合項無法消去 | 主軸定義即使慣性積為零，必須明確說明此性質的使用 |
| 4 | $M_\eta$ 的力臂方向（是 $\xi$ 還是 $\eta$） | $M_\eta = \iint \sigma\cdot\xi\,dA$（對 $\eta$ 軸的力矩，力臂是 $\xi$） |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
求係數 $a$、$b$、$d$，以 $N$、$M_\xi$、$M_\eta$、$A$、$I_\xi$、$I_\eta$ 表示

### 本題關鍵公式（依計算順序）

$$\text{Step 1: } N = \iint_A \sigma\,dA \Rightarrow N = d \cdot A$$

$$\text{Step 2: } M_\xi = \iint_A \sigma\cdot\eta\,dA \Rightarrow M_\xi = b \cdot \boxed{I_\xi}$$

$$\text{Step 3: } M_\eta = \iint_A \sigma\cdot\xi\,dA \Rightarrow M_\eta = a \cdot \boxed{I_\eta}$$

$$\text{Step 4（解係數）: } d = \frac{N}{A},\quad b = \frac{M_\xi}{I_\xi},\quad a = \frac{M_\eta}{I_\eta}$$

$$\text{Step 5（代回）: } \sigma = \frac{M_\eta}{I_\eta}\xi + \frac{M_\xi}{I_\xi}\eta + \frac{N}{A}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 前提：形心主軸的積分性質

由於 $\xi$、$\eta$ 是以**形心**為原點的**主慣性軸**，有以下六個積分性質：

$$\iint_A dA = A \tag{P1}$$

$$\iint_A \xi\,dA = 0, \qquad \iint_A \eta\,dA = 0 \tag{P2，形心條件}$$

$$\iint_A \xi\eta\,dA = 0 \tag{P3，主軸條件（慣性積為零）}$$

$$\iint_A \xi^2\,dA = I_\eta, \qquad \iint_A \eta^2\,dA = I_\xi \tag{P4，慣性矩定義}$$

---

### Step 1：由軸力 N 求係數 d

截面合軸力等於正向應力對全截面的積分：

$$N = \iint_A \sigma\,dA = \iint_A (a\xi + b\eta + d)\,dA$$

$$= a\underbrace{\iint_A \xi\,dA}_{=0\ (P2)} + b\underbrace{\iint_A \eta\,dA}_{=0\ (P2)} + d\underbrace{\iint_A dA}_{=A\ (P1)}$$

$$N = dA$$

$$\boxed{d = \frac{N}{A}}$$

*策略注解：形心條件 (P2) 使 $a$、$b$ 項消去，$d$ 可直接由 $N/A$ 決定。*

---

### Step 2：由彎矩 $M_\xi$ 求係數 b

$M_\xi$ 為截面正向應力對 $\xi$ 軸的力矩（力臂為 $\eta$）：

$$M_\xi = \iint_A \sigma\cdot\eta\,dA = \iint_A (a\xi + b\eta + d)\,\eta\,dA$$

$$= a\underbrace{\iint_A \xi\eta\,dA}_{=0\ (P3)} + b\underbrace{\iint_A \eta^2\,dA}_{=I_\xi\ (P4)} + d\underbrace{\iint_A \eta\,dA}_{=0\ (P2)}$$

$$M_\xi = bI_\xi$$

$$\boxed{b = \frac{M_\xi}{I_\xi}}$$

*策略注解：主軸條件 (P3) 消去 $a$ 項，形心條件 (P2) 消去 $d$ 項，$b$ 由 $M_\xi/I_\xi$ 單獨決定。*

---

### Step 3：由彎矩 $M_\eta$ 求係數 a

$M_\eta$ 為截面正向應力對 $\eta$ 軸的力矩（力臂為 $\xi$）：

$$M_\eta = \iint_A \sigma\cdot\xi\,dA = \iint_A (a\xi + b\eta + d)\,\xi\,dA$$

$$= a\underbrace{\iint_A \xi^2\,dA}_{=I_\eta\ (P4)} + b\underbrace{\iint_A \eta\xi\,dA}_{=0\ (P3)} + d\underbrace{\iint_A \xi\,dA}_{=0\ (P2)}$$

$$M_\eta = aI_\eta$$

$$\boxed{a = \frac{M_\eta}{I_\eta}}$$

---

### Step 4：最終結果彙整

將 $a$、$b$、$d$ 代回 $\sigma = a\xi + b\eta + d$：

$$\boxed{\sigma = \frac{M_\eta}{I_\eta}\,\xi + \frac{M_\xi}{I_\xi}\,\eta + \frac{N}{A}}$$

係數對照表：

| 係數 | 與內力的關係 | 物理意義 |
|------|------------|---------|
| $a$ | $\dfrac{M_\eta}{I_\eta}$ | $M_\eta$（對 $\eta$ 軸的彎矩）產生隨 $\xi$ 線性變化的應力 |
| $b$ | $\dfrac{M_\xi}{I_\xi}$ | $M_\xi$（對 $\xi$ 軸的彎矩）產生隨 $\eta$ 線性變化的應力 |
| $d$ | $\dfrac{N}{A}$ | 軸力 $N$ 產生均勻應力（與 $\xi$、$\eta$ 無關） |

---

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**Q1：若 ξ、η 不是主軸，公式如何修正？**

若使用非主軸（慣性積 $I_{\xi\eta} \neq 0$），應力公式需加上交叉項：

$$\sigma = \frac{M_\xi I_\eta + M_\eta I_{\xi\eta}}{I_\xi I_\eta - I_{\xi\eta}^2}\,\eta + \frac{M_\eta I_\xi + M_\xi I_{\xi\eta}}{I_\xi I_\eta - I_{\xi\eta}^2}\,\xi + \frac{N}{A}$$

可見主軸（$I_{\xi\eta}=0$）大幅簡化公式，這正是選主軸的原因。

**Q2：中性軸的位置？**

令 $\sigma = 0$：

$$\frac{M_\eta}{I_\eta}\,\xi + \frac{M_\xi}{I_\xi}\,\eta + \frac{N}{A} = 0$$

中性軸為一直線，其斜率與方向由 $M_\xi/M_\eta$ 決定。有軸力 $N \neq 0$ 時中性軸不過形心。

**Q3：本題的實際應用意義？**

此公式是不對稱彎曲（biaxial bending）與偏心壓縮問題的基礎。工程上，若荷載不在對稱面內，必須同時考慮 $M_\xi$、$M_\eta$ 兩個分量，最大應力出現在最遠離中性軸的角落纖維。
