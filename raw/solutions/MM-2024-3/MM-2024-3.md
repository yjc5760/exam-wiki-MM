# 考題編號：MM-2024-3

**主分類：** `MM-U3-4` 柱之挫屈載重分析
**分析法：** 彈性分析
**標籤：** `剛性桿挫屈` `彈簧支撐` `穩定性分析` `擾動平衡法` `能量法` `臨界載重` `單自由度系統`

---

## 1. 原始題目重述 (Problem Restatement)

如圖所示，由兩根**剛性桿**用鉸接連結而成的柱，兩桿長度各為 $L$，柱頂有壓力 $P$ 作用：

- $A$ 端（底）：鉸支承；
- $B$ 端（頂）：滾支承（可沿柱軸向滑動、側向受拘束）；
- $C$ 點（中間鉸）：有彈簧常數 $k$ 之線彈簧側向支撐。

試求此柱的**臨界載重** $P_{cr}$（答案以 $kL$ 的形式表示）。（40 分）

![MM-2024-3 題目附圖：垂直柱由兩段剛性桿於C點鉸接組成，底端A鉸支承、頂端B滾支承受壓力P，中點C由水平線彈簧k支撐](MM-2024-3-fig-1.png)

*圖說：垂直柱總高 2L，由上下兩段剛性桿（各長 L）以鉸 C 相連。A（底）鉸支承、B（頂）滾支承（側向拘束、軸向自由），P 自頂端下壓。C 點接水平線彈簧 k（另端固定）。*

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

本題是**穩定性概念**的純粹測驗——「挫屈不一定要用歐拉公式」。剛性桿系統中彈性全部集中在彈簧，挫屈與否取決於「**幾何剛度（P 的失穩效應）**與**彈簧恢復勁度**的競爭」。出題者意圖：

- 測驗考生是否理解**臨界載重的本質定義**：存在非平凡（傾斜）平衡位置的最小載重，而非背誦 $\pi^2EI/(KL)^2$；
- **擾動平衡法**：給系統一個微小位移 δ，寫位移後的平衡方程式，非平凡解條件即 $P_{cr}$；
- **能量法對照**：$P$ 下降做的功 vs 彈簧儲能，臨界條件為兩者相等（中性平衡）；
- 配分 40 分（全卷最高），暗示需要**完整的分析過程**：自由體圖、反力、兩種方法相互驗證，而非只給答案。

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

1. 確認運動自由度：A、B 側向受拘束，剛性桿系統僅 C 可側移 ⇒ **單自由度**（廣義座標 δ = C 點側移）。
2. 畫位移後構形：下桿、上桿各傾 $\theta = \delta/L$（小變位）。
3. 整體平衡求反力 → 取上桿（或下桿）對 C 取矩 → 含 δ 的平衡式。
4. 非平凡解（δ ≠ 0）條件 ⇒ $P_{cr}$。
5. 能量法驗證：$P\,\Delta_{vert} = \frac{1}{2}k\delta^2$。

**關鍵陷阱：**

| # | 陷阱 | 應對策略 |
|---|------|---------|
| 1 | 誤用歐拉公式 $\pi^2EI/(KL)^2$ | 剛性桿 EI→∞，歐拉不適用；穩定性由彈簧控制 |
| 2 | 對「原始（未位移）構形」寫平衡 ⇒ 只得 δ=0 | 穩定性分析必須對**位移後構形**寫平衡 |
| 3 | 忘記 B 端滾支承的水平反力 | B 側向受拘束會出水平反力 $B_x$，整體取矩求之 |
| 4 | P 的力臂取錯：P 通過 B（未側移），對 A 取矩力臂為 0 | 先釐清各點位移：A、B 不動，僅 C 移 δ |
| 5 | 能量法漏算 P 的下降量 | 每桿縮短垂直投影 $L(1-\cos\theta)\approx L\theta^2/2$，兩桿共 $\delta^2/L$ |

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`求臨界載重 Pcr（以 kL 表示）`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: } \text{自由度判定：僅 } C \text{ 側移 } \delta,\quad \theta = \frac{\delta}{L}$$

$$\text{Step 2: } \Sigma M_A = 0\ (\text{整體}):\ B_x(2L) = (k\delta)L \Rightarrow B_x = \frac{k\delta}{2}$$

$$\text{Step 3: } \Sigma M_C = 0\ (\text{上桿}):\ P\delta = \boxed{B_x}\,L$$

$$\text{Step 4: } \delta \neq 0 \Rightarrow P_{cr} = \frac{kL}{2}$$

$$\text{Step 5（驗證）: } P\cdot\frac{\delta^2}{L} = \frac{1}{2}k\delta^2 \Rightarrow P_{cr} = \frac{kL}{2}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

> 📊 互動圖：`MM-2024-3-buckling-viz.html`（位移構形與 P–k 平衡競爭視覺化）

### Step 1：自由度判定與位移後構形

A 端（鉸）與 B 端（滾支承）皆無側向位移；兩桿皆剛性 ⇒ 系統唯一的運動模式為**中間鉸 C 側移 δ**：

- 下桿 AC：繞 A 旋轉 $\theta_1 = \delta/L$；
- 上桿 CB：繞 B 旋轉 $\theta_2 = \delta/L$（B 沿軸向下滑以維持桿長，小變位下側向不動）；
- 彈簧受壓縮（伸長）δ，產生恢復力 $F_s = k\delta$（方向與 δ 相反）。

> **策略註解：** 單自由度確認是本題最關鍵的一步——它保證臨界載重只有一個，且可用純代數求解（陷阱 1、2）。

### Step 2：整體平衡求 B 端水平反力

位移後構形之整體自由體：P（沿原鉛直線通過 B、A）、A 反力 $(A_x, A_y)$、B 反力 $B_x$（水平）、彈簧力 $k\delta$（作用於 C，高度 L）。

$$\Sigma M_A = 0:\quad B_x (2L) - (k\delta)(L) = 0 \;\Rightarrow\; B_x = \frac{k\delta}{2}$$

（P 通過 A 正上方之 B 點且 B 未側移，對 A 無力臂。）

$$\Sigma F_x = 0:\quad A_x = k\delta - B_x = \frac{k\delta}{2};\qquad \Sigma F_y = 0:\quad A_y = P$$

### Step 3：上桿對中間鉸 C 取矩（消去鉸內力）

上桿 CB 自由體：B 端受 $P$（向下）與 $B_x$（水平）；C 端受鉸內力（取矩消去）。C 至 B 之向量：側向 $-\delta$、垂直 $+L$：

$$\Sigma M_C = 0:\quad P\,\delta - B_x\,L = 0 \;\Rightarrow\; P\,\delta = \frac{k\delta}{2}\,L$$

### Step 4：臨界條件（非平凡解）

$$\left(P - \frac{kL}{2}\right)\delta = 0$$

$\delta \neq 0$（傾斜平衡位形存在）之條件：

$$\boxed{P_{cr} = \frac{kL}{2}}$$

**下桿交叉驗證**——下桿 AC 對 C 取矩：A 端力 $A_x = k\delta/2$、$A_y = P$，C 至 A 向量：側向 $-\delta$、垂直 $-L$：

$$\Sigma M_C = 0:\quad P\,\delta - A_x L = P\delta - \frac{k\delta}{2}L = 0 \;\Rightarrow\; P_{cr} = \frac{kL}{2}\ ✓$$

### Step 5：能量法驗證（中性平衡）

C 側移 δ 時，每桿之垂直投影縮短 $L(1-\cos\theta) \approx L\theta^2/2 = \delta^2/2L$，B 端（P 作用點）總下降量：

$$\Delta = 2 \times \frac{\delta^2}{2L} = \frac{\delta^2}{L}$$

中性平衡條件（P 做功 = 彈簧儲能）：

$$P\,\Delta = \frac{1}{2}k\delta^2 \;\Rightarrow\; P\,\frac{\delta^2}{L} = \frac{1}{2}k\delta^2 \;\Rightarrow\; \boxed{P_{cr} = \frac{kL}{2}}\ ✓$$

> **策略註解：** 兩法答案一致。物理判讀：$P < kL/2$ 時彈簧扳得回（穩定）；$P > kL/2$ 時 P 的傾覆效應勝出（不穩定）。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

1. **為何答案不含 EI？** 桿件為剛性（EI→∞），不儲存彎曲應變能；系統唯一的彈性來源是彈簧。若桿改為彈性，問題變成「彈性柱＋中點彈性支撐」，需解超越方程式，$P_{cr}$ 介於 $kL/2$ 與歐拉值之間取低者——考場上看到「剛性桿」三字就應直接走擾動平衡法。

2. **B 端滾支承的判讀**：本解依圖將 B 解讀為「側向拘束、軸向（鉛直）自由」——P 才能對系統做功、柱才能整體縮短。若誤解為「B 完全自由」，系統變成可整體傾倒的機構（對 A 取矩：$P$ 有力臂），臨界載重變為 $kL/4$（單桿傾倒模式不存在於本幾何，因 C 為鉸……實際上 B 自由時模式不同）。**作答時用一句話寫明對支承的解讀**，是 40 分大題自保的關鍵。

3. **多自由度延伸**：若 A 端也改為側向彈簧支撐（或三桿兩鉸兩彈簧），則為 2 自由度系統，需寫成特徵值問題 $[K_e - P K_g]\{\delta\} = 0$，行列式為零求最小特徵值——本題型的研究所進階版。技師考至今多停留在單自由度，但 40 分配分顯示其重要性。

4. **與歐拉挫屈的觀念橋接**：歐拉柱可視為「無限多個微小彈簧分布」的連續極限；兩者的共同本質都是「幾何剛度抵銷彈性剛度」。理解這點，挫屈單元（MM-U3-4）的所有題型可用同一框架應對。
