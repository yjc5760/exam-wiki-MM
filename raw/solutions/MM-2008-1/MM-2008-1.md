## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

本題旨在測試考生對於**靜定梁受力平衡**、**力系等效簡化**以及**特殊構造（剛性懸臂架、內鉸）內力傳遞**的掌握程度。

- **圖一（a）的核心觀念：**
  測試「等效力系簡化」。作用在懸臂架端點 D 的垂直集中載重，傳遞到梁軸線 C 點時，必須等效為一個**垂直集中力**與一個**集中力偶矩**的共同作用。若考生直接將力轉移或忽視力偶矩，即無法得到正確解答。
- **圖二（b）的核心觀念：**
  測試「鉸接梁之段落分析」。鉸接處（內鉸 C）的彎矩必須為零，這使得結構可以拆解為兩段獨立的靜定桿件進行分析（先分析簡支梁段 CB，再將 C 點傳遞之反力轉移至懸臂段 AC）。

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

### 步驟化作戰計畫

#### 對於圖一（a）：
1. **等效載重簡化：** 將 D 點之 60 kN 垂直集中力簡化至梁軸線上的 C 點，求得 C 點之等效剪力 $P = 60 \text{ kN}$（向下）與等效力偶矩 $M_C = 60 \text{ kN} \times 2 \text{ m} = 120 \text{ kN}\cdot\text{m}$（順時針）。
2. **求支承反力：** 利用整體力矩平衡與垂直力平衡求出 $R_A$ 與 $R_B$。
3. **繪製 SFD 與 BMD：** 依據反力與等效載重，分段寫出剪力與彎矩方程式，繪製圖表。

#### 對於圖一（b）：
1. **結構拆解分析：** 自內鉸 C 點切開，將梁分為 AC 與 CB 兩段。CB 段為簡支梁（C 端可視為滾支承/鉸支承，B 端為滾支承），AC 段為固定端懸臂梁。
2. **分析 CB 段：** 計算 CB 段的支承反力 $R_B$ 與 C 點剪力 $V_C$。
3. **分析 AC 段：** 將 $V_C$ 作為集中力（向下）施加於 AC 段的 C 端，結合均佈載重 $q$，計算固定端 A 的支承反力 $R_A$ 與彎矩 $M_A$。
4. **分段繪製 SFD 與 BMD：** 結合兩段的剪力與彎矩方程式繪製圖形。

### 關鍵陷阱及應對策略

1. **力矩方向與正負號誤判（圖一 a）：**
   D 點的 60 kN 向下力相對於 C 點產生的是**順時針**力偶矩。在繪製 BMD 時，順時針集中力偶矩會使彎矩圖在 C 點發生突增（向上跳躍 $120 \text{ kN}\cdot\text{m}$）。必須注意極限值正負號的轉換。
2. **支承反力計算中的雙重計算（圖一 a）：**
   在對 A 點取力矩平衡求 $R_B$ 時，若已直接使用原力 60 kN 對 A 點的力矩臂（$6\text{ m} + 2\text{ m} = 8\text{ m}$），則**不可**再重複疊加 C 點的等效力偶矩 120 kN-m。兩者是等效替代關係，非累加關係。
3. **鉸接處內力傳遞方向（圖一 b）：**
   在簡支段 CB 求得 C 點受力（向上）後，轉移到 AC 段時，必須變為大小相等、方向相反的力（向下）。
4. **BMD 二次曲線極值位置（圖一 b）：**
   段落 CB 上的剪力會在中間某點降為零，該點對應彎矩的最大值（拋物線頂點），此極值必須在 BMD 中清晰標注。

### 最終目標
繪出圖一（a）與（b）桿件 AB 之剪力圖（SFD）與彎矩圖（BMD）。

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

#### 對於圖一（a）：
$$\text{Step 1: } M_C = P \cdot e$$
$$\text{Step 2: } \sum M_A = 0 \implies \boxed{R_B} \cdot L - P \cdot (L_1 + e) = 0 \quad \text{或} \quad \boxed{R_B} \cdot L - P \cdot L_1 - \boxed{M_C} = 0$$
$$\text{Step 3: } \sum F_y = 0 \implies \boxed{R_A} + \boxed{R_B} - P = 0$$
$$\text{Step 4: } M(x) = \boxed{R_A} x \quad (0 \le x < L_1)$$
$$\text{Step 5: } M(x) = \boxed{R_A} x + \boxed{M_C} - P(x - L_1) \quad (L_1 < x \le L)$$

#### 對於圖一（b）：
$$\text{Step 1: } \sum M_C^{\text{right}} = 0 \implies \boxed{R_B} \cdot L_2 - \frac{1}{2} q L_2^2 = 0$$
$$\text{Step 2: } \sum F_y^{\text{right}} = 0 \implies \boxed{V_C} + \boxed{R_B} - q L_2 = 0$$
$$\text{Step 3: } \sum F_y^{\text{left}} = 0 \implies \boxed{R_A} - q L_1 - \boxed{V_C} = 0$$
$$\text{Step 4: } \sum M_A = 0 \implies \boxed{M_A} - \frac{1}{2} q L_1^2 - \boxed{V_C} L_1 = 0$$
$$\text{Step 5: } M_{\max} = \frac{1}{2} \boxed{V_C} x'_0 - \frac{1}{2} q (x'_0)^2 \quad \text{其中 } x'_0 = \frac{\boxed{V_C}}{q} \text{ 爲剪力爲零點}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 4.1 求解圖一（a）之 SFD 與 BMD

#### 步驟 1：力系等效簡化
剛性懸臂架固定於 C 點（$x = 6 \text{ m}$），外力 $60 \text{ kN}$ 垂直向下作用於 D 點（$x = 8 \text{ m}$，距 C 點水平偏心距為 $2 \text{ m}$）。
將 D 點外力等效簡化至梁軸線上 C 點：
- 垂直向下集中力：
  $$P_C = 60 \text{ kN}$$
- 順時針集中力偶矩：
  $$M_C = 60 \text{ kN} \times 2 \text{ m} = 120 \text{ kN}\cdot\text{m} \quad (\circlearrowright)$$

#### 步驟 2：計算支承反力
設鉸支承 A 與滾支承 B 的向上反力分別為 $R_A$ 與 $R_B$。
對 A 點取力矩平衡：
$$\sum M_A = 0 \implies R_B \times 10 - P_C \times 6 - M_C = 0$$
$$R_B \times 10 - 60 \times 6 - 120 = 0$$
$$10 R_B = 360 + 120 = 480 \implies R_B = 48 \text{ kN} \quad (\uparrow)$$

對垂直方向取力平衡：
$$\sum F_y = 0 \implies R_A + R_B - P_C = 0$$
$$R_A + 48 - 60 = 0 \implies R_A = 12 \text{ kN} \quad (\uparrow)$$

#### 步驟 3：分段剪力與彎矩方程式

- **段落 AC ($0 \le x < 6 \text{ m}$):**
  $$V(x) = R_A = 12 \text{ kN}$$
  $$M(x) = R_A \cdot x = 12x \text{ kN}\cdot\text{m}$$
  - 於 $x = 0 \text{ m}$ (A 點)：$M(0) = 0$
  - 於 $x = 6^- \text{ m}$ (C 點左側)：$M(6^-) = 12 \times 6 = 72 \text{ kN}\cdot\text{m}$

- **段落 CB ($6 \text{ m} < x \le 10 \text{ m}$):**
  $$V(x) = R_A - P_C = 12 - 60 = -48 \text{ kN}$$
  $$M(x) = R_A \cdot x + M_C - P_C(x - 6) = 12x + 120 - 60(x - 6) = -48x + 480 \text{ kN}\cdot\text{m}$$
  - 亦可簡化為由右側算起：$M(x) = R_B \cdot (10 - x) = 48(10 - x) \text{ kN}\cdot\text{m}$
  - 於 $x = 6^+ \text{ m}$ (C 點右側)：$M(6^+) = 48 \times (10 - 6) = 192 \text{ kN}\cdot\text{m}$
  - 於 $x = 10 \text{ m}$ (B 點)：$M(10) = 48 \times 0 = 0$

#### 步驟 4：繪製剪力圖（SFD）與彎矩圖（BMD）
- **SFD:**
  - 自 A 點 ($x=0$) 開始，剪力由 $0$ 突升至 $+12 \text{ kN}$，保持常數至 C 點左側。
  - 在 C 點 ($x=6$)，因等效集中力 $60 \text{ kN}$ 向下，剪力突然下降至 $-48 \text{ kN}$。
  - 自 C 點至 B 點 ($x=10$) 保持 $-48 \text{ kN}$，於 B 點受反力 $+48 \text{ kN}$ 作用回到 $0$。
- **BMD:**
  - 自 A 點 ($x=0$) 的 $0$ 開始線性增加，斜率為 $+12$，至 C 點左側達到 $+72 \text{ kN}\cdot\text{m}$。
  - 在 C 點 ($x=6$)，受順時針集中力偶矩 $120 \text{ kN}\cdot\text{m}$ 作用，彎矩圖垂直向上跳躍 $120$，達到 $+192 \text{ kN}\cdot\text{m}$。
  - 自 C 點至 B 點 ($x=10$) 線性遞減，斜率為 $-48$，於 B 點剛好歸零。

### 4.2 求解圖一（b）之 SFD 與 BMD

#### 步驟 1：拆解結構分析簡支梁段 CB
內鉸 C 處彎矩為零 ($M_C = 0$)。將結構在 C 點拆開，優先分析右側簡支段 CB（長度 $3 \text{ m}$），其承受均佈載重 $10 \text{ kN/m}$。
對 C 點取力矩平衡：
$$\sum M_C^{\text{right}} = 0 \implies R_B \times 3 - q \times 3 \times 1.5 = 0$$
$$3 R_B = 10 \times 3 \times 1.5 = 45 \implies R_B = 15 \text{ kN} \quad (\uparrow)$$

對垂直方向取力平衡（求 C 點傳遞之剪力 $V_C$）：
$$\sum F_y^{\text{right}} = 0 \implies V_C + R_B - q \times 3 = 0$$
$$V_C + 15 - 30 = 0 \implies V_C = 15 \text{ kN}$$
（此處 $V_C = 15 \text{ kN}$ 指的是 AC 段在 C 點施加給 CB 段向上的支承力。根據作用力與反作用力，CB 段在 C 點施加給 AC 段一個**向下**的集中力 $V_C = 15 \text{ kN}$）。

#### 步驟 2：分析左側懸臂梁段 AC
AC 段（長度 $5 \text{ m}$）承受均佈載重 $10 \text{ kN/m}$，且在 C 點（$x = 5 \text{ m}$）承受來自右段的傳遞集中載重 $V_C = 15 \text{ kN}$（向下）。
設固定端 A 點的反力為 $R_A$（向上）與反力矩 $M_A$（逆時針）。
垂直力平衡：
$$\sum F_y = 0 \implies R_A - q \times 5 - V_C = 0$$
$$R_A = 10 \times 5 + 15 = 65 \text{ kN} \quad (\uparrow)$$

對 A 點取力矩平衡：
$$\sum M_A = 0 \implies M_A - q \times 5 \times 2.5 - V_C \times 5 = 0$$
$$M_A = 125 + 15 \times 5 = 125 + 75 = 200 \text{ kN}\cdot\text{m} \quad (\circlearrowleft)$$

#### 步驟 3：分段剪力與彎矩方程式

- **段落 AC ($0 \le x < 5 \text{ m}$):**
  $$V(x) = R_A - q x = 65 - 10x \text{ kN}$$
  $$M(x) = -M_A + R_A \cdot x - \frac{1}{2} q x^2 = -200 + 65x - 5x^2 \text{ kN}\cdot\text{m}$$
  - 於 $x = 0 \text{ m}$ (A 點)：$V(0) = 65 \text{ kN}$，$M(0) = -200 \text{ kN}\cdot\text{m}$
  - 於 $x = 5^- \text{ m}$ (C 點左側)：$V(5^-) = 65 - 50 = 15 \text{ kN}$，$M(5^-) = -200 + 65(5) - 5(25) = 0$

- **段落 CB ($5 \text{ m} < x \le 8 \text{ m}$):**
  設局部座標 $x' = x - 5$（自 C 點向右）：
  $$V(x') = V_C - q x' = 15 - 10x' \text{ kN}$$
  $$M(x') = V_C \cdot x' - \frac{1}{2} q (x')^2 = 15x' - 5(x')^2 \text{ kN}\cdot\text{m}$$
  - 於 $x' = 0 \text{ m}$ (C 點右側，即 $x = 5 \text{ m}$ )：$V = 15 \text{ kN}$，$M = 0$
  - 於 $x' = 3 \text{ m}$ (B 點，即 $x = 8 \text{ m}$ )：$V = 15 - 30 = -15 \text{ kN}$，$M = 15(3) - 5(9) = 0$
  - **最大彎矩位置（剪力為零處）：**
    $$V(x'_0) = 0 \implies 15 - 10x'_0 = 0 \implies x'_0 = 1.5 \text{ m} \quad (\text{即 } x = 6.5 \text{ m})$$
    $$M_{\max} = M(x'_0 = 1.5) = 15(1.5) - 5(1.5)^2 = 22.5 - 11.25 = 11.25 \text{ kN}\cdot\text{m}$$

#### 步驟 4：繪製剪力圖（SFD）與彎矩圖（BMD）
- **SFD:**
  - 自 A 點 ($x=0$) 開始，剪力由 $0$ 突升至 $+65 \text{ kN}$。
  - 自 A 到 C 呈線性遞減（斜率為 $-10$），至 C 點左側降至 $+15 \text{ kN}$。
  - 在 C 點 ($x=5$) 剪力圖連續，無突變跳躍（剪力仍為 $+15 \text{ kN}$）。
  - 自 C 到 B 繼續以斜率 $-10$ 線性遞減，於 $x = 6.5 \text{ m}$ 處剪力通過 $0$，在 B 點左側達到 $-15 \text{ kN}$。
  - 於 B 點受反力 $+15 \text{ kN}$ 作用回到 $0$。
- **BMD:**
  - 自 A 點 ($x=0$) 的 $-200 \text{ kN}\cdot\text{m}$ 開始，呈開口向下的拋物線。因為剪力一直為正值且減小，彎矩值朝正向增加，於 C 點 ($x=5$) 剛好達到 $0$。
  - 自 C 點 ($x=5$) 開始，拋物線繼續向正向攀升，在 $x = 6.5 \text{ m}$ 處達到最大值 $M_{\max} = 11.25 \text{ kN}\cdot\text{m}$。
  - 隨後彎矩值下降，至 B 點 ($x=8$) 剛好回到 $0$。整個拋物線的開口朝下，在 C 點斜率連續（剪力無突變）。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 關於剛性懸臂架的處理細節：
有些考生在簡化圖一（a）時，會誤以為 T 型剛架上的 60 kN 載重會在 C 點造成水平推力。因為 60 kN 力是垂直向下的，所以在 C 點只有垂直剪力與彎矩，無任何水平力分量。因此，支承 A 的水平反力 $H_A = 0$，這在繪製 SFD 與 BMD 時是預設成立的。

### 內鉸處的內力連續性：
在圖一（b）中，內鉸 C 兩側的剪力是連續的（皆為 $15 \text{ kN}$），這反映在 SFD 上是一條連續無跳躍的斜直線。若內鉸處有外加集中荷載，則剪力圖會發生突變；本題無外加集中力於 C 點，故剪力連續，彎矩圖在 C 點的斜率也是連續的（因為 $dM/dx = V$，C 點兩側 $V$ 相等，故斜率相等）。這是一個極佳的圖形檢驗指標。
