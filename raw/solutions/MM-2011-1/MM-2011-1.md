# 考題編號：MM-2011-1

**主分類：** `MM-U1-3` 應力及應變分析原理與應用  
**副分類：**（無）  
**分析法：** 彈性分析（斜截面應力、極值分析）  
**標籤：** `主應力` `斜截面全應力` `全應力與法線夾角` `極值問題` `Mohr應力圓` `平面應力`

---

## 1. 原始題目重述 (Problem Restatement)

已知平面主應力狀態 $\sigma_x > \sigma_y > 0$，如圖一所示（無剪應力，即已為主應力狀態）。斜截面的法線方向 $n_\alpha$ 與 $x$ 軸夾角 $\alpha$，截面上的全應力 $p_\alpha$ 與法線 $n_\alpha$ 夾角為 $\varphi$。

**試求 $\varphi$ 的最大值（$\varphi_{max}$）。**（25 分）

![MM-2011-1 題目附圖：圖一，左側為受主應力σx(水平)和σy(垂直)的應力單元體，斜截面法線n_α與x軸夾角α；右側放大圖顯示斜截面上的正應力σ_α（沿法線方向）、剪應力τ_α（沿截面方向）、全應力p_α（兩者合力），φ為p_α與法線n_α的夾角](MM-2011-1-fig-1.png)

*圖說：主應力狀態（$\sigma_x > \sigma_y > 0$，無剪應力）。法線 $n_\alpha$ 與 $x$ 軸夾角 $\alpha$，全應力 $p_\alpha$ 與法線方向夾角 $\varphi$（即 $\varphi$ = 全應力偏離法線的角度）。*

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
1. **斜截面應力公式**：在主應力狀態下，各角度截面的正應力和剪應力（由轉軸公式）
2. **全應力的方向**：全應力 $p = \sqrt{\sigma_\alpha^2 + \tau_\alpha^2}$，其方向與法線方向有偏角 $\varphi$
3. **極值分析**：對 $\varphi$ 取導數等於零，求極值角度 $\alpha^*$ 和對應的 $\varphi_{max}$

**出題者意圖：**
- 考驗學生對「全應力」（total stress，非單純的正應力或剪應力）概念的理解
- 測驗微分求極值的能力（tan φ 的最大值，而非 φ 本身）
- 結果可用 Mohr 應力圓的幾何直觀驗證

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**步驟化作戰計畫：**
1. 寫出斜截面正應力 $\sigma_\alpha$ 和剪應力 $\tau_\alpha$（主應力狀態公式）
2. 建立 $\tan\varphi = \tau_\alpha / \sigma_\alpha$（全應力與法線的夾角關係）
3. 對 $\alpha$ 求極值：令 $\frac{d(\tan\varphi)}{d\alpha} = 0$
4. 解出最優角度 $\alpha^*$，代回求 $\varphi_{max}$

**關鍵陷阱：**

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ⚠ φ ≠ 最大剪應力角 | 最大剪應力在 α=45° 處，但 φ_max 不在 45° | 嚴格用 d(tanφ)/dα=0 求解 |
| ⚠ 求最大 tanφ 而非最大 τ | tanφ=τ/σ，不是單純求 τ 或 σ 的極值 | 商函數導數 |
| ⚠ 最終結果要用 sin 形式 | arctan 形式可接受，但 sinφ_max = (σx-σy)/(σx+σy) 更簡潔 | 利用輔助三角 |

---

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
求 $\varphi_{max} = \arcsin\!\left(\dfrac{\sigma_x - \sigma_y}{\sigma_x + \sigma_y}\right)$

### 本題關鍵公式

$$\sigma_\alpha = \sigma_x\cos^2\alpha + \sigma_y\sin^2\alpha = \frac{\sigma_x+\sigma_y}{2} + \frac{\sigma_x-\sigma_y}{2}\cos 2\alpha$$

$$\tau_\alpha = \frac{\sigma_x-\sigma_y}{2}\sin 2\alpha$$

$$\tan\varphi = \frac{\tau_\alpha}{\sigma_\alpha} \xrightarrow{\text{令 }u=\tan\alpha} \frac{(\sigma_x-\sigma_y)u}{\sigma_x+\sigma_y u^2}$$

$$\frac{d(\tan\varphi)}{du} = 0 \implies u^* = \tan\alpha^* = \sqrt{\frac{\sigma_x}{\sigma_y}}$$

$$\boxed{\varphi_{max} = \arcsin\!\frac{\sigma_x-\sigma_y}{\sigma_x+\sigma_y}}$$

### L1：題目直接給定

| 符號 | 說明 |
|------|------|
| $\sigma_x$ | x 方向主應力（較大，正）|
| $\sigma_y$ | y 方向主應力（較小，正）|
| $\alpha$ | 斜截面法線與 x 軸夾角 |
| $\varphi$ | 全應力 $p_\alpha$ 與法線方向夾角 |

### L2：需知識點推導

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $\sigma_\alpha$ | 轉軸公式（主應力狀態） | |
| $\tau_\alpha$ | 轉軸公式（主應力狀態） | |
| $\tan\varphi$ | $\tau_\alpha/\sigma_\alpha$（幾何關係）| |
| $\alpha^*$（最優角）| $d(\tan\varphi)/d\alpha = 0$ 解 | |
| $\varphi_{max}$ | 代入 $\alpha^*$ 後化簡 | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| 商函數導數 = 0 的條件 | $(f/g)' = 0 \Rightarrow f'g - fg' = 0$（即分子 = 0）| |
| 換元 $u=\tan\alpha$ | 三角函數轉換為有理函數，便於求導 | |
| 輔助三角求 sin | 已知 tanφ 的分子分母，斜邊 = $\sqrt{分子²+分母²}$ | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 步驟 1：斜截面應力公式

已知主應力狀態（無剪應力），法線與 $x$ 軸夾角 $\alpha$ 的斜截面上：

$$\sigma_\alpha = \sigma_x\cos^2\alpha + \sigma_y\sin^2\alpha$$

$$\tau_\alpha = \frac{\sigma_x - \sigma_y}{2}\sin 2\alpha$$

### 步驟 2：全應力與法線的夾角關係

由幾何關係（全應力 $p_\alpha$ 分解為法向分量 $\sigma_\alpha$ 和切向分量 $\tau_\alpha$）：

$$\tan\varphi = \frac{\tau_\alpha}{\sigma_\alpha}$$

### 步驟 3：換元求極值

令 $u = \tan\alpha$（$u > 0$，$\alpha \in (0°, 90°)$），利用：

$$\sin^2\alpha = \frac{u^2}{1+u^2},\quad \cos^2\alpha = \frac{1}{1+u^2},\quad \sin 2\alpha = \frac{2u}{1+u^2}$$

代入：

$$\sigma_\alpha = \frac{\sigma_x + \sigma_y u^2}{1+u^2}, \quad \tau_\alpha = \frac{(\sigma_x-\sigma_y)u}{1+u^2}$$

$$\tan\varphi = \frac{\tau_\alpha}{\sigma_\alpha} = \frac{(\sigma_x-\sigma_y)u}{\sigma_x + \sigma_y u^2}$$

對 $u$ 求導並令分子 = 0：

$$\frac{d(\tan\varphi)}{du} = 0 \implies (\sigma_x-\sigma_y)(\sigma_x + \sigma_y u^2) - (\sigma_x-\sigma_y)u \cdot 2\sigma_y u = 0$$

因 $\sigma_x > \sigma_y$，$(\sigma_x - \sigma_y) \neq 0$，可約去：

$$\sigma_x + \sigma_y u^2 - 2\sigma_y u^2 = 0 \implies \sigma_x - \sigma_y u^2 = 0$$

$$\boxed{u^* = \tan\alpha^* = \sqrt{\frac{\sigma_x}{\sigma_y}}}$$

### 步驟 4：代入求 φ_max

代入 $u^* = \sqrt{\sigma_x/\sigma_y}$（即 $\sigma_y u^{*2} = \sigma_x$）：

$$\tan\varphi_{max} = \frac{(\sigma_x-\sigma_y)\cdot\sqrt{\sigma_x/\sigma_y}}{\sigma_x + \sigma_y\cdot(\sigma_x/\sigma_y)} = \frac{(\sigma_x-\sigma_y)\sqrt{\sigma_x/\sigma_y}}{2\sigma_x} = \frac{\sigma_x-\sigma_y}{2\sqrt{\sigma_x\sigma_y}}$$

**化為 sin 形式**（構造輔助直角三角形）：

- 對邊（分子）：$\sigma_x - \sigma_y$
- 鄰邊（分母）：$2\sqrt{\sigma_x\sigma_y}$
- 斜邊：$\sqrt{(\sigma_x-\sigma_y)^2 + 4\sigma_x\sigma_y} = \sqrt{(\sigma_x+\sigma_y)^2} = \sigma_x+\sigma_y$

$$\sin\varphi_{max} = \frac{\sigma_x-\sigma_y}{\sigma_x+\sigma_y}$$

$$\boxed{\varphi_{max} = \arcsin\frac{\sigma_x-\sigma_y}{\sigma_x+\sigma_y}}$$

**此時對應的截面角度 $\alpha^*$：**

$$\tan\alpha^* = \sqrt{\frac{\sigma_x}{\sigma_y}} \implies \alpha^* = \arctan\sqrt{\frac{\sigma_x}{\sigma_y}}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 Mohr 應力圓的幾何解釋

在 Mohr 圓中，全應力 $p_\alpha$ 對應圓上一點 $(\sigma_\alpha, \tau_\alpha)$，$\varphi$ 是該點到原點連線與 $\sigma$ 軸的夾角。$\varphi_{max}$ 是從原點向 Mohr 圓作切線時的切線角度。

切線條件：$OP^2 = OC^2 - r^2$（點 O 到圓心 C 的距離平方減半徑平方）

Mohr 圓圓心 $C = ((\sigma_x+\sigma_y)/2, 0)$，半徑 $r = (\sigma_x-\sigma_y)/2$。

$$\sin\varphi_{max} = \frac{r}{OC} = \frac{(\sigma_x-\sigma_y)/2}{(\sigma_x+\sigma_y)/2} = \frac{\sigma_x-\sigma_y}{\sigma_x+\sigma_y} \checkmark$$

此結果與代數推導完全一致，且物理意義直觀。

### 5.2 特殊情形驗算

- $\sigma_y = 0$（單軸應力）：$\sin\varphi_{max} = 1 \implies \varphi_{max} = 90°$（全應力方向垂直法線，即純剪）
  - 但此情形 $\sigma_x > 0, \sigma_y = 0$，在 α=45° 時 τ/σ 最大，這是對的 ✓
- $\sigma_y = \sigma_x$（等雙軸應力）：$\sin\varphi_{max} = 0 \implies \varphi_{max} = 0°$（全應力恆沿法線，無偏角）✓

### 5.3 $\varphi$ 與 $\tau$ 極值的區別

| 極值 | 對應 α | 條件 |
|------|--------|------|
| $\tau_{max}$ | $\alpha = 45°$ | $d\tau/d\alpha = 0$ |
| $\varphi_{max}$ | $\alpha^* = \arctan\sqrt{\sigma_x/\sigma_y}$ | $d(\tau/\sigma)/d\alpha = 0$ |

當 $\sigma_x = \sigma_y$ 時，$\alpha^* = 45° = \tau_{max}$ 的位置（特殊情形，此時 $\varphi_{max}=0$，不矛盾）。
