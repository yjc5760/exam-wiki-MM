### 考題編號：MM-2004-4

**主分類：** `MM-U3-2` 梁桿件變位及內力分析
**副分類：** （無）
**分析法：** 彈性分析
**標籤：** `彈簧支承梁` `靜不定梁` `彈力常數` `對稱性` `撓度` `彈簧反力` `直接積分法`

---

## 1. 題目

剛度為 $EI$ 之梁，由四條彈簧支撐，每條彈簧之彈力常數皆為 $k$，且 $k = EI/L^3$。在梁的中點受到集中載重 $P$ 作用（圖四）。求：

**(一)** 每條彈簧之支反力。（15 分）

**(二)** 集中載重 $P$ 作用處之撓度。（10 分）

---

*圖說：梁總長 $4L$，彈簧位置：外側 $x=0$ 及 $x=4L$，內側 $x=L$ 及 $x=3L$（$x$ 自左端起算）。集中載重 $P$ 在梁中點 $x=2L$。每條彈簧彈力常數 $k = EI/L^3$。梁剛度 $EI$，無其他支撐（純彈簧支撐，free-free 梁）。*

---

## 3. 解題戰略地圖

**三層掃描：**

| 層次 | 內容 |
|------|------|
| **[目標]** | 4 條彈簧的支反力 $R_a$（外側）、$R_b$（內側）；P 點撓度 $y_P$ |
| **[已知]** | $EI$、$k=EI/L^3$、梁總長 $4L$、彈簧位置 $x=0,L,3L,4L$、$P$ 在 $x=2L$ |
| **[待算]** | 利用對稱性化簡 → 一次靜不定 → 直接積分法求諧和條件 |

**靜不定度分析：**

- 4 個未知彈簧反力，1 個平衡方程式（$\sum F_y = P$）→ **三次靜不定**
- 利用**對稱性**（$P$ 在中點，結構對稱）：$R_a = R_4 = R_a$（外側），$R_b = R_2 = R_b$（內側）→ **一次靜不定**
- 需要 1 個諧和條件（相容方程式）

**方法：對稱半梁直接積分法**

取左半梁（$0 \le x \le 2L$），在對稱點 $x=2L$ 施加**零斜率**條件（對稱邊界），搭配兩個彈簧撓度諧和條件，求解 $R_a$、$R_b$。

**陷阱：**
1. 此梁**無固定端**，兩端均自由（彈簧是唯一支撐），不能用 $y(0)=0$ 或 $y'(0)=0$
2. BC 是：$y(0) = R_a/k$、$y(L) = R_b/k$（彈簧壓縮量 = 反力/彈簧常數）
3. 端點 $M(0) = 0$（自由端無彎矩），此條件由 $M(x) = R_a \cdot x$ 在 $x=0$ 處自動滿足

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
求外側彈簧反力 $R_a$、內側彈簧反力 $R_b$，以及中點撓度 $y_P$

### 本題關鍵公式（依計算順序）

$$\text{Step 1（平衡）: } 2R_a + 2R_b = P$$

$$\text{Step 2（積分，0≤x≤L）: } EI\cdot y = -\frac{R_a x^3}{6} + Ax + B$$

$$\text{Step 3（積分，L≤x≤2L）: } EI\cdot y = -\frac{R_a x^3}{6} - \frac{R_b(x-L)^3}{6} + Cx + D$$

$$\text{Step 4（對稱 BC）: } y'(2L) = 0 \Rightarrow C = L^2\!\left(2R_a + \frac{R_b}{2}\right)$$

$$\text{Step 5（彈簧諧和）: } y(0) = \frac{R_a}{k},\quad y(L) = \frac{R_b}{k}$$

$$\text{Step 6（代入 } k = EI/L^3\text{）: } R_a = \frac{3P}{40},\quad R_b = \frac{17P}{40}$$

### L1：題目直接給定

| 符號 | 說明 |
|------|------|
| $EI$ | 梁剛度 |
| $k = EI/L^3$ | 彈簧常數（已含 EI 的關係） |
| $P$ | 中點集中載重 |
| $L$ | 彈簧間距 |

### L2：需知識點推導

**Step 1–3：彎矩 + 積分**

| 符號 | 公式/來源 | 卡關? |
|------|----------|:-----:|
| $M(x)$ | 左半段對稱剩：$R_a \cdot x$（0到L）；$R_a x + R_b(x-L)$（L到2L） | |
| $EI \cdot y''$ | $= -M(x)$（下為正，撓曲符號） | |
| 積分常數 | $A = C$（連續性），$B = D$ | |

**Step 4–5：BC 求積分常數與 $R_a/R_b$ 比值**

| 符號 | 公式/來源 | 卡關? |
|------|----------|:-----:|
| $y'(2L) = 0$ | 對稱條件求 $C$ | |
| $y(0) = R_a/k$ | 外側彈簧諧和（撓度 = 力/彈簧常數） | |
| $y(L) = R_b/k$ | 內側彈簧諧和 | |
| $EI/k = L^3$ | $k = EI/L^3$ 的直接代入，消去 $EI$ | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|:-----:|
| Free-free 梁的 BC | 自由端無彎矩（$M=0$），非 $y=0$ 或 $y'=0$。彈簧支承代入 $y = R/k$ 而非 $y = 0$ | |
| 對稱邊界條件的使用 | 對稱軸處 $y' = 0$（斜率為零），而非 $y = 0$；對稱軸處的剪力不為零（有 $P/2$ 的貢獻） | |
| $k = EI/L^3$ 的意義 | 此條件使 $EI/k = L^3$，讓積分結果化簡為純數比例，避免留下 $EI$ 和 $k$ 的個別符號 | |

## 4. 步驟化詳細計算

### 前置：系統描述與對稱化

**彈簧位置與命名：**

$$x = 0 \text{（外側，} R_a\text{）},\quad x=L\text{（內側，}R_b\text{）},\quad x=3L\text{（內側，}R_b\text{）},\quad x=4L\text{（外側，}R_a\text{）}$$

**平衡方程式：**

$$2R_a + 2R_b = P \implies R_a + R_b = \frac{P}{2} \tag{1}$$

**分析左半梁（$0 \le x \le 2L$），自由端 $x=0$ 起，以 $y$ 向下為正：**

$$EI \cdot y'' = -M(x) \quad (\text{彎矩正 = 下凸，}y\text{ 向下為正})$$

彎矩（以左側合力取矩）：

$$M(x) = R_a \cdot x \quad (0 \le x \le L)$$

$$M(x) = R_a \cdot x + R_b(x - L) \quad (L \le x \le 2L)$$

---

### 小題（一）：求各彈簧支反力

**第一段積分（$0 \le x \le L$）：**

$$EI \cdot y'' = -R_a x$$

$$EI \cdot y' = -\frac{R_a x^2}{2} + A$$

$$EI \cdot y = -\frac{R_a x^3}{6} + Ax + B \tag{2}$$

**第二段積分（$L \le x \le 2L$）：**

$$EI \cdot y'' = -R_a x - R_b(x-L)$$

$$EI \cdot y' = -\frac{R_a x^2}{2} - \frac{R_b(x-L)^2}{2} + C$$

$$EI \cdot y = -\frac{R_a x^3}{6} - \frac{R_b(x-L)^3}{6} + Cx + D \tag{3}$$

**連續性條件（$x = L$）：**

$$y'_{\text{left}} = y'_{\text{right}} \implies A = C$$

$$y_{\text{left}} = y_{\text{right}} \implies B = D$$

**BC1：對稱條件 $y'(2L) = 0$（代入式(3)）：**

$$0 = -\frac{R_a(2L)^2}{2} - \frac{R_b \cdot L^2}{2} + C$$

$$C = 2R_a L^2 + \frac{R_b L^2}{2} = L^2\!\left(2R_a + \frac{R_b}{2}\right) \tag{4}$$

故 $A = C = L^2(2R_a + R_b/2)$。

**BC2：外側彈簧諧和 $y(0) = R_a/k$（代入式(2)）：**

$$EI \cdot y(0) = B = \frac{EI \cdot R_a}{k} = R_a L^3 \quad \left(\text{用 } \frac{EI}{k} = L^3\right) \tag{5}$$

故 $B = R_a L^3$。

**BC3：內側彈簧諧和 $y(L) = R_b/k$（代入式(2)）：**

$$EI \cdot y(L) = -\frac{R_a L^3}{6} + A \cdot L + B = \frac{EI \cdot R_b}{k} = R_b L^3$$

代入 $A = L^2(2R_a + R_b/2)$ 及 $B = R_a L^3$：

$$-\frac{R_a L^3}{6} + L^3\!\left(2R_a + \frac{R_b}{2}\right) + R_a L^3 = R_b L^3$$

除以 $L^3$：

$$-\frac{R_a}{6} + 2R_a + \frac{R_b}{2} + R_a = R_b$$

$$R_a\!\left(-\frac{1}{6} + 2 + 1\right) + \frac{R_b}{2} = R_b$$

$$R_a \cdot \frac{17}{6} = R_b - \frac{R_b}{2} = \frac{R_b}{2}$$

$$R_a = \frac{R_b}{2} \cdot \frac{6}{17} = \frac{3R_b}{17} \tag{6}$$

**代入平衡式(1)：**

$$\frac{3R_b}{17} + R_b = \frac{P}{2} \implies R_b \cdot \frac{20}{17} = \frac{P}{2} \implies \boxed{R_b = \frac{17P}{40}}$$

$$\boxed{R_a = \frac{P}{2} - R_b = \frac{P}{2} - \frac{17P}{40} = \frac{3P}{40}}$$

**各彈簧支反力整理：**

| 位置 | $x = 0$ | $x = L$ | $x = 3L$ | $x = 4L$ |
|------|:---:|:---:|:---:|:---:|
| 支反力 | $R_a = \dfrac{3P}{40}$ | $R_b = \dfrac{17P}{40}$ | $R_b = \dfrac{17P}{40}$ | $R_a = \dfrac{3P}{40}$ |

**驗證：** $2R_a + 2R_b = 2 \cdot \dfrac{3P}{40} + 2 \cdot \dfrac{17P}{40} = \dfrac{6P+34P}{40} = P$ ✅

---

### 小題（二）：P 點撓度（$x = 2L$）

代入式(3)，取 $x = 2L$：

$$EI \cdot y(2L) = -\frac{R_a(2L)^3}{6} - \frac{R_b \cdot L^3}{6} + C \cdot 2L + D$$

其中 $C = L^2(2R_a + R_b/2)$、$D = R_a L^3$：

$$= L^3\!\left(-\frac{4R_a}{3} - \frac{R_b}{6} + 4R_a + R_b + R_a\right) = L^3\!\left[\frac{11R_a}{3} + \frac{5R_b}{6}\right]$$

代入 $R_a = 3P/40$、$R_b = 17P/40$：

$$= L^3\!\left[\frac{11}{3} \cdot \frac{3P}{40} + \frac{5}{6} \cdot \frac{17P}{40}\right] = L^3\!\left[\frac{11P}{40} + \frac{85P}{240}\right]$$

$$= L^3\!\left[\frac{66P}{240} + \frac{85P}{240}\right] = \frac{151PL^3}{240}$$

$$\boxed{y_P = \frac{151PL^3}{240\,EI}}$$

---

## 5. 最終答案彙整

| 項目 | 答案 |
|------|------|
| 外側彈簧反力（$x=0, 4L$） | $R_a = \dfrac{3P}{40}$（各） |
| 內側彈簧反力（$x=L, 3L$） | $R_b = \dfrac{17P}{40}$（各） |
| P 點撓度（$x=2L$） | $y_P = \dfrac{151PL^3}{240\,EI}$（向下） |

**彈簧撓度驗核：**

$$y(0) = \frac{R_a}{k} = \frac{3P/40}{EI/L^3} = \frac{3PL^3}{40\,EI}$$

$$y(L) = \frac{R_b}{k} = \frac{17P/40}{EI/L^3} = \frac{17PL^3}{40\,EI}$$

$$y(2L) = \frac{151PL^3}{240\,EI}$$

比較：$y(0) < y(L) < y(2L)$（撓度自外側彈簧向中點遞增）✅

---

## 6. 核心觀念

**本題的靜不定求解結構：**

利用對稱性，4次靜不定化為**1次靜不定**：
1. 對稱條件 $y'(2L) = 0$ → 求積分常數 $C$
2. 外側彈簧諧和 $y(0) = R_a/k$ → 求積分常數 $B$
3. 內側彈簧諧和 $y(L) = R_b/k$ → 求 $R_a/R_b$ 比值
4. 平衡 $R_a + R_b = P/2$ → 求絕對值

**$k = EI/L^3$ 的作用：** 使得 $EI/k = L^3$，讓彈簧撓度（$R/k = RL^3/EI$）與梁撓度（$\propto RL^3/EI$）具有相同量綱結構，從而消去 $EI$ 和 $k$ 的個別數值，讓比例 $R_a/R_b = 3/17$ 成為純數。

**物理意義：** 內側彈簧（距 P 為 $L$）承擔的載重（$17P/40$）遠大於外側彈簧（距 P 為 $2L$）的 $3P/40$，比值 $17:3$，反映梁的彎曲剛度將載重優先傳遞給較近的彈簧支撐。
