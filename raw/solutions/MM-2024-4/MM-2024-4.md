# 考題編號：MM-2024-4

**主分類：** `MM-U4-1` 軸力桿件、扭力桿件與梁之塑性分析
**分析法：** 塑性分析
**標籤：** `理想彈塑性` `塑性極限載重` `纜繩系統` `剛性梁` `降伏順序` `應力重分配` `極限分析`

---

## 1. 原始題目重述 (Problem Restatement)

如圖所示，四條纜繩共同吊著一支水平梁，梁中點受一向下力 $P$ 作用：

- 四條纜繩具有**相同斷面積 $A$** 與**相同降伏應力 $\sigma_y$**；
- 纜繩材料為**理想彈塑性**（elastic-perfectly plastic）；
- 水平梁為**剛性**。

依圖：兩條**鉛直纜繩**分別吊於梁之左右兩端（圖中以直角記號標示垂直），兩條**斜纜繩**自天花板錨點拉至梁中點，與梁（水平）夾角各為 $60°$，$P$ 作用於梁中點。

試求該系統所能承受之最大外力 $P_{max}$（以 $A\sigma_y$ 表示）。（20 分）

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

本題是**塑性極限分析（limit analysis）**的招牌題型：靜不定纜繩系統的「逐繩降伏 → 應力重分配 → 全面降伏即崩塌」。出題者意圖：

- **極限載重只需平衡條件**：當所有纜繩都到達 $\sigma_y$，由力平衡直接得 $P_{max}$——題目連 $E$ 都不給，就是在暗示「最終答案與彈性性質無關」；
- **降伏順序的判斷**（等變形源系統）：剛性梁下移 δ 時，垂直繩應變 $\delta/h$、斜繩應變 $\delta\sin^2 60°/h$——垂直繩應變較大、先降伏。考生須能說明「誰先降伏、為何系統還能繼續承載」；
- **理想彈塑性的角色**：先降伏的繩力鎖定在 $A\sigma_y$ 不再增加，增量載重由未降伏的斜繩承接，直到全部降伏形成機構；
- 對比觀念：彈性設計（首繩降伏）$P_y$ vs 塑性極限 $P_{max}$，兩者之比即儲備強度。

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

1. 幾何與對稱性：梁剛性、左右對稱、P 在中點 ⇒ 梁保持水平、整體下移 δ（單自由度）。
2. 運動學：垂直繩伸長 $\delta$；斜繩伸長 $\delta\sin 60°$，且斜繩較長（$h/\sin 60°$）⇒ 應變比 $\varepsilon_{斜}/\varepsilon_{垂} = \sin^2 60° = 3/4$。
3. 彈性階段平衡 → 首繩降伏（垂直繩）→ $P_y$。
4. 重分配階段：垂直繩力鎖定 $A\sigma_y$，斜繩續增至 $\sigma_y$。
5. 極限狀態：四繩皆 $A\sigma_y$，鉛直平衡 ⇒ $P_{max}$。

**關鍵陷阱：**

| # | 陷阱 | 應對策略 |
|---|------|---------|
| 1 | 誤判降伏順序（以為斜繩受力大先降伏） | 等位移源系統比**應變**：$\varepsilon_{斜} = 0.75\,\varepsilon_{垂}$ ⇒ 垂直繩先降伏 |
| 2 | 斜繩伸長量寫成 $\delta$ | 伸長 = 位移在繩軸向之投影 = $\delta\sin 60°$ |
| 3 | 斜繩應變漏除以「較長的繩長」 | $L_{斜} = h/\sin 60°$ ⇒ 應變多除一個 $\sin 60°$，合計 $\sin^2 60°$ |
| 4 | 極限平衡忘記斜繩只貢獻鉛直分量 | $P_{max} = 2A\sigma_y + 2A\sigma_y\sin 60°$ |
| 5 | 以為缺 E、缺尺寸無法解 | 極限載重 = 純平衡問題；E、h 只影響「過程」不影響「終點」 |

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`求系統塑性極限載重 Pmax（以 Aσy 表示）`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: } \varepsilon_{垂} = \frac{\delta}{h},\qquad \varepsilon_{斜} = \frac{\delta\sin 60°}{h/\sin 60°} = \frac{\delta\sin^2 60°}{h} = \frac{3}{4}\,\boxed{\varepsilon_{垂}}$$

$$\text{Step 2（彈性平衡）: } P = 2F_{垂} + 2F_{斜}\sin 60°,\qquad F_{斜} = \frac{3}{4}F_{垂}$$

$$\text{Step 3（首繩降伏）: } P_y = \left(2 + \frac{3\sqrt{3}}{4}\right)A\sigma_y \approx 3.30\,A\sigma_y$$

$$\text{Step 4（極限狀態）: } P_{max} = 2A\sigma_y + 2A\sigma_y\sin 60° = (2+\sqrt{3})\,A\sigma_y$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

**設天花板至梁的鉛直高度為 $h$（消去項，不影響答案）；梁剛性且對稱 ⇒ 梁保持水平、垂直下移 δ。**

### Step 1：運動學——各繩應變

- **垂直繩（×2，梁端）**：伸長 $= \delta$，繩長 $= h$：
$$\varepsilon_{垂} = \frac{\delta}{h}$$
- **斜繩（×2，梁中點，與水平夾 60°）**：伸長 = 位移沿繩向投影 $= \delta\sin 60°$；繩長 $= h/\sin 60°$：
$$\varepsilon_{斜} = \frac{\delta\sin 60°}{h/\sin 60°} = \frac{\delta\sin^2 60°}{h} = \frac{3}{4}\cdot\frac{\delta}{h}$$

$$\Rightarrow\ \varepsilon_{斜} = \frac{3}{4}\,\varepsilon_{垂} \;\Rightarrow\; \text{垂直繩應變恆較大 ⇒ \textbf{垂直繩先降伏}}$$

### Step 2：彈性階段平衡

同 $E$、同 $A$ ⇒ $F_{斜} = \dfrac{3}{4}F_{垂}$。剛性梁整體之鉛直平衡（斜繩取鉛直分量）：

$$P = 2F_{垂} + 2F_{斜}\sin 60° = 2F_{垂} + 2\left(\frac{3}{4}F_{垂}\right)\frac{\sqrt{3}}{2} = \left(2 + \frac{3\sqrt{3}}{4}\right)F_{垂}$$

### Step 3：首繩降伏（彈性極限）

垂直繩先達 $\sigma_y$，即 $F_{垂} = A\sigma_y$（此時 $\sigma_{斜} = 0.75\,\sigma_y$，仍彈性）：

$$P_y = \left(2 + \frac{3\sqrt{3}}{4}\right)A\sigma_y \approx 3.30\,A\sigma_y$$

> **策略註解：** $P > P_y$ 後，垂直繩（理想彈塑性）繩力鎖定 $A\sigma_y$，**增量載重全部由兩條斜繩承接**——系統仍穩定，但勁度下降。

### Step 4：塑性極限狀態

繼續加載至斜繩亦降伏（$F_{斜} = A\sigma_y$）。此時四繩全數降伏，梁可無限下移（崩塌機構），鉛直平衡給出極限載重：

$$P_{max} = 2(A\sigma_y) + 2(A\sigma_y)\sin 60° = 2A\sigma_y + \sqrt{3}\,A\sigma_y$$

$$\boxed{P_{max} = (2+\sqrt{3})\,A\sigma_y \approx 3.73\,A\sigma_y}$$

**運動學檢核**：極限狀態之增量位移模式（梁續下移）使全部四繩持續伸長 ⇒ 塑性功皆為正，機構成立 ✓；強度儲備 $P_{max}/P_y = 3.732/3.299 = 1.13$。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

1. **水平平衡自動滿足**：兩斜繩水平分量 $A\sigma_y\cos 60°$ 左右對稱抵銷，垂直繩無水平分量——對稱構形使梁無水平漂移。若幾何不對稱（如僅單側斜繩），需另檢核水平平衡與梁的轉動，難度大增。

2. **梁自重與彎矩**：題目未給梁重（忽略）；又因梁為剛性，梁內彎矩非本題標的。注意垂直繩在梁端、P 與斜繩在中點——若梁為彈性梁，端吊中載會產生彎曲，題目用「剛性」一詞把問題鎖定在纜繩系統。

3. **「最大外力」的解讀**：理想彈塑性系統的 $P_{max}$ 即塑性極限載重（崩塌載重）。若題目改問「彈性範圍內最大載重」，答案則為 $P_y \approx 3.30\,A\sigma_y$。本解兩者皆列，考場上以 $P_{max} = (2+\sqrt{3})A\sigma_y$ 作答並附 $P_y$ 對照，最為完整。

4. **卸載與殘留力延伸**：自 $P_{max}$ 附近卸載至零，垂直繩有塑性伸長，系統將出現**自平衡殘留力系**（斜繩殘拉、垂直繩可能鬆弛或殘壓不可（繩不能受壓）——繩系卸載需檢查「繩鬆弛」條件）。「纜繩不能承壓」是本題型卸載延伸的獨特陷阱，與桿系不同。
