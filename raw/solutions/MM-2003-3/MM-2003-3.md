# 考題編號：MM-2003-3

**主分類：** `MM-U3-2` 梁桿件變位及內力分析  
**副分類：** `MM-U2-2` 梁桿件斷面應力計算（沿用第二題梁）  
**分析法：** 彈性分析  
**標籤：** `梁變位` `轉角` `撓度` `二重積分法` `靜定梁` `懸臂段` `彎矩圖積分`

## 1. 原始題目重述 (Problem Restatement)

承 MM-2003-2，梁幾何、載重、斷面、材料常數完全相同（見圖二），求：

1. **B 點之轉角** $\theta_B$
2. **C 點之垂直變位** $\delta_{yC}$

**已知（承第二題）：**

| 量 | 值 |
|----|-----|
| $\lambda$ | 40 in |
| $E$ | $30 \times 10^6\ \text{psi}$ |
| $I$ | $206/3 \approx 68.67\ \text{in}^4$ |
| $EI$ | $2{,}060 \times 10^6\ \text{lb·in}^2$ |
| $R_A$ | 2,250 lb（↑） |
| $R_B$ | 3,750 lb（↑） |
| $M(A) = M_A$ | $-60{,}000\ \text{lb·in}$ |
| $M(C) = M_C$ | $-90{,}000\ \text{lb·in}$ |
| $M(B) = M_B$ | $-120{,}000\ \text{lb·in}$ |

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
AB 主跨範圍無外加載重，M 圖為線性（已由第二題求出），可直接對 $EI v'' = M(x)$ 二重積分，利用兩端支承 $v(A)=v(B)=0$ 定積分常數，再求指定位置的斜率與撓度。

**出題者意圖：**
- 前後兩題形成完整的「應力→變形」分析鏈
- 考核二重積分法的熟練度：M(x) 已知 → 積分兩次 → 邊界條件定常數
- 驗算：B 和 E 之間也可用懸臂段能量法或疊加法交叉確認

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 設 $\xi$ 為從 A 向右量起的座標（$\xi = x - \lambda$，$0 \le \xi \le 2\lambda$）
2. 在 AB 段寫 $M(\xi) = -60{,}000 - 750\xi$（線性，由第二題直接帶入）
3. 二重積分 $EI v'' = M(\xi)$
4. 代入 $v(0)=0$（A）、$v(2\lambda)=0$（B）求積分常數 $C_1$、$C_2$
5. 代入 $\xi = 2\lambda$ 求 $\theta_B = v'(2\lambda)$
6. 代入 $\xi = \lambda$ 求 $\delta_{yC} = v(\lambda)$

**關鍵陷阱：**

| # | 陷阱 | 應對 |
|---|------|------|
| 1 | **對全梁積分（含懸臂段）**：D～A 段 M 為拋物線，若要整段積分需分段，邊界條件變複雜 | 只取 AB 主跨積分，兩端已知支承條件 $v=0$ |
| 2 | **混淆 $\xi$ 與 $x$ 的 M 方程式**：若帶入以 D 為原點的 M(x) 公式，$C_2$ 不會為零 | 換座標至 A 為原點，令 $\xi = x - \lambda$ |
| 3 | **EI 的單位**：E 用 psi（lb/in²），I 用 in⁴，EI 單位為 lb·in²，撓度單位為 in，轉角為 rad | 全程保持英制 |
| 4 | **正負號**：M < 0 → 梁向上彎（hogging）→ 主跨中段 $v > 0$（向上撓曲）；θ_B 為負（順時針） | 物理上用 hogging 梁向上拱理解正負 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標

B 點轉角 $\theta_B$（rad）及 C 點垂直撓度 $\delta_{yC}$（in）。

### 本題關鍵公式（依計算順序）

$$\text{Step 1: } EI v'' = M(\xi) = -60{,}000 - 750\xi$$

$$\text{Step 2: } EI v' = -60{,}000\xi - 375\xi^2 + C_1$$

$$\text{Step 3: } EI v = -30{,}000\xi^2 - 125\xi^3 + C_1\xi + C_2$$

$$\text{Step 4（邊界）: } v(0)=0 \Rightarrow C_2=0;\quad v(2\lambda)=0 \Rightarrow C_1 = 60{,}000\lambda + 500\lambda^2$$

$$\text{Step 5: } \theta_B = v'(2\lambda) = \frac{-\lambda(60{,}000 + 1{,}000\lambda)}{\boxed{EI}}$$

$$\text{Step 6: } \delta_{yC} = v(\lambda) = \frac{\lambda^2(30{,}000 + 375\lambda)}{\boxed{EI}}$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $\lambda$ | 40 in | 各段長度 |
| $E$ | $30 \times 10^6\ \text{psi}$ | 彈性模數 |
| $I$ | $206/3\ \text{in}^4$ | 由第二題算出 |
| $M_A$ | $-60{,}000\ \text{lb·in}$ | 由第二題算出 |
| $V_{AB}$ | $-750\ \text{lb}$ | 由第二題算出 |

### L2：需知識點推導

**【坐標轉換與 M(ξ) 建立】**

| 量 | 推導 | 卡關? |
|----|------|-------|
| 局部座標 | $\xi = x - \lambda$，$0 \le \xi \le 2\lambda$ | |
| $M(\xi)$ | $M_A + V_{AB} \cdot \xi = -60{,}000 - 750\xi$ | |

**【積分常數】**

| 量 | 條件 | 結果 | 卡關? |
|----|------|------|-------|
| $C_2$ | $v(0) = 0$ | $C_2 = 0$ | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：建立積分座標與 M(ξ)

取 A 為原點，$\xi$ 向右為正，$0 \le \xi \le 2\lambda = 80\ \text{in}$。

AB 段無集中載重或均布載重，$M(\xi)$ 直接由第二題結果帶入（線性）：

$$M(\xi) = M_A + V_{AB} \cdot \xi = -60{,}000 - 750\xi \quad \text{(lb·in)}$$

> **策略註解：** 只需在 AB 主跨上做積分，不需考慮懸臂段，因為支承 $v(A)=v(B)=0$ 已自動包含懸臂段對主跨的影響（透過彎矩圖）。

---

### Step 2：二重積分

**一次積分（求斜率）：**

$$EI v'(\xi) = \int M(\xi)\,d\xi = -60{,}000\xi - 375\xi^2 + C_1$$

**二次積分（求撓度）：**

$$EI v(\xi) = -30{,}000\xi^2 - 125\xi^3 + C_1\xi + C_2$$

---

### Step 3：代入邊界條件求 $C_1$、$C_2$

**B.C. 1：** $v(0) = 0$（A 為支承，無沉陷）

$$EI \cdot 0 = 0 + 0 + C_1(0) + C_2 \quad \Rightarrow \quad C_2 = 0$$

**B.C. 2：** $v(2\lambda) = v(80) = 0$（B 為支承，無沉陷）

$$0 = -30{,}000(80)^2 - 125(80)^3 + 80C_1$$

$$0 = -30{,}000 \times 6{,}400 - 125 \times 512{,}000 + 80C_1$$

$$0 = -192{,}000{,}000 - 64{,}000{,}000 + 80C_1$$

$$80C_1 = 256{,}000{,}000$$

$$\boxed{C_1 = 3{,}200{,}000\ \text{lb·in}^2}$$

> **代數驗算（符號化）：**
> $$C_1 = \frac{30{,}000(2\lambda)^2 + 125(2\lambda)^3}{2\lambda} = 60{,}000\lambda + 500\lambda^2 = 60{,}000(40) + 500(1{,}600) = 3{,}200{,}000 \checkmark$$

---

### Step 4：求彈性模數 × 慣性矩（EI）

$$EI = 30 \times 10^6 \times \frac{206}{3} = \frac{30 \times 206}{3} \times 10^6 = 2{,}060 \times 10^6\ \text{lb·in}^2$$

---

### Step 5：求 B 點轉角 $\theta_B$

代入 $\xi = 2\lambda = 80\ \text{in}$：

$$EI\,\theta_B = EI\,v'(80) = -60{,}000(80) - 375(80)^2 + 3{,}200{,}000$$

$$= -4{,}800{,}000 - 2{,}400{,}000 + 3{,}200{,}000$$

$$= -4{,}000{,}000\ \text{lb·in}^2$$

$$\theta_B = \frac{-4{,}000{,}000}{2{,}060 \times 10^6}$$

$$\boxed{\theta_B = -1.942 \times 10^{-3}\ \text{rad（順時針，梁在 B 點向右下傾斜）}}$$

> **物理解釋：** 負號代表在 B 點斜率向右下降，即 BE 懸臂段向下垂（E 端集中載重 P 拉低右端）。

---

### Step 6：求 C 點垂直變位 $\delta_{yC}$

代入 $\xi = \lambda = 40\ \text{in}$：

$$EI\,\delta_{yC} = EI\,v(40) = -30{,}000(40)^2 - 125(40)^3 + 3{,}200{,}000(40)$$

$$= -30{,}000 \times 1{,}600 - 125 \times 64{,}000 + 128{,}000{,}000$$

$$= -48{,}000{,}000 - 8{,}000{,}000 + 128{,}000{,}000$$

$$= 72{,}000{,}000\ \text{lb·in}^3$$

$$\delta_{yC} = \frac{72{,}000{,}000}{2{,}060 \times 10^6}$$

$$\boxed{\delta_{yC} = +0.03495\ \text{in} \approx 0.0350\ \text{in（向上，↑）}}$$

> **物理解釋：** 正號代表 C 點向上撓曲，符合 AB 主跨全段負彎矩（hogging）造成中段向上拱起的物理預期。

---

### Step 7：代數驗算

$$\delta_{yC} = \frac{\lambda^2(30{,}000 + 375\lambda)}{EI} = \frac{(40)^2(30{,}000 + 15{,}000)}{2{,}060 \times 10^6} = \frac{1{,}600 \times 45{,}000}{2{,}060 \times 10^6} = \frac{72{,}000{,}000}{2{,}060 \times 10^6} \checkmark$$

---

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 為何取 AB 段積分即可？

懸臂段 DA 的均布載重和 BE 的集中載重，已透過支承反力 $R_A$、$R_B$ 及彎矩方程式 $M(\xi)$ 完整地「傳遞」到 AB 主跨的積分中。取主跨積分只要正確使用兩端支承的 $v=0$ 邊界條件，就能正確求得全梁任意點的撓度。

### 交叉驗算：懸臂段端點撓度

BE 段（自由懸臂）的撓度可用另一種方式推導：
- B 點轉角 $\theta_B = -1.942 \times 10^{-3}\ \text{rad}$（已知）
- BE 段無彎矩（只有 P 在 E），BE 段是以 B 為固定端的懸臂梁

E 點撓度（相對於 B）：
$$\delta_E = \theta_B \cdot \lambda + \frac{P\lambda^3}{3EI}$$

（第一項為 B 點轉角引起的剛體位移，第二項為 BE 懸臂彎曲）

$$= (-1.942 \times 10^{-3})(40) + \frac{3{,}000 \times (40)^3}{3 \times 2{,}060 \times 10^6}$$

$$= -0.07767 + \frac{192{,}000{,}000}{6{,}180 \times 10^6}$$

$$= -0.07767 + 0.03107 = -0.04660\ \text{in（向下）}$$

> 此為 E 點相對於 B 點的額外撓度。因 B 本身無支承沉陷（$\delta_B = 0$），E 點總撓度約 0.047 in 向下，方向合理（P 向下施加於 E）。
