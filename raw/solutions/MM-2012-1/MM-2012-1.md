## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
1. **對稱靜定桁架**：由對稱性，兩桿受力相同，可用靜力方程直接求軸力
2. **雙線性 σ-ε 材料**：需分兩階段討論（彈性 + 超過降伏後），各階段 δ-P 關係形式不同
3. **幾何關係**：桿的軸向伸長 $\delta_{rod}$ 與節點垂直位移 $\delta$ 的幾何轉換（小變形假設）

**出題者意圖：**
- 測驗雙線性材料（bilinear）的分段應用，區分「降伏前」與「降伏後」兩個解
- 測驗靜定對稱桁架的靜力分析（不需繁複方程）
- 禁止計算器 → 答案為純符號式，考驗代數整理能力

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**步驟化作戰計畫：**
1. 用對稱性和靜力平衡求各桿軸力 F 和應力 σ（用 P、θ 表示）
2. 判斷降伏條件：$\sigma \leq \sigma_0$（Phase 1）或 $\sigma > \sigma_0$（Phase 2）
3. 各階段用 σ-ε 關係求 ε，再由幾何關係求節點位移 δ
4. 整理為 δ-P 的分段公式

**關鍵陷阱：**

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ⚠ 幾何轉換方向 | 桿的伸長方向沿桿軸，不等於節點垂直位移；需用 sinθ 轉換 | $\delta_{rod} = \delta \cdot \sin\theta$（小變形）|
| ⚠ 雙線性兩段都要給 | Phase 1 和 Phase 2 都必須寫，漏一段扣分 | 以  = 2A\sigma_0\sin\theta$ 為臨界點分段 |
| ⚠ E₂ 的意義 | E₂ 是「降伏後」的斜率（切線模數），不是 E₁ 的替代 | ε = σ₀/E₁ + (σ-σ₀)/E₂ |
| ⚠ 桿受拉還是受壓 | 由圖，B 在下方受 P，桿 AB/CB 受**拉力**（被向下拉伸）| 確認為拉力（σ > 0），降伏條件正確 |

\text{Step 1（靜力）：} \quad 2F\sin\theta = P \implies F = \frac{P}{2\sin\theta}

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 步驟 1：靜力分析（求桿軸力 F）

由圖一，B 點為自由節點，受垂直力 P 向下。兩桿 AB、CB 對稱，各與水平夾角 θ，方向指向 A 和 C（上方）。

對 B 點進行靜力平衡：

**水平方向：** {AB}\cos\theta = F_{CB}\cos\theta$（自動滿足，兩桿對稱）

**垂直方向：**
F_{AB}\sin\theta + F_{CB}\sin\theta = P

由對稱性 {AB} = F_{CB} = F$：

2F\sin\theta = P \implies \boxed{F = \frac{P}{2\sin\theta}} \quad \text{（拉力）}

> **策略註解：** 從圖中確認兩桿受**拉力**（B 往下，桿被拉長），σ > 0。降伏條件為 σ 達到 σ₀（拉伸降伏）。

### 步驟 2：桿內應力

\sigma = \frac{F}{A} = \frac{P}{2A\sin\theta}

**降伏臨界載重 $：**（令 σ = σ₀）

\sigma_0 = \frac{P_0}{2A\sin\theta} \implies P_0 = 2A\sigma_0\sin\theta

### 步驟 3：幾何關係（桿伸長 → 節點位移）

小變形假設下，B 點垂直下移 δ，桿 AB 的伸長量（軸向）為：

\delta_{rod} = \delta\sin\theta

因此：

\delta = \frac{\delta_{rod}}{\sin\theta} = \frac{\varepsilon L}{\sin\theta}

### 步驟 4：Phase 1（彈性， \leq P_0$，即 $\sigma \leq \sigma_0$）

雙線性材料彈性段：$\varepsilon = \sigma/E_1$

\varepsilon = \frac{\sigma}{E_1} = \frac{P}{2AE_1\sin\theta}

\delta = \frac{\varepsilon L}{\sin\theta} = \frac{PL}{2AE_1\sin^2\theta}

\boxed{\delta = \frac{PL}{2AE_1\sin^2\theta} \quad (P \leq P_0 = 2A\sigma_0\sin\theta)}

### 步驟 5：Phase 2（降伏後， > P_0$，即 $\sigma > \sigma_0$）

雙線性材料降伏後段：$\varepsilon = \dfrac{\sigma_0}{E_1} + \dfrac{\sigma - \sigma_0}{E_2}$

代入 $\sigma = P/(2A\sin\theta)$：

\varepsilon = \frac{\sigma_0}{E_1} + \frac{\dfrac{P}{2A\sin\theta} - \sigma_0}{E_2}

= \sigma_0\!\left(\frac{1}{E_1} - \frac{1}{E_2}\right) + \frac{P}{2AE_2\sin\theta}

代入幾何關係 $\delta = \varepsilon L/\sin\theta$：

\delta = \frac{L}{\sin\theta}\left[\sigma_0\!\left(\frac{1}{E_1} - \frac{1}{E_2}\right) + \frac{P}{2AE_2\sin\theta}\right]

= \frac{L\sigma_0}{\sin\theta}\!\left(\frac{1}{E_1} - \frac{1}{E_2}\right) + \frac{PL}{2AE_2\sin^2\theta}

\boxed{\delta = \frac{PL}{2AE_2\sin^2\theta} + \frac{L\sigma_0(E_2 - E_1)}{E_1 E_2\sin\theta}\cdot(-1)\cdot(-1)}

整理為最簡形式：

\boxed{\delta = \frac{PL}{2AE_2\sin^2\theta} + \frac{L\sigma_0(E_1 - E_2)}{E_1 E_2\sin\theta} \quad (P > P_0)}

> **等效表達（用 $ 表示）：**
> \delta = \underbrace{\frac{P_0 L}{2AE_1\sin^2\theta}}_{\delta_0,\text{ 降伏時位移}} + \underbrace{\frac{(P-P_0)L}{2AE_2\sin^2\theta}}_{\text{降伏後額外位移}}
> 此形式直觀：降伏後以 E₂ 斜率繼續增加

### 步驟 6：完整 δ-P 關係式總結

\boxed{
\delta = \begin{cases}
\dfrac{PL}{2AE_1\sin^2\theta} & 0 \leq P \leq 2A\sigma_0\sin\theta \\[12pt]
\dfrac{P_0 L}{2AE_1\sin^2\theta} + \dfrac{(P-P_0)L}{2AE_2\sin^2\theta} & P > 2A\sigma_0\sin\theta
\end{cases}
}

其中  = 2A\sigma_0\sin\theta$（降伏起始載重），$\delta_0 = \dfrac{\sigma_0 L}{E_1\sin\theta}$（對應降伏時的節點位移）。

**連續性驗算（在  = P_0$ 處兩式相等）：**

Phase 1 末端：$\delta = P_0 L/(2AE_1\sin^2\theta) = \sigma_0 L/(E_1\sin\theta)$ ✓  
Phase 2 起始（=0$）：$\delta = P_0 L/(2AE_1\sin^2\theta) = \sigma_0 L/(E_1\sin\theta)$ ✓

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 幾何關係的精確推導

設 B 點原始座標位置在 A 正下方（或對稱線上），桿 AB 長 L。B 下移 δ 後，桿長變為 ' = \sqrt{...}$（精確）。

小變形假設（$\delta \ll L$）下，桿伸長量 $\approx \delta\sin\theta$，此為本題的線性化假設。

### 5.2 雙線性 δ-P 圖的物理意義

- Phase 1：斜率 $= 2AE_1\sin^2\theta/L$（線性彈性，如彈簧 $）
- Phase 2：斜率 $= 2AE_2\sin^2\theta/L$（降伏後較軟， < k_1$ 因  < E_1$）
-  = P_0$ 處：斜率從 $ 跳為 $（雙線性折點）

### 5.3 極限載重（bonus）
