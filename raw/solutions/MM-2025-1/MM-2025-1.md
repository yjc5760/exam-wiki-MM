## 1. 原始題目重述 (Problem Restatement)

等肢角鋼斷面如圖 1 所示，試求：

1. 此斷面之幾何中心 G.C. 位置 $\langle C_x, C_y \rangle$；
2. 此斷面之面積二次慣量 $I_{xx}$、$I_{yy}$、$I_{xy}$；
3. 就一旋轉 $\theta$ 角度的座標而言，其面積二次慣量 $I_{x'x'}$、$I_{y'y'}$、$I_{x'y'}$ 之**主值**以及與其對應之**主軸角**。

已知：$L = 0.150$ m、$t = 0.015$ m。（25 分）

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

本題是斷面性質計算的「全餐題」：形心 → 慣性矩 → 慣性積 → 主值與主軸，一條龍考完 MM-U1-1 的所有基本功。出題者的意圖：

- **檢驗平行軸定理的雙重應用**：不只 $I = \bar{I} + Ad^2$（慣性矩），還包括 $I_{xy} = \bar{I}_{xy} + A\,d_x d_y$（慣性積，且 $d_x, d_y$ **帶正負號**）。
- **檢驗慣性積符號觀念**：等肢角鋼是經典的「$I_{xy} \neq 0$ 非對稱斷面」，形心慣性積為**負**值，這是多數考生最容易出錯的地方。
- **檢驗退化情形的處理**：等肢角鋼 $I_{xx} = I_{yy}$，主軸公式 $\tan 2\theta_p = \dfrac{-2I_{xy}}{I_{xx}-I_{yy}}$ 分母為零——考生必須知道此時 $2\theta_p = \pm 90°$（主軸在 45°），且要能判斷哪一軸是最大主值。這正是「會背公式」與「懂公式」的分水嶺。

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

1. 將 L 形分割為兩個**不重疊**矩形：垂直肢 $A_1 = t \times L$、水平肢 $A_2 = (L-t) \times t$。
2. 以左下角為參考原點求形心 $(C_x, C_y)$；等肢對稱 ⇒ $C_x = C_y$。
3. 平行軸定理求形心軸之 $I_{xx}$（= $I_{yy}$，由對稱性）。
4. 平行軸定理求 $I_{xy} = \sum A_i\,d_{xi}\,d_{yi}$（各矩形自身 $\bar{I}_{xy}=0$）。
5. 慣性矩莫爾圓：圓心 $I_{avg}$、半徑 $R$ → 主值 $I_{1,2} = I_{avg} \pm R$。
6. 主軸角：$I_{xx}=I_{yy}$ ⇒ $\theta_p = 45°$；以 $I_{x'y'}(\theta)=0$ 與 $I_{x'x'}(45°)$ 驗證何者為最大軸。

**關鍵陷阱：**

| # | 陷阱 | 應對策略 |
|---|------|---------|
| 1 | **分割重疊**：兩肢都取 $t \times L$ 會把角隅 $t \times t$ 算兩次 | 第二塊取 $(L-t) \times t$，或用「大正方形減缺口」法並驗算面積 |
| 2 | **$I_{xy}$ 符號**：$d_x$、$d_y$ 必須帶正負號相乘 | 垂直肢在形心左上（$d_x<0, d_y>0$）、水平肢在右下（$d_x>0, d_y<0$），兩項皆負 ⇒ $I_{xy}<0$ |
| 3 | **主軸公式分母為零**：$I_{xx}=I_{yy}$ 時 $\tan 2\theta_p \to \infty$ | 直接判 $2\theta_p = \pm 90°$，即 $\theta_p = \pm 45°$；勿硬代公式 |
| 4 | **誤判最大軸方向**：45° 與 135° 哪個是 $I_{max}$？ | 將 $\theta = 45°$ 代入 $I_{x'x'} = I_{avg} - I_{xy}\sin 2\theta$：$I_{xy}<0$ ⇒ $I_{x'x'}(45°) = I_{avg} + |I_{xy}| = I_{max}$（對稱軸為強軸） |
| 5 | **單位**：題目給 m，數值小數點極多易錯 | 全程用 mm 計算，最後換算 m⁴（$1\ \text{mm}^4 = 10^{-12}\ \text{m}^4$） |

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`求形心 (Cx, Cy)、形心軸 Ixx、Iyy、Ixy，以及主慣性矩 I1、I2 與主軸角 θp`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: } A_1 = tL,\quad A_2 = (L-t)t,\quad A = A_1 + A_2$$

$$\text{Step 2: } C_x = \frac{\sum A_i x_i}{\boxed{A}} \;(= C_y \text{，等肢對稱})$$

$$\text{Step 3: } I_{xx} = \sum \left[ \frac{b_i h_i^3}{12} + A_i \left(y_i - \boxed{C_y}\right)^2 \right] \;(= I_{yy})$$

$$\text{Step 4: } I_{xy} = \sum A_i \left(x_i - \boxed{C_x}\right)\left(y_i - \boxed{C_y}\right)$$

$$\text{Step 5: } I_{1,2} = \frac{\boxed{I_{xx}}+\boxed{I_{yy}}}{2} \pm \sqrt{\left(\frac{\boxed{I_{xx}}-\boxed{I_{yy}}}{2}\right)^2 + \boxed{I_{xy}}^2}$$

$$\text{Step 6: } \tan 2\theta_p = \frac{-2\boxed{I_{xy}}}{\boxed{I_{xx}}-\boxed{I_{yy}}} \;\Rightarrow\; \theta_p = 45°\text{（分母為零之退化情形）}$$

| 符號 | 數值 | 說明 |
|------|------|------|
| $L$ | 150 mm | 肢長 |
| $t$ | 15 mm | 肢厚 |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

> 📊 互動圖：`MM-2025-1-section-viz.html`（斷面分割、形心、主軸方向視覺化）

**單位：全程 mm、mm²、mm⁴，最後換算 SI。**

### Step 1：分割與面積

以**水平肢下緣、垂直肢左緣交點（左下角）為參考原點 O**。

- 矩形 ①（垂直肢）：寬 $t=15$、高 $L=150$，佔 $x\in[0,15]$、$y\in[0,150]$
  $$A_1 = 15 \times 150 = 2250 \text{ mm}^2,\quad (x_1, y_1) = (7.5,\ 75)$$
- 矩形 ②（水平肢，扣除重疊）：寬 $L-t=135$、高 $t=15$，佔 $x\in[15,150]$、$y\in[0,15]$
  $$A_2 = 135 \times 15 = 2025 \text{ mm}^2,\quad (x_2, y_2) = (82.5,\ 7.5)$$

$$A = 2250 + 2025 = 4275 \text{ mm}^2 = 42.75 \text{ cm}^2$$

> **策略註解：** 矩形②取 $(L-t)$ 避免角隅 $15\times15$ 重複計算（陷阱 1）。

### Step 2：形心位置

$$C_x = \frac{A_1 x_1 + A_2 x_2}{A} = \frac{2250(7.5) + 2025(82.5)}{4275} = \frac{16875 + 167062.5}{4275} = 43.03 \text{ mm}$$

等肢對稱（對 45° 軸鏡射不變）⇒

$$\boxed{C_x = C_y = 43.03 \text{ mm} = 0.04303 \text{ m}}$$

### Step 3：形心軸慣性矩 $I_{xx}$、$I_{yy}$

各矩形至形心軸距離（帶正負號）：

| 塊 | $d_x = x_i - C_x$ | $d_y = y_i - C_y$ |
|----|------------------|------------------|
| ① | $7.5 - 43.03 = -35.53$ | $75 - 43.03 = +31.97$ |
| ② | $82.5 - 43.03 = +39.47$ | $7.5 - 43.03 = -35.53$ |

$$I_{xx} = \left[\frac{15 \times 150^3}{12} + 2250(31.97)^2\right] + \left[\frac{135 \times 15^3}{12} + 2025(35.53)^2\right]$$

$$I_{xx} = \left[4{,}218{,}750 + 2{,}300{,}211\right] + \left[37{,}969 + 2{,}555{,}791\right] = 9{,}112{,}721 \text{ mm}^4$$

由等肢對稱性 $I_{yy} = I_{xx}$：

$$\boxed{I_{xx} = I_{yy} = 9.113 \times 10^6 \text{ mm}^4 = 9.113 \times 10^{-6} \text{ m}^4}$$

### Step 4：形心軸慣性積 $I_{xy}$

各矩形自身對稱 ⇒ $\bar{I}_{xy,i} = 0$，僅平行軸項：

$$I_{xy} = A_1 d_{x1} d_{y1} + A_2 d_{x2} d_{y2}$$

$$I_{xy} = 2250(-35.53)(+31.97) + 2025(+39.47)(-35.53)$$

$$I_{xy} = -2{,}555{,}791 - 2{,}839{,}769 = -5{,}395{,}560 \text{ mm}^4$$

$$\boxed{I_{xy} = -5.396 \times 10^6 \text{ mm}^4 = -5.396 \times 10^{-6} \text{ m}^4}$$

> **策略註解：** 塊①在形心「左上」、塊②在形心「右下」——材料集中於第二、四象限，故 $I_{xy} < 0$（陷阱 2）。此符號直接決定主軸方向的判讀。

### Step 5：主慣性矩（慣性矩莫爾圓）

$$I_{avg} = \frac{I_{xx} + I_{yy}}{2} = 9{,}112{,}721 \text{ mm}^4$$

$$R = \sqrt{\left(\frac{I_{xx}-I_{yy}}{2}\right)^2 + I_{xy}^2} = \sqrt{0 + (5{,}395{,}560)^2} = 5{,}395{,}560 \text{ mm}^4$$

$$I_1 = I_{avg} + R = 14{,}508{,}281 \text{ mm}^4,\qquad I_2 = I_{avg} - R = 3{,}717{,}161 \text{ mm}^4$$

$$\boxed{I_{max} = I_{x'x'} = 1.451 \times 10^7 \text{ mm}^4 = 1.451 \times 10^{-5} \text{ m}^4}$$

$$\boxed{I_{min} = I_{y'y'} = 3.717 \times 10^6 \text{ mm}^4 = 3.717 \times 10^{-6} \text{ m}^4}$$

主軸上慣性積恆為零：

$$\boxed{I_{x'y'} = 0}$$

### Step 6：主軸角

$$\tan 2\theta_p = \frac{-2I_{xy}}{I_{xx} - I_{yy}} = \frac{-2(-5.396\times10^6)}{0} \to \infty \;\Rightarrow\; 2\theta_p = \pm 90° \;\Rightarrow\; \theta_p = \pm 45°$$

判斷何者為最大軸——將 $\theta = +45°$（逆時針）代入轉換式：

$$I_{x'x'}(45°) = I_{avg} + \frac{I_{xx}-I_{yy}}{2}\cos 90° - I_{xy}\sin 90° = I_{avg} - I_{xy} = I_{avg} + |I_{xy}| = I_{max} ✓$$

$$\boxed{\theta_p = +45°\text{（逆時針）：} x' \text{ 軸沿角鋼對稱軸} \Rightarrow I_{x'x'} = I_{max} = 1.451\times10^7 \text{ mm}^4}$$

$$\theta_p = 135°\ (\text{或} -45°)\text{：} y' \text{ 軸垂直對稱軸} \Rightarrow I_{y'y'} = I_{min} = 3.717\times10^6 \text{ mm}^4$$

> **策略註解：** 物理檢核——等肢角鋼兩肢對 45° 對稱軸張開，材料離對稱軸最遠（肢尖距離 $\approx L/\sqrt{2} = 106$ mm），故繞對稱軸（$x'$）之 $I$ 最大；繞垂直軸（$y'$）材料分布最集中，$I$ 最小。此 $I_{min}$ 軸即實務上角鋼壓桿挫屈的控制軸。與型鋼表 L150×150×15 對照（$A \approx 43.0$ cm²、$I_{max} \approx 1430$ cm⁴、$I_{min} \approx 372$ cm⁴，含圓角微差）數量級完全吻合 ✓。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

1. **慣性積符號慣例**：本解採 $I_{xy} = \int xy\,dA$ 之定義（多數教科書如 Gere、Hibbeler）。若有教科書定義 $I_{xy} = -\int xy\,dA$，數值符號相反，但主值與主軸方位的物理結論不變。考場上**寫明定義式**再代入，可避免閱卷爭議。

2. **主軸角的表達**：答 $\theta_p = 45°$ 或 $-45°$ 皆為主軸（兩主軸互相垂直），但必須**指明對應哪個主值**。最完整的寫法是「$x'$ 軸自 $x$ 軸逆時針轉 45°，$I_{x'x'} = I_{max}$；$y'$ 軸對應 $I_{min}$」。只寫角度不寫對應主值會被扣分。

3. **莫爾圓 vs 公式法**：$I_{xx} = I_{yy}$ 時莫爾圓圓心恰在 $(I_{avg}, 0)$、現況點 $(I_{xx}, I_{xy})$ 位於圓的正下方（$I_{xy}<0$），轉 $2\theta = 90°$ 即達最大點——圖解一目了然，建議考場以莫爾圓草圖佐證。

4. **實務延伸**：角鋼壓桿（如桁架斜撐）的挫屈強度由 $I_{min}$（45° 弱軸）控制，而非 $I_{xx}$。這是 MM-U3-4 挫屈題與本題的天然連結，歷年常見「先算斷面性質再算挫屈」的複合題型。
