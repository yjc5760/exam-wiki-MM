## 1. 原始題目重述 (Problem Restatement)

矩形彈性板 ABCD：$150\ \text{mm} \times 100\ \text{mm}$（$x$ 方向 150 mm，$y$ 方向 100 mm）

材料性質：
- 剪彈性係數 $G = 30\ \text{GPa}$
- 波松比 $\nu = 0.33$
- 由 $G = E/[2(1+\nu)]$ 可得 $E = 2G(1+\nu) = 2\times30\times1.33 = 79.8\ \text{GPa}$

板在 $xy$ 平面的角點位移（依附圖，單位 mm）：

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
1. 從角點位移計算均勻應變 $\varepsilon_x$，$\varepsilon_y$，$\gamma_{xy}$
2. 由廣義虎克定律（含 $\sigma_z$ 的平面應力或平面應變假設）求應力
3. 用莫爾圓（應變版）求主應變與最大剪應變

**出題者意圖：**
- 測試從位移場計算應變的能力
- 測試廣義虎克定律的熟練度（特別是 $\sigma_z \neq 0$ 的情況）
- 測試莫爾圓的應用（應變版）

**關鍵：本題要求求出 $\sigma_z$，說明這是「平面應變（plane strain）」假設：$\varepsilon_z = 0$，因此 $\sigma_z \neq 0$。**

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 用角點位移計算均勻工程應變（$\varepsilon_x$，$\varepsilon_y$，$\gamma_{xy}$）
2. 判斷平面應變假設（$\varepsilon_z=0$）→ 用廣義虎克定律求三個正應力和剪應力
3. 用莫爾圓（應變圓）求 D 點主應變 $\varepsilon_{1,2}$ 和主方向
4. 求最大剪應變 $\gamma_{\max}$

**陷阱：**
1. ⭐ **平面應力 vs 平面應變**：題目要求 $\sigma_z$，故為**平面應變**（$\varepsilon_z=0$）
2. **均勻應變的計算**：對矩形板的均勻變形，應變 = 位移差 / 原長
3. **$\gamma_{xy}$ 的定義**：工程剪應變 = $\partial u/\partial y + \partial v/\partial x$（兩方向剪切變形之和）
4. **主應變 vs 最大剪應變**：$\gamma_{\max}^{平面} = \varepsilon_1 - \varepsilon_2$，但絕對最大剪應變需考慮第三主應變

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

求 D 點的應力分量 $(\sigma_x,\sigma_y,\sigma_z,\tau_{xy})$、主應變 $(\varepsilon_1,\varepsilon_2)$、最大剪應變 $\gamma_{\max}$。

### 本題關鍵公式（依計算順序）

$$\text{Step 1：} \quad \varepsilon_x = \frac{\Delta u}{\Delta x},\quad \varepsilon_y = \frac{\Delta v}{\Delta y},\quad \gamma_{xy} = \frac{\Delta u}{\Delta y} + \frac{\Delta v}{\Delta x}$$

$$\text{Step 2（平面應變）：} \quad \varepsilon_z = 0 \Rightarrow \sigma_z = \nu(\sigma_x + \sigma_y)$$

$$\text{Step 3：} \quad \sigma_x = \frac{E}{(1+\nu)(1-2\nu)}[(1-\nu)\varepsilon_x + \nu\varepsilon_y]$$

$$\sigma_y = \frac{E}{(1+\nu)(1-2\nu)}[\nu\varepsilon_x + (1-\nu)\varepsilon_y]$$

$$\tau_{xy} = G\gamma_{xy}$$

$$\text{Step 4（莫爾圓應變）：} \quad \varepsilon_{1,2} = \frac{\varepsilon_x+\varepsilon_y}{2} \pm \sqrt{\left(\frac{\varepsilon_x-\varepsilon_y}{2}\right)^2+\left(\frac{\gamma_{xy}}{2}\right)^2}$$

$$\text{Step 5：} \quad \gamma_{\max}^{\text{平面}} = \varepsilon_1 - \varepsilon_2$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $G$ | $30\ \text{GPa} = 30{,}000\ \text{MPa}$ | 剪彈性係數 |
| $\nu$ | $0.33$ | 波松比 |
| $E$ | $2G(1+\nu)=79.8\ \text{GPa}$ | 彈性模數（從 $G,\nu$ 推導） |
| $L_x$ | $150\ \text{mm}$ | 板 x 方向長度 |
| $L_y$ | $100\ \text{mm}$ | 板 y 方向高度 |

### L2：需知識點推導

| 符號 | 公式 | 卡關? |
|------|------|-------|
| $\varepsilon_x$ | $u_C/L_x = 0.042/150$ | |
| $\varepsilon_y$ | $v_A/L_y = 0.045/100$ | |
| $\gamma_{xy}$ | $u_A/L_y + v_C/L_x = 0.030/100 + 0.030/150$ | |
| $K$ | $E/[(1+\nu)(1-2\nu)]$（平面應變係數） | |
| $\sigma_z$ | $\nu(\sigma_x+\sigma_y)$ | |

### L3：深層知識

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| 平面應變 vs 平面應力 | 題目需求 $\sigma_z$ → 平面應變 $\varepsilon_z=0$；若 $\sigma_z=0$ 則為平面應力 | |
| 均勻應變場 | 矩形板均勻變形時，應變 = 角點位移差 / 板長（一次近似） | |
| 莫爾圓應變 | 座標：$(\varepsilon, \gamma/2)$，半徑 $R=\sqrt{[(\varepsilon_x-\varepsilon_y)/2]^2+(\gamma_{xy}/2)^2}$ | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：計算均勻工程應變

對均勻變形的矩形板，各應變分量：

$$\varepsilon_x = \frac{u_C - u_D}{L_x} = \frac{0.042 - 0}{150} = 2.80\times10^{-4}$$

$$\varepsilon_y = \frac{v_A - v_D}{L_y} = \frac{0.045 - 0}{100} = 4.50\times10^{-4}$$

$$\gamma_{xy} = \frac{u_A - u_D}{L_y} + \frac{v_C - v_D}{L_x} = \frac{0.030}{100} + \frac{0.030}{150} = 3.00\times10^{-4} + 2.00\times10^{-4} = 5.00\times10^{-4}$$

> 說明：$\gamma_{xy}$ 為兩個分量的和，分別來自 $\partial u/\partial y$（角 DA 段的水平位移梯度）和 $\partial v/\partial x$（角 DC 段的垂直位移梯度）。

### Step 2：廣義虎克定律（平面應變：$\varepsilon_z=0$）

**彈性係數：**
$$E = 2G(1+\nu) = 2\times30{,}000\times1.33 = 79{,}800\ \text{MPa}$$

**平面應變係數：**
$$K = \frac{E}{(1+\nu)(1-2\nu)} = \frac{79{,}800}{1.33\times0.34} = \frac{79{,}800}{0.4522} = 176{,}500\ \text{MPa}$$

**正應力：**

$$\sigma_x = K[(1-\nu)\varepsilon_x + \nu\varepsilon_y]$$
$$= 176{,}500\times[(0.67)(2.80\times10^{-4}) + (0.33)(4.50\times10^{-4})]$$
$$= 176{,}500\times[1.876\times10^{-4} + 1.485\times10^{-4}]$$
$$= 176{,}500\times3.361\times10^{-4}$$

$$\boxed{\sigma_x = 59.3\ \text{MPa}}$$

$$\sigma_y = K[\nu\varepsilon_x + (1-\nu)\varepsilon_y]$$
$$= 176{,}500\times[(0.33)(2.80\times10^{-4}) + (0.67)(4.50\times10^{-4})]$$
$$= 176{,}500\times[9.24\times10^{-5} + 3.015\times10^{-4}]$$
$$= 176{,}500\times3.939\times10^{-4}$$

$$\boxed{\sigma_y = 69.5\ \text{MPa}}$$

**平面外應力（$\varepsilon_z=0$ 的條件推導出 $\sigma_z$）：**

由廣義虎克定律第三式：
$$\varepsilon_z = \frac{1}{E}[\sigma_z - \nu(\sigma_x+\sigma_y)] = 0$$
$$\Rightarrow \sigma_z = \nu(\sigma_x+\sigma_y) = 0.33\times(59.3+69.5) = 0.33\times128.8$$

$$\boxed{\sigma_z = 42.5\ \text{MPa}}$$

**剪應力：**
$$\tau_{xy} = G\gamma_{xy} = 30{,}000\times5.00\times10^{-4}$$

$$\boxed{\tau_{xy} = 15.0\ \text{MPa}}$$

**答（a）彙總：**

| 應力分量 | 數值 |
|---------|------|
| $\sigma_x$ | $59.3\ \text{MPa}$（拉） |
| $\sigma_y$ | $69.5\ \text{MPa}$（拉） |
| $\sigma_z$ | $42.5\ \text{MPa}$（拉） |
| $\tau_{xy}$ | $15.0\ \text{MPa}$ |

### Step 3：D 點主應變（10 分）

使用平面應變的莫爾圓（以 $\varepsilon_x$，$\varepsilon_y$，$\gamma_{xy}$ 計算）：

**圓心與半徑：**

$$\varepsilon_{\text{avg}} = \frac{\varepsilon_x + \varepsilon_y}{2} = \frac{(2.80 + 4.50)\times10^{-4}}{2} = 3.65\times10^{-4}$$

$$R = \sqrt{\left(\frac{\varepsilon_x-\varepsilon_y}{2}\right)^2 + \left(\frac{\gamma_{xy}}{2}\right)^2}$$

$$= \sqrt{\left(\frac{(2.80-4.50)\times10^{-4}}{2}\right)^2 + \left(\frac{5.00\times10^{-4}}{2}\right)^2}$$

$$= \sqrt{(-0.85\times10^{-4})^2 + (2.50\times10^{-4})^2}$$

$$= \sqrt{(0.85)^2 + (2.50)^2}\times10^{-4}$$

$$= \sqrt{0.7225 + 6.25}\times10^{-4} = \sqrt{6.9725}\times10^{-4} = 2.640\times10^{-4}$$

**主應變：**

$$\varepsilon_1 = \varepsilon_{\text{avg}} + R = (3.65 + 2.640)\times10^{-4}$$

$$\boxed{\varepsilon_1 = 6.29\times10^{-4}}$$

$$\varepsilon_2 = \varepsilon_{\text{avg}} - R = (3.65 - 2.640)\times10^{-4}$$

$$\boxed{\varepsilon_2 = 1.01\times10^{-4}}$$

**主應變方向（$\varepsilon_1$ 方向與 x 軸夾角）：**

$$\tan 2\theta_p = \frac{\gamma_{xy}}{\varepsilon_x - \varepsilon_y} = \frac{5.00\times10^{-4}}{(2.80-4.50)\times10^{-4}} = \frac{5.00}{-1.70} = -2.941$$

$$2\theta_p = \arctan(-2.941) = -71.2°\quad\text{（或 } 180°-71.2°=108.8°\text{）}$$

$$\theta_p = -35.6°\quad\text{（或 }54.4°\text{）}$$

由於 $\varepsilon_x < \varepsilon_y$，$\gamma_{xy} > 0$，$\varepsilon_1$ 對應的主方向（$\theta_{p1}$）：

$$\theta_{p1} = 54.4°\text{（x 軸逆時針旋轉 54.4°）}$$

**答（b）：** 主應變 $\varepsilon_1 = 6.29\times10^{-4}$，$\varepsilon_2 = 1.01\times10^{-4}$，主方向 $\theta_{p1}=54.4°$（逆時針）。

**應變元素圖（示意）：**
```
        ε₁ = 6.29×10⁻⁴ (沿 θ=54.4° 方向)
           ↕
    ←→←→←→←→←→←→
       ε₂ = 1.01×10⁻⁴ (沿 θ=54.4°+90°=144.4° 方向)
       主方向：γ = 0
```

### Step 4：D 點最大剪應變（5 分）

**平面內最大剪應變：**

$$\gamma_{\max}^{\text{平面}} = 2R = 2\times2.640\times10^{-4} = \varepsilon_1 - \varepsilon_2$$

$$\boxed{\gamma_{\max}^{\text{平面}} = 5.28\times10^{-4}}$$

**三維最大剪應變（考慮第三主應變 $\varepsilon_3$）：**

平面應變下 $\varepsilon_z = 0$（即第三主應變 $\varepsilon_3 = 0$）：

$$\varepsilon_3 = 0$$

三個主應變大小順序：$\varepsilon_2 = 1.01\times10^{-4} < \varepsilon_3 = 0 < \varepsilon_1 = 6.29\times10^{-4}$？

但 $\varepsilon_2 = 1.01\times10^{-4} > 0 > \varepsilon_3 = 0$，故：

$\varepsilon_3 = 0 < \varepsilon_2 < \varepsilon_1$

三維最大剪應變：

$$\gamma_{\max}^{\text{3D}} = \varepsilon_1 - \varepsilon_3 = 6.29\times10^{-4} - 0 = 6.29\times10^{-4}$$

> 結論：三維最大剪應變（$6.29\times10^{-4}$）大於平面內最大剪應變（$5.28\times10^{-4}$），出現在 $xz$ 平面（第 1、3 主應變所在平面）。

**最大剪應變方向：** 發生在 45° 於 $\varepsilon_1$ 與 $\varepsilon_3$ 之間的平面。

**答（c）：**

$$\boxed{\gamma_{\max}^{\text{平面}} = 5.28\times10^{-4},\quad \gamma_{\max}^{\text{3D}} = 6.29\times10^{-4}}$$

**對應應力元素圖（最大剪應力）：**

最大剪應力：$\tau_{\max} = G\,\gamma_{\max}/2$（注意：最大剪應變 $\gamma_{\max}$ = $2\times$莫爾圓半徑）

在主應變方向旋轉 45° 處，剪應力最大：

$$\tau_{\max}^{\text{平面}} = G\times\frac{\gamma_{\max}^{\text{平面}}}{2} = 30{,}000\times\frac{5.28\times10^{-4}}{2} = 7.92\ \text{MPa}$$

**對應的正應力：**
$$\sigma_{\text{avg}} = \frac{\varepsilon_1+\varepsilon_2}{2}\times\frac{E(1-\nu)}{(1+\nu)(1-2\nu)} + \text{...}$$

或直接由應力莫爾圓：$\sigma_{\text{avg}} = (\sigma_x'+\sigma_y')/2$，其中 $\sigma_{x}', \sigma_y'$ 為主方向的正應力（等於主應力 $\sigma_1,\sigma_2$）。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**爭議點：**
1. **角點位移讀取**：本解依 PDF 文字萃取的數值進行合理分配。若附圖顯示的位移方向或大小不同，代入相同公式重算即可。
2. **平面應變 vs 平面應力**：題目要求求出 $\sigma_z$，這明確指向**平面應變假設**（$\varepsilon_z=0 \Rightarrow \sigma_z\neq0$）。若為平面應力（$\sigma_z=0$），則不需求 $\sigma_z$。

**考場建議：**
- 先確認假設（平面應變 or 平面應力）：看題目有無要求求 $\sigma_z$
- $E$、$G$、$\nu$ 三者只有兩個獨立：$E = 2G(1+\nu)$，記得先算 $E$
- 莫爾圓應變的座標：橫軸 $\varepsilon$，縱軸 $\gamma/2$（**除以 2**，不是 $\gamma$）

**進階：**
- 絕對最大剪應變（含三維）= $\varepsilon_1 - \varepsilon_3$（最大主應變 - 最小主應變）
- 對平面應變：第三主應變 $\varepsilon_3 = 0$，若 $\varepsilon_1 > 0 > \varepsilon_2$，絕對最大 = $\varepsilon_1 - \varepsilon_2$（與平面內相同）；若 $\varepsilon_2 > 0$，則絕對最大 = $\varepsilon_1 - 0 = \varepsilon_1$
