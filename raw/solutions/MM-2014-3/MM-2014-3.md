## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 平面應力的座標轉換——在同一點上，不同座標系描述的應力分量之間，以旋轉角 $\theta$ 互相關聯。轉換公式本質上就是莫爾圓的代數形式。

**出題者意圖：**
- (一) 測驗考生能否正確應用方向餘弦推導應力轉換公式（不是死記，要能推導）
- (二) 測驗逆向思維：由極座標應力回推直角座標應力，考驗考生是否理解轉換的對稱性

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**步驟化作戰計畫：**
1. 確認極座標方向單位向量在直角座標下的分量（方向餘弦）
2. 用應力轉換公式（$\sigma_{n} = \sigma_x l^2 + \sigma_y m^2 + 2\tau_{xy}lm$ 型式）推導 $\sigma_r$、$\sigma_\theta$、$\tau_{r\theta}$
3. 化簡為雙角 $2\theta$ 形式（更緊湊）
4. (二) 對調角色：x 方向在極座標下對應 $\theta_{rot} = -\theta$，或直接由 (一) 的結果作逆解

**關鍵陷阱：**

| # | 陷阱 | 正確處理 |
|---|------|---------|
| 1 | $\tau_{r\theta}$ 的正負號 | 剪應力正號規則：$\tau_{r\theta} = -({\sigma_x - \sigma_y})/{2}\cdot\sin2\theta + \tau_{xy}\cos2\theta$，不可漏負號 |
| 2 | (二) 誤認為只要把 (一) 的公式中 $r \leftrightarrow x$ 代換 | 不能直接對調；應對 (一) 的三式作代數逆解，或重新以 $(-\theta)$ 旋轉推導 |
| 3 | $\sigma_r + \sigma_\theta$ 的不變量 | $\sigma_r + \sigma_\theta = \sigma_x + \sigma_y$，可用於驗算 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
(一) $(\sigma_r, \sigma_\theta, \tau_{r\theta})$ 以 $(\sigma_x, \sigma_y, \tau_{xy}, \theta)$ 表示；(二) $\sigma_x$ 以 $(\sigma_r, \sigma_\theta, \tau_{r\theta}, \theta)$ 表示

### 本題關鍵公式（依計算順序）

$$\text{Step 1（方向餘弦）: } \hat{r} = (\cos\theta,\, \sin\theta),\quad \hat{\theta} = (-\sin\theta,\, \cos\theta)$$

$$\text{Step 2（正向應力轉換）: } \sigma_r = \sigma_x\cos^2\theta + \sigma_y\sin^2\theta + 2\tau_{xy}\sin\theta\cos\theta$$

$$\text{Step 3（雙角化簡）: } \sigma_r = \frac{\sigma_x+\sigma_y}{2} + \frac{\sigma_x-\sigma_y}{2}\cos2\theta + \tau_{xy}\sin2\theta$$

$$\text{Step 4（剪應力轉換）: } \tau_{r\theta} = -\frac{\sigma_x-\sigma_y}{2}\sin2\theta + \tau_{xy}\cos2\theta$$

$$\text{Step 5（逆解 σ_x）: } \sigma_x = \frac{\sigma_r+\sigma_\theta}{2} + \frac{\sigma_r-\sigma_\theta}{2}\cos2\theta - \tau_{r\theta}\sin2\theta$$

### L1：題目直接給定

| 符號 | 說明 |
|------|------|
| $\sigma_x, \sigma_y, \tau_{xy}$ | 直角座標應力分量 |
| $\sigma_r, \sigma_\theta, \tau_{r\theta}$ | 極座標應力分量 |
| $\theta$ | r 方向與 x 軸夾角 |

### L2：需知識點推導

| 量 | 公式／來源 | 卡關? |
|----|-----------|-------|
| $\hat{r}$ 的分量 | $(\cos\theta, \sin\theta)$ | |
| $\hat{\theta}$ 的分量 | $(-\sin\theta, \cos\theta)$（垂直於 $\hat{r}$，逆時針） | |
| 斜面正向應力 | $\sigma_n = \sigma_x l^2 + \sigma_y m^2 + 2\tau_{xy}lm$ | |
| 斜面剪應力 | $\tau = (\sigma_y - \sigma_x)lm + \tau_{xy}(l^2 - m^2)$（以 $l=\cos\theta, m=\sin\theta$） | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| $\hat{\theta}$ 方向 | $\hat{\theta}$ 是極座標的切線方向，逆時針為正，分量 $(-\sin\theta, \cos\theta)$ | |
| 不變量 | $\sigma_r + \sigma_\theta = \sigma_x + \sigma_y$（第一不變量，旋轉下不變） | |
| 逆解策略 | 兩組方程式 (A)(D) 和 (C) 分別求和與差，即可解出 $\sigma_x$ | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 準備：方向餘弦

極座標 $r$ 方向（$\hat{r}$）在直角座標下的方向餘弦：

$$l_r = \cos\theta, \quad m_r = \sin\theta$$

極座標 $\theta$ 方向（$\hat{\theta}$，逆時針切線）的方向餘弦：

$$l_\theta = -\sin\theta, \quad m_\theta = \cos\theta$$

---

### 第一部分（15 分）：由 $(\sigma_x, \sigma_y, \tau_{xy})$ 求 $(\sigma_r, \sigma_\theta, \tau_{r\theta})$

**求 $\sigma_r$**（$r$ 面上的正向應力，法線方向 $= \hat{r}$）：

$$\sigma_r = \sigma_x l_r^2 + \sigma_y m_r^2 + 2\tau_{xy} l_r m_r$$
$$= \sigma_x\cos^2\theta + \sigma_y\sin^2\theta + 2\tau_{xy}\sin\theta\cos\theta$$

利用雙角公式 $\cos^2\theta = \frac{1+\cos2\theta}{2}$，$\sin^2\theta = \frac{1-\cos2\theta}{2}$，$2\sin\theta\cos\theta = \sin2\theta$：

$$\boxed{\sigma_r = \frac{\sigma_x+\sigma_y}{2} + \frac{\sigma_x-\sigma_y}{2}\cos2\theta + \tau_{xy}\sin2\theta}$$

**求 $\sigma_\theta$**（$\theta$ 面上的正向應力，法線方向 $= \hat{\theta}$）：

$$\sigma_\theta = \sigma_x l_\theta^2 + \sigma_y m_\theta^2 + 2\tau_{xy} l_\theta m_\theta$$
$$= \sigma_x\sin^2\theta + \sigma_y\cos^2\theta - 2\tau_{xy}\sin\theta\cos\theta$$

$$\boxed{\sigma_\theta = \frac{\sigma_x+\sigma_y}{2} - \frac{\sigma_x-\sigma_y}{2}\cos2\theta - \tau_{xy}\sin2\theta}$$

*策略注解：注意 $\sigma_r + \sigma_\theta = \sigma_x + \sigma_y$（第一不變量）✓*

**求 $\tau_{r\theta}$**（$r$ 面上 $\hat{\theta}$ 方向的剪應力）：

$$\tau_{r\theta} = \sigma_x l_r l_\theta + \sigma_y m_r m_\theta + \tau_{xy}(l_r m_\theta + l_\theta m_r)$$
$$= \sigma_x(\cos\theta)(-\sin\theta) + \sigma_y(\sin\theta)(\cos\theta) + \tau_{xy}[(\cos\theta)(\cos\theta) + (-\sin\theta)(\sin\theta)]$$
$$= (\sigma_y - \sigma_x)\sin\theta\cos\theta + \tau_{xy}(\cos^2\theta - \sin^2\theta)$$

$$\boxed{\tau_{r\theta} = -\frac{\sigma_x-\sigma_y}{2}\sin2\theta + \tau_{xy}\cos2\theta}$$

---

### 第二部分（10 分）：由 $(\sigma_r, \sigma_\theta, \tau_{r\theta})$ 求 $\sigma_x$

從第一部分的三個結果出發：

$$\sigma_r = \frac{\sigma_x+\sigma_y}{2} + \frac{\sigma_x-\sigma_y}{2}\cos2\theta + \tau_{xy}\sin2\theta \tag{A}$$

$$\sigma_\theta = \frac{\sigma_x+\sigma_y}{2} - \frac{\sigma_x-\sigma_y}{2}\cos2\theta - \tau_{xy}\sin2\theta \tag{B}$$

$$\tau_{r\theta} = -\frac{\sigma_x-\sigma_y}{2}\sin2\theta + \tau_{xy}\cos2\theta \tag{C}$$

**Step 1：** (A) + (B) 得到不變量：

$$\sigma_r + \sigma_\theta = \sigma_x + \sigma_y \tag{D}$$

**Step 2：** (A) $-$ (B) 得到：

$$\sigma_r - \sigma_\theta = (\sigma_x - \sigma_y)\cos2\theta + 2\tau_{xy}\sin2\theta \tag{E}$$

**Step 3：** 消去 $\tau_{xy}$。

將 (E) $\times \cos2\theta$：

$$(\sigma_r - \sigma_\theta)\cos2\theta = (\sigma_x - \sigma_y)\cos^2 2\theta + 2\tau_{xy}\sin2\theta\cos2\theta \tag{E'}$$

將 (C) $\times (-2\sin2\theta)$：

$$-2\tau_{r\theta}\sin2\theta = (\sigma_x - \sigma_y)\sin^2 2\theta - 2\tau_{xy}\cos2\theta\sin2\theta \tag{C'}$$

(E') + (C') 得到：

$$(\sigma_r - \sigma_\theta)\cos2\theta - 2\tau_{r\theta}\sin2\theta = (\sigma_x - \sigma_y)(\cos^2 2\theta + \sin^2 2\theta) = \sigma_x - \sigma_y \tag{F}$$

**Step 4：** (D) + (F)：

$$(\sigma_r + \sigma_\theta) + [(\sigma_r - \sigma_\theta)\cos2\theta - 2\tau_{r\theta}\sin2\theta] = 2\sigma_x$$

$$\boxed{\sigma_x = \frac{\sigma_r+\sigma_\theta}{2} + \frac{\sigma_r-\sigma_\theta}{2}\cos2\theta - \tau_{r\theta}\sin2\theta}$$

展開為乘積形式：

$$\sigma_x = \sigma_r\cos^2\theta + \sigma_\theta\sin^2\theta - 2\tau_{r\theta}\sin\theta\cos\theta$$

---

### 結果彙整

| 轉換方向 | 公式 |
|---------|------|
| $\sigma_r$ | $\dfrac{\sigma_x+\sigma_y}{2} + \dfrac{\sigma_x-\sigma_y}{2}\cos2\theta + \tau_{xy}\sin2\theta$ |
| $\sigma_\theta$ | $\dfrac{\sigma_x+\sigma_y}{2} - \dfrac{\sigma_x-\sigma_y}{2}\cos2\theta - \tau_{xy}\sin2\theta$ |
| $\tau_{r\theta}$ | $-\dfrac{\sigma_x-\sigma_y}{2}\sin2\theta + \tau_{xy}\cos2\theta$ |
| $\sigma_x$（逆） | $\dfrac{\sigma_r+\sigma_\theta}{2} + \dfrac{\sigma_r-\sigma_\theta}{2}\cos2\theta - \tau_{r\theta}\sin2\theta$ |

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**Q1：$\sigma_x$ 公式與 $\sigma_r$ 公式有何對稱關係？**

比較兩式：
- $\sigma_r = (\sigma_x+\sigma_y)/2 + (\sigma_x-\sigma_y)/2\cdot\cos2\theta + \tau_{xy}\sin2\theta$
- $\sigma_x = (\sigma_r+\sigma_\theta)/2 + (\sigma_r-\sigma_\theta)/2\cdot\cos2\theta - \tau_{r\theta}\sin2\theta$

差異在於 $\sin2\theta$ 前的剪應力符號相反。這是因為從 x-y 到 r-θ 是旋轉 $+\theta$，從 r-θ 回到 x-y 是旋轉 $-\theta$，$\sin(-2\theta) = -\sin2\theta$。

**Q2：莫爾圓的觀點**

以上轉換公式即莫爾圓參數方程。令：
$$\left(\sigma_r - \frac{\sigma_x+\sigma_y}{2}\right)^2 + \tau_{r\theta}^2 = \left(\frac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2 = R^2$$

$(\sigma_r, \tau_{r\theta})$ 在莫爾圓上，以 $2\theta$ 為參數。直角座標應力對應 $\theta=0$（點 $(\sigma_x, -\tau_{xy})$），極座標應力對應旋轉 $2\theta$ 後的點。
