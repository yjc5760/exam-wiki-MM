**主分類：** `MM-U3-2` 梁桿件變位及內力分析
**副分類：** （無）
**分析法：** 能量法
**標籤：** `靜不定梁` `彈簧支承` `重疊法` `贅力法` `懸臂梁` `集中載重` `撓度` `諧和條件`

---

## 1. 原始題目重述 (Problem Restatement)

一均勻梁，彎曲剛度為 $EI$，**左端 A 固定（$x = 0$）**，在 $x = L$ 處設有一**垂直彈簧**（彈簧係數 $k = EI/L^3$），梁右端 $x = 2L$（B 端）承受向下的**集中載重 $P$**。

**幾何與條件：**
- 梁總長：$2L$（固定端到 B 端）
- 彈簧位置：距固定端 $L$（梁跨中）
- 彈簧係數：$k = \dfrac{EI}{L^3}$
- B 端載重：$P$（向下）

**求解要求：**
1. 彈簧反力 $R$
2. 中點（彈簧處，$x = L$）的位移
3. B 端（$x = 2L$）的位移（並請註明正負方向）

**提示公式**（懸臂梁在距固定端 $a$ 處受集中力 $P$ 的撓度）：

$$v(x) = -\frac{Px^2}{6EI}(3a - x), \quad 0 \le x \le a$$

$$v(x) = -\frac{Pa^2}{6EI}(3x - a), \quad a \le x \le L_{\text{total}}$$

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

### 核心觀念
本題是**一次靜不定梁**（超靜定度 1），採用**重疊法（Method of Superposition）**求解：
1. 以彈簧反力 $R$ 為**贅力**，釋放彈簧（去掉彈簧，改為在 $x = L$ 施加向上力 $R$）→ 得到靜定懸臂梁
2. 分兩個載重狀態疊加（P 作用、R 作用）求 $x = L$ 處的撓度
3. 利用**諧和條件**（彈簧壓縮量 = 梁在該點的向下撓度）解出 $R$
4. 代回求各點位移

### 出題者意圖
- 測驗**靜不定梁重疊法**的完整流程（釋放贅力 → 疊加 → 諧和條件）
- 提供撓度公式暗示要用**重疊法**，而非二重積分法
- 考驗學生能否正確套用提示公式中「a 值」的對應關係（不同載重工況下 a 值不同）
- 陷阱：彈簧施加的是**向上**的力（對梁），須確保符號正確

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

### 作戰計畫
```
Step 1：靜不定度判斷：懸臂梁（固定端提供 3 個反力）+ 彈簧（1 個額外約束）→ 1 次靜不定
Step 2：取贅力：將彈簧反力 R（向上）作為贅力，釋放彈簧後得靜定懸臂梁
Step 3：計算「工況一」—— 僅有 P 作用（無彈簧）時，x = L 處向下的撓度 v_P(L)
Step 4：計算「工況二」—— 僅有 R 作用（在 x = L 向上）時，x = L 處的撓度 v_R(L)
Step 5：諧和條件：v_P(L) + v_R(L) = -R/k（彈簧壓縮，向下為負）→ 解出 R
Step 6：代入 R 求各點最終位移
```

### 關鍵陷阱

**陷阱 1：諧和條件的方向設定**
> 設向下為負，彈簧受壓時縮短，梁在彈簧處向下移 $\delta$，則彈簧壓縮量也是 $\delta$（方向相同）。  
> 彈簧反力對梁是**向上（正方向）**，故 $R > 0$。  
> 諧和條件：$v(L) = -R/k$（向下的撓度等於彈簧壓縮量，負號因向下為負）

**陷阱 2：提示公式中「a」的對應值**
> 工況一（P 作用在 x = 2L）：$a = 2L$，要求 $x = L$ 的撓度 → 用 $0 \le x \le a$ 公式，代 $x = L, a = 2L$  
> 工況二（R 作用在 x = L）：$a = L$，要求 $x = L$ 的撓度 → 用 $0 \le x \le a$ 公式，代 $x = L, a = L$（此時在端點，兩段公式取極限，用第一段即可）

**陷阱 3：B 端（x = 2L）位移的計算**
> B 端位移需用工況一 + 工況二（含 R），此時 $x = 2L = a_P$（P 的作用點），用第二段公式  
> 對工況二（R 在 x = L），求 x = 2L 的撓度：$a_R = L$，$x = 2L > a_R$，用第二段公式

**陷阱 4：彈簧係數的單位與代入**
> $k = EI/L^3$ 代入時，確保單位一致：$[k] = \text{N/mm}$，$[EI] = \text{N·mm}^2$，$[L^3] = \text{mm}^3$，$k \cdot L^3 = EI$ ✓

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 問題簡化：釋放贅力

將彈簧反力 $R$（方向向上，$R > 0$）設為贅力。去掉彈簧後，得到**靜定懸臂梁**，承受以下兩個分離工況：

- **工況一（Case 1）：** 懸臂梁，僅 $P$（向下）作用在 $x = 2L$
- **工況二（Case 2）：** 懸臂梁，僅 $R$（向上）作用在 $x = L$

> **諧和條件（Compatibility Condition）：**
> 在實際結構中，$x = L$ 處梁的向下撓度 $|\delta_L|$ = 彈簧壓縮量 $= R/k$
>
> 設向下為負（撓度 $v < 0$ 表示向下）：
> $$v_{\text{total}}(L) = -\frac{R}{k}$$

---

### Step 1：工況一（僅 P 在 x = 2L）求 x = L 的撓度

提示公式，$a = 2L$（P 的作用點），求 $x = L$（$x < a$，用第一段）：

$$v_P(L) = -\frac{P \cdot L^2}{6EI}(3a - L)\bigg|_{a=2L} = -\frac{PL^2}{6EI}(3 \times 2L - L) = -\frac{PL^2}{6EI}(5L)$$

$$\boxed{v_P(L) = -\frac{5PL^3}{6EI}}$$

（負號：向下）

---

### Step 2：工況二（僅 R 在 x = L 向上）求 x = L 的撓度

提示公式作用在 $a = L$（R 的作用點），求 $x = L = a$（在作用點正好，兩段公式均適用，取第一段 $0 \le x \le a$）：

R 為向**上**，為提示公式中 P 的反向，故在公式中以 $(-R)$ 代替 P（或直接寫正方向的撓度為正）：

$$v_R(L) = +\frac{R \cdot L^2}{6EI}(3L - L) = \frac{RL^2}{6EI}(2L) = \frac{RL^3}{3EI}$$

$$\boxed{v_R(L) = +\frac{RL^3}{3EI}}$$

（正號：向上，因 R 向上使梁向上撓）

---

### Step 3：諧和條件求彈簧反力 $R$

$$v_P(L) + v_R(L) = -\frac{R}{k}$$

代入：

$$-\frac{5PL^3}{6EI} + \frac{RL^3}{3EI} = -\frac{R}{k}$$

代入 $k = \dfrac{EI}{L^3}$，故 $\dfrac{1}{k} = \dfrac{L^3}{EI}$：

$$-\frac{5PL^3}{6EI} + \frac{RL^3}{3EI} = -\frac{RL^3}{EI}$$

等式兩側乘以 $\dfrac{EI}{L^3}$：

$$-\frac{5P}{6} + \frac{R}{3} = -R$$

移項：

$$\frac{R}{3} + R = \frac{5P}{6}$$

$$R\left(\frac{1}{3} + 1\right) = \frac{5P}{6}$$

$$R \cdot \frac{4}{3} = \frac{5P}{6}$$

$$\boxed{R = \frac{5P}{6} \times \frac{3}{4} = \frac{5P}{8}}$$

> **策略註解：** 代入 $k = EI/L^3$ 後，$\dfrac{1}{k} = \dfrac{L^3}{EI}$，右側 $-R/k = -RL^3/EI$，讓方程式無因次化（除以 $PL^3/EI$），非常乾淨。

---

### Step 4：求中點（x = L，彈簧處）的位移

由諧和條件直接得：

$$v(L) = -\frac{R}{k} = -\frac{5P/8}{EI/L^3} = -\frac{5PL^3}{8EI}$$

$$\boxed{v(L) = -\frac{5PL^3}{8EI}\ \text{（向下）}}$$

**驗算（疊加法）：**

$$v(L) = v_P(L) + v_R(L) = -\frac{5PL^3}{6EI} + \frac{(5P/8)L^3}{3EI}$$

$$= -\frac{5PL^3}{6EI} + \frac{5PL^3}{24EI} = PL^3\left(-\frac{20}{24} + \frac{5}{24}\right) = -\frac{15PL^3}{24EI} = -\frac{5PL^3}{8EI}\ ✓$$

---

### Step 5：求 B 端（x = 2L）的位移

**工況一（P 在 x = 2L，求 x = 2L）：** 點在載重作用點，用第一段（$x = a = 2L$）：

$$v_P(2L) = -\frac{P(2L)^2}{6EI}(3 \times 2L - 2L) = -\frac{4PL^2}{6EI}(4L) = -\frac{16PL^3}{6EI} = -\frac{8PL^3}{3EI}$$

**工況二（R 在 x = L，求 x = 2L > a = L）：** 用第二段（$a \le x$）：

$$v_R(2L) = +\frac{R \cdot L^2}{6EI}(3 \times 2L - L) = \frac{(5P/8)L^2}{6EI}(5L) = \frac{25PL^3}{48EI}$$

（R 向上，符號取正）

**B 端最終位移：**

$$v(2L) = v_P(2L) + v_R(2L) = -\frac{8PL^3}{3EI} + \frac{25PL^3}{48EI}$$

通分（分母 48）：

$$= PL^3 \left(-\frac{128}{48} + \frac{25}{48}\right) = -\frac{103PL^3}{48EI}$$

$$\boxed{v(2L) = -\frac{103PL^3}{48EI}\ \text{（向下）}}$$

---

### 📋 最終結果彙整

| 求解項目 | 結果 | 方向 |
|---------|------|------|
| 彈簧反力 $R$ | $\dfrac{5P}{8}$ | 向上（對梁） |
| 中點 $x=L$ 撓度 | $-\dfrac{5PL^3}{8EI}$ | 向下 |
| B 端 $x=2L$ 撓度 | $-\dfrac{103PL^3}{48EI}$ | 向下 |

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 諧和條件符號的嚴謹建立

本題最容易出錯在諧和條件的符號。嚴謹做法：

設 $v$ 向下為負（標準 beam 座標），彈簧受壓縮 $\delta_s > 0$（長度縮短），則：
- 梁在 $x = L$ 的向下撓度 = $|\delta_s|$
- 因此 $v(L) = -\delta_s$（向下為負）
- 彈簧受力平衡：$R = k \cdot \delta_s$（R 為彈簧對梁的反力，向上）
- 故 $\delta_s = R/k$，代入得 $v(L) = -R/k$ ✓

### 5.2 本題靜不定度的確認

固定端（$x = 0$）：提供 3 個反力（$V_A, M_A, H_A$，但水平方向無水平力，故有效約束為 $V_A, M_A$，共 2 個垂直方向約束）
彈簧（$x = L$）：提供 1 個垂直約束

靜定懸臂梁需 2 個約束（$V_A, M_A$），額外多 1 個（彈簧）→ **1 次靜不定** ✓

### 5.3 若無提示公式，如何用二重積分法？

若不使用提示公式，需對含彈簧的懸臂梁直接積分：
- 在 $0 \le x \le L$：$EI v'' = M(x) = -P(2L - x) + R(L - x)$
- 在 $L \le x \le 2L$：$EI v'' = M(x) = -P(2L - x)$

解出積分常數需用：邊界條件（$x = 0$ 時 $v = 0, v' = 0$）+ 連續條件（$x = L$ 時 $v$ 和 $v'$ 連續）+ 諧和條件（$x = L$ 時梁撓度 = 彈簧壓縮）

共 5 個方程解 5 個未知數（4 個積分常數 + $R$），計算量遠大於重疊法。**出題者提示公式，明示要用重疊法。**

### 5.4 結果合理性驗算

$R = 5P/8$，表示彈簧分擔了 62.5% 的等效支撐力。若無彈簧（$k = 0$），懸臂梁自由端 $v(2L) = -\dfrac{P(2L)^3}{3EI} = -\dfrac{8PL^3}{3EI} = -\dfrac{128PL^3}{48EI}$，比有彈簧時的 $-\dfrac{103PL^3}{48EI}$ 更大（向下更多）✓——彈簧確實減小了自由端撓度，物理上合理。

彈簧位移 $v(L) = -\dfrac{5PL^3}{8EI} = -\dfrac{30PL^3}{48EI}$，對應彈簧力 $R = k \cdot |v(L)| = \dfrac{EI}{L^3} \cdot \dfrac{5PL^3}{8EI} = \dfrac{5P}{8}$ ✓

### 5.5 相關題型延伸

- **彈簧在 B 端**（自由端有彈簧支撐）：類似做法，但諧和條件在 B 端
- **兩個彈簧**：超靜定度 2，需聯立兩個諧和條件
- **均布載重代替集中力**：提示公式改用均布載重懸臂梁標準解
