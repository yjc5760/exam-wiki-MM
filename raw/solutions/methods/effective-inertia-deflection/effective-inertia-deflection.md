# 有效慣性矩法（撓度計算）

**Method ID：** effective-inertia-deflection
**知識分類：** 解題工具
**適用題型：** RC-U3-1 梁工作性要求（含撓度、裂縫）
**適用規範：** ACI 318 §24.2.3–24.2.4（Branson 有效慣性矩法）

---

## 核心原理

RC 梁開裂後剛度介於全斷面慣性矩 $I_g$ 與開裂斷面慣性矩 $I_{cr}$ 之間。Branson（1965）提出以作用彎矩 $M_a$ 與開裂彎矩 $M_{cr}$ 的比值進行加權平均，得到有效慣性矩 $I_e$，用於計算服務載重下的即時撓度。

---

## 邊界條件／對應關係表

| 條件 | 處理方式 |
|------|---------|
| $M_a < M_{cr}$（未開裂） | $I_e = I_g$（直接用全斷面） |
| $M_a \geq M_{cr}$（已開裂） | 用 Branson 公式 |
| 計算結果 $I_e > I_g$ | 取 $I_e = I_g$（上限） |
| 懸臂梁 | $M_a$ 取固定端彎矩（最大值） |
| 連續梁 | $I_e$ 可取跨中與支承端之加權平均 |

**長期撓度係數 $\xi$（ACI）：**

| 持載時間 | $\xi$ |
|---------|:-----:|
| 3 個月  | 1.0   |
| 6 個月  | 1.2   |
| 1 年    | 1.4   |
| 5 年以上 | 2.0  |

---

## 步驟流程

### Step 1：計算開裂彎矩 $M_{cr}$

$$M_{cr} = \frac{f_r \cdot I_g}{y_t}$$

其中破裂模數：

$$f_r = 2\sqrt{f'_c} \text{（kgf/cm}^2\text{）} \quad \text{或} \quad f_r = 0.7\sqrt{f'_c} \text{（MPa）}$$

$y_t$ = 未開裂斷面形心至受拉緣距離。

### Step 2：計算未開裂全斷面慣性矩 $I_g$

矩形斷面：$I_g = bh^3/12$

T 形梁或複合斷面：依形心位置計算。

### Step 3：計算開裂斷面慣性矩 $I_{cr}$（轉換斷面法）

令鋼筋模數比 $n = E_s/E_c$，以受壓區中性軸位置 $kd$ 為分界：

**求 $kd$（矩形梁）：**

$$\frac{b(kd)^2}{2} = n A_s (d - kd) + (n-1) A'_s (kd - d')$$

（雙筋梁；單筋梁去掉壓力筋項）

**計算 $I_{cr}$：**

$$I_{cr} = \frac{b(kd)^3}{3} + n A_s (d - kd)^2 + (n-1) A'_s (kd - d')^2$$

### Step 4：計算有效慣性矩 $I_e$（Branson 公式）

$$I_e = \left(\frac{M_{cr}}{M_a}\right)^3 I_g + \left[1 - \left(\frac{M_{cr}}{M_a}\right)^3\right] I_{cr} \leq I_g$$

$M_a$ = 服務載重下的最大彎矩（不含折減係數）。

### Step 5：計算即時撓度 $\Delta_i$

| 載重與支承條件 | 公式 |
|-------------|------|
| 簡支梁，均佈載重 | $\Delta_i = \dfrac{5wL^4}{384 E_c I_e}$ |
| 簡支梁，跨中集中 | $\Delta_i = \dfrac{PL^3}{48 E_c I_e}$ |
| 懸臂梁，均佈載重 | $\Delta_i = \dfrac{wL^4}{8 E_c I_e}$ |
| 懸臂梁，端部集中 | $\Delta_i = \dfrac{PL^3}{3 E_c I_e}$ |

$E_c = 15000\sqrt{f'_c}$（kgf/cm²）或 $E_c = 4700\sqrt{f'_c}$（MPa）

### Step 6：計算長期撓度

$$\lambda_\Delta = \frac{\xi}{1 + 50\rho'}$$

其中 $\rho' = A'_s / (b \cdot d)$（壓力筋比；無壓力筋時 $\rho' = 0$）。

**長期附加撓度：**

$$\Delta_{long} = \lambda_\Delta \cdot \Delta_i(DL+\text{持續LL})$$

**總撓度：**

$$\Delta_{total} = \Delta_i(DL + LL) + \Delta_{long}$$

---

## 常用型式與結果表

| 情形 | $\lambda_\Delta$ 簡化 |
|------|-------------------|
| 無壓力筋，5 年以上持載 | $\lambda_\Delta = 2.0$ |
| 無壓力筋，1 年持載 | $\lambda_\Delta = 1.4$ |
| $\rho' = \rho/2$，5 年 | $\lambda_\Delta = 2.0/(1+50\cdot\rho/2)$ |

---

## 常見陷阱

1. **$I_e$ 不可超過 $I_g$**：若 $M_a < M_{cr}$，直接取 $I_g$。
2. **$\rho'$ 是壓力筋比**：無壓力筋時 $\rho' = 0$，$\lambda = \xi$（不等於 1）。
3. **$\xi$ 值依持載時間**：5 年以上取 2.0，考題常給「已使用多年」暗示用 2.0。
4. **雙筋梁 $I_{cr}$**：壓力鋼筋使用 $(n-1)$ 而非 $n$（已含混凝土面積）。
5. **懸臂梁的 $M_a$**：取固定端（最大）彎矩，非跨中。
