## 1. 原始題目重述（Problem Restatement）

如圖二所示，一懸臂梁（cantilever beam）受端部集中載重 $P$ 作用（向下），梁軸沿 $x$ 方向。斷面由**兩個相同的 C 型（厂型）槽鋼**拼合而成的**箱型（閉合）斷面**。

**斷面幾何（厂型槽，每個）：**
- 翼板寬度：$b$
- 腹板高度（含翼板厚度）：$h$
- 板厚（翼板與腹板均相同）：$t$
- 假設為**薄壁斷面**（$t \ll b,\, h$），忽略角落重疊面積

**兩種拼合方式（均為對稱箱型）：**
- **(c) 上下焊接（top-bottom joints）**：兩個 C 型槽的腹板（垂直板）相對，翼板在外側，焊縫位於**左右兩腹板的中間水平面**（即上下兩個 C 型的水平翼板端部）→ 焊縫在左側與右側腹板
- **(d) 左右焊接（left-right joints）**：兩個 C 型槽的翼板（水平板）相對，焊縫位於**上下翼板中央**（即上翼板接縫 + 下翼板接縫）→ 焊縫在頂部與底部

> **判斷方式**：從圖二觀察，(c) 的紅色箭頭指向左右側（腹板位置），表示剪力流在左右接縫累積；(d) 的紅色箭頭指向上下（翼板位置），表示剪力流在頂底接縫累積。

## 2. 考題核心精神與出題者意圖（Core Concepts & Examiner's Intent）

**核心精神：** 剪力流（shear flow）在**組合梁接合面**的應用——理解 $q = VQ/I$ 中的 $Q$ 是如何依「切開面」位置而變化，以及不同焊接位置對焊接剪力需求的影響。

**出題者意圖：**
1. 測試考生是否能正確判斷組合斷面的**接合面位置**，並針對該位置計算對應的 $Q$
2. 兩個 C 型槽組成完全相同的箱型斷面，但焊縫位置不同，導致接合面的 $Q$ 值不同 → 焊接所需剪力不同
3. 考驗考生能否識別兩種配置幾何上的**對稱性**：(c) 有兩條焊縫（左右各一），(d) 有兩條焊縫（上下各一），每條焊縫的剪力流量因 $Q$ 不同而有差別

**關鍵洞察：**
- 兩種斷面的**總 $I$（對形心軸）完全相同**（因為斷面幾何完全一樣，只是焊縫位置不同）
- 差異在於「切開焊縫後」所餘面積的一次矩 $Q$

## 3. 解題戰略地圖與陷阱分析（Strategic Roadmap & Trap Analysis）

**作戰順序：**

① 建立薄壁箱型斷面的幾何模型（取 $t \ll b,h$ 的薄壁近似）

② 計算斷面總慣性矩 $I$（兩種配置的 $I$ 相同，僅算一次）

③ **情況(c) 上下焊接**：
- 辨識焊縫位置（左右腹板高度中點的水平切面）
- 計算切面以外（即上半個 C 型槽）的面積一次矩 $Q_c$
- 焊接剪力流 $q_c = VQ_c / I$（每條焊縫；注意此切面有兩條焊縫）

④ **情況(d) 左右焊接**：
- 辨識焊縫位置（上翼板中央垂直切面 + 下翼板中央垂直切面）
- 計算切面以外（即右半個 C 型槽）的面積一次矩 $Q_d$
- 焊接剪力流 $q_d = VQ_d / I$

⑤ 比較 $q_c$ 與 $q_d$

**四個關鍵陷阱：**

| 陷阱 | 錯誤思路 | 正確應對 |
|------|---------|---------|
| T1 | 兩種配置的 $I$ 不同 | 兩種配置幾何完全相同，$I$ 相同 |
| T2 | 情況(c) 的焊縫在腹板正中央，$Q$ 取腹板一半 | 焊縫在**上下兩個 C 型的接合處**（腹板水平切面），$Q$ 取整個上半個 C 型（含翼板）|
| T3 | 忘記剪力流 $q$ 的方向（箱型斷面剪力流沿壁中線連續流動） | 箱型斷面剪力流為連續封閉流；焊縫所受剪力 = 焊縫兩側的剪力流之差（對開口處）或積分值 |
| T4 | 每條焊縫的剪力流算法混淆 | 對組合梁接合面：$q = VQ/I$，$Q$ 取「被焊縫切開後脫離部分」的面積對整體形心的一次矩 |

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`求兩種焊接配置（上下焊接、左右焊接）各自焊縫每單位長度所需承受的剪力（剪力流 q）`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: 斷面慣性矩} \quad I = 2\left[\frac{bh^3}{12} - \frac{(b-t)(h-2t)^3}{12}\right] \approx \frac{t}{6}\left[bh^3 - (b-t)(h-2t)^3\right]$$

（或薄壁近似逐段計算：腹板 + 兩翼板的慣性矩之和，乘以 2 因兩個 C 型）

$$\text{Step 2（薄壁近似）: } I = 2\left[\frac{t h^3}{12} + 2 \cdot bt\left(\frac{h}{2}-\frac{t}{2}\right)^2 + \frac{b^3 t}{12}\right] \approx 2\left[\frac{t h^3}{12} + \frac{bt h^2}{2}\right]$$

$$\text{（精確薄壁）Step 2: } I \approx \frac{t h^3}{6} + b t h^2$$

$$\text{Step 3(c): } Q_c = bt\left(\frac{h}{2} - \frac{t}{2}\right) + t \cdot \frac{h}{2} \cdot \frac{h}{4} \approx \frac{bth}{2} + \frac{th^2}{8}$$

$$\text{Step 3(d): } Q_d = t \cdot h \cdot \frac{h}{4} \cdot 2 + bt\cdot 0 \approx \frac{th^2}{4}$$

（取上半斷面的一次矩）

$$\text{Step 4: 剪力流} \quad q = \frac{V \cdot \boxed{Q}}{I}$$

### L1：題目直接給定

_看到題目就能讀出的數字，不需要任何公式。_

| 符號 | 數值 | 說明 |
|------|------|------|
| $P$ | $P$（符號） | 端部集中向下載重 |
| $b$ | $b$ | C 型槽翼板寬度 |
| $h$ | $h$ | C 型槽總高（含翼板厚度） |
| $t$ | $t$ | 板厚（翼板 = 腹板） |
| 薄壁 | $t \ll b,\, h$ | 薄壁斷面假設（題意） |
| $V$ | $V = P$ | 懸臂梁任意截面剪力 = $P$ |

### L2：需知識點推導

_需要知道公式名稱與適用條件，套入 L1 即可算出。_

**Step 1：斷面慣性矩 $I$（兩種配置相同）**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|:-----:|
| $I_{\text{腹板}}$ | $2 \times \dfrac{t h^3}{12}$（腹板對形心矩） | |
| $I_{\text{翼板}}$ | $2 \times 2 \cdot bt \cdot \left(\dfrac{h-t}{2}\right)^2$（翼板用平行軸定理）≈ $2bth^2/2$ | |
| $I$ | $I \approx \dfrac{th^3}{6} + bth^2$（薄壁近似） | |

**Step 2：$Q_c$（上下焊接，切於腹板水平中點）**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|:-----:|
| $Q_{翼板}$ | $bt \cdot \dfrac{h}{2}$（翼板面積 × 形心距形心）≈ $\dfrac{bth}{2}$ | |
| $Q_{腹板半}$ | $\dfrac{h}{2} \cdot t \cdot \dfrac{h}{4} = \dfrac{th^2}{8}$（腹板上半面積 × 半距） | |
| $Q_c$ | $\dfrac{bth}{2} + \dfrac{th^2}{8}$ | |

**Step 3：$Q_d$（左右焊接，切於翼板中央垂直線）**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|:-----:|
| $Q_{右腹板}$ | $ht \cdot 0 = 0$（腹板在形心兩側對稱，垂直中線的 $\bar{y}$ 項不為零）→ 需重新考慮正確切面 | |
| 說明 | 左右焊接，切面為翼板水平中線，脫離部分 = 整個上翼板（單個 C 型的上翼板） | |
| $Q_d$ | $bt \cdot \dfrac{h-t}{2} \approx \dfrac{bth}{2}$ | |

**Step 4：剪力流 $q$**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|:-----:|
| $q_c$（每條焊縫） | $V Q_c / I$（左右各一條焊縫） | |
| $q_d$（每條焊縫） | $V Q_d / I$（上下各一條焊縫） | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|:-----:|
| $Q$ 的取法（切面以外） | $Q$ 必須對**整體斷面形心軸**計算；取「焊縫被切開後，其中一側部分」的面積 × 該部分形心到整體形心的距離 | |
| 組合梁每條焊縫的剪力流 | $q = VQ/I$ 的 $Q$ 對應**單條焊縫切開後脫離部分**，若有對稱兩條焊縫，每條各承受 $VQ/I$ | |
| 兩種焊接配置 $Q$ 不同的原因 | 切開面不同，$Q$ 對應不同的「脫離面積」，幾何不同故 $Q$ 值不同 | |
| 箱型斷面 vs 開口斷面的剪力流差異 | 箱型斷面有不定（indeterminate）剪力流分量；但組合梁問題只看接合面，用 $q=VQ/I$ 計算接合面剪力（此為設計問題，不是分析閉合斷面剪力分布的完整問題） | |

## 4. 步驟化詳細計算過程（Step-by-Step Detailed Calculation）

> 📊 互動圖：`MM-2023-2-sfd-bmd-viz.html`（剪力流分布示意）

### 前提：薄壁假設與坐標系

取**整體斷面形心在幾何中心**（因兩個 C 型槽對稱拼合，形心必在幾何中心）。
定義 $y$ 軸向上為正，整體高度 = $h$，整體寬度 = $2b$（兩翼板各寬 $b$）。

> 策略註解：兩個 C 型槽的外緣構成一個箱型斷面，總寬 $2b$、總高 $h$。薄壁假設下，忽略角落面積重複計算。

### Step 1：斷面慣性矩 $I$（兩種配置相同）

**薄壁分段法（最清楚）：**

箱型斷面 = 兩個 C 型槽，每個 C 型槽由三段薄板組成：
- 腹板（垂直板）：高 $h$，厚 $t$，面積 $= ht$，形心在 $y = 0$（整體形心）
- 上翼板：寬 $b$，厚 $t$，面積 $= bt$，形心在 $y = \dfrac{h-t}{2} \approx \dfrac{h}{2}$
- 下翼板：寬 $b$，厚 $t$，面積 $= bt$，形心在 $y = -\dfrac{h-t}{2} \approx -\dfrac{h}{2}$

一個 C 型槽對形心軸的慣性矩（薄壁近似，$t/h \to 0$）：

$$I_{\text{one C}} = \frac{t \cdot h^3}{12} + 2\left[bt \cdot \left(\frac{h}{2}\right)^2 + \frac{b \cdot t^3}{12}\right]$$

$$\approx \frac{th^3}{12} + \frac{bth^2}{2} + 0$$

兩個 C 型槽（整體斷面）：

$$\boxed{I = 2 I_{\text{one C}} = \frac{th^3}{6} + bth^2 = th^2\left(\frac{h}{6} + b\right)}$$

> 策略註解：翼板的 $bt^3/12$ 項遠小於 $bt(h/2)^2$，薄壁假設下可忽略。

---

### Step 2：情況(c) — 上下焊接

#### 焊縫位置分析

上下焊接（圖(c)）：兩個 C 型槽的**腹板**（垂直板）相對拼合，拼合處形成兩條水平焊縫，分別位於：
- 左側腹板水平中點：$y = 0$（斷面形心高度）
- 右側腹板水平中點：$y = 0$

即焊縫沿梁軸方向為**兩條水平縫**，各位於左右腹板高度中點。

#### 計算 $Q_c$

切開焊縫（以左側腹板水平縫為例），脫離部分為「上半個 C 型槽」：
- 腹板上半段：高 $h/2$，厚 $t$，面積 $= \dfrac{h}{2} \cdot t$，形心在 $y = \dfrac{h}{4}$
- 上翼板：寬 $b$，厚 $t$，面積 $= bt$，形心在 $y \approx \dfrac{h}{2}$

$$Q_c = \frac{ht}{2} \cdot \frac{h}{4} + bt \cdot \frac{h}{2}$$

$$\boxed{Q_c = \frac{th^2}{8} + \frac{bth}{2} = \frac{th}{8}(h + 4b)}$$

#### 每條焊縫剪力流 $q_c$

$$\boxed{q_c = \frac{V \cdot Q_c}{I} = \frac{P \cdot \dfrac{th}{8}(h+4b)}{th^2\!\left(\dfrac{h}{6}+b\right)} = \frac{P(h+4b)}{8h\!\left(\dfrac{h}{6}+b\right)}}$$

化簡（乘以 $6/6$）：

$$\boxed{q_c = \frac{3P(h+4b)}{4h(h+6b)}}$$

> 策略註解：此為每條焊縫（左側或右側）的剪力流，兩條焊縫各自承受相同的 $q_c$（對稱性）。

---

### Step 3：情況(d) — 左右焊接

#### 焊縫位置分析

左右焊接（圖(d)）：兩個 C 型槽的**翼板**（水平板）相對拼合，拼合處形成兩條垂直焊縫，分別位於：
- 上翼板中央（$y = h/2$，沿梁軸的縱向縫）
- 下翼板中央（$y = -h/2$，沿梁軸的縱向縫）

即焊縫為**兩條沿梁軸方向的縱向縫**，分別在頂部翼板與底部翼板的正中央。

#### 計算 $Q_d$

切開上翼板焊縫（位於頂部），脫離部分為「右半個 C 型槽的上翼板」：
- 右側上翼板：寬 $b$，厚 $t$，面積 $= bt$，形心在 $y \approx h/2$

但注意：此題的切面是**翼板中央的縱向切面**，切開後脫離的是整個右側 C 型槽（包含其腹板與兩個翼板）。

更正確的作法：
- 脫離部分 = 右側一個完整 C 型槽
- 右腹板：面積 $= ht$，形心在 $y = 0$（貢獻 $Q = 0$）
- 右上翼板：面積 $= bt$，形心 $y \approx h/2$（貢獻 $Q = \dfrac{bth}{2}$）
- 右下翼板：面積 $= bt$，形心 $y \approx -h/2$（貢獻 $Q = -\dfrac{bth}{2}$）

> ⚠ 上下翼板的一次矩符號相反，抵消！

這說明切在**頂部或底部翼板焊縫**的 $Q$，不應取整個 C 型槽，而應只取「焊縫切開面以外，且**在剪力方向上形成不平衡力**的部分」。

**重新分析：** 對於左右焊接，焊縫在上翼板，切開後上翼板的右半脫離；若焊縫在水平翼板，「以外部分」 = 從右翼板最右端到切縫位置的板段。

實際上，左右焊接的焊縫在**翼板中央**（水平方向對接），切開方向為**垂直（縱向）**，因此 $Q_d$ 對應的「脫離部分」是：

取**上翼板焊縫**（位於 $y = h/2$ 附近，沿 $x$ 方向的縱縫，切面為水平面）：

> 修正：仔細觀察圖(d)——箭頭在上下翼板處，表示焊縫在**上下翼板中央的水平縱向接縫**（兩 C 型槽的翼板端對端接合）。此時切面為水平切面，位於 $y = +h/2$（近頂部）。

脫離部分 = 整個上翼板（已薄壁，高度 $\approx 0$，但面積 = $bt$ per C，兩C合計 = $2bt$？）

重新辨識幾何：

**左右焊接**實際幾何：
- 左側 C 型槽開口朝右，右側 C 型槽開口朝左
- 兩者翼板（水平板）端對端接合，焊縫在頂部翼板的中央對接縫 + 底部翼板的中央對接縫
- 切開頂部焊縫後，脫離部分 = **整個頂部翼板**（由兩個翼板端點接合而成的整體翼板，寬 $2b$）

等等——從圖(b)的 C 型斷面看，每個 C 型槽的尺寸：腹板高 $h$（含上下翼板），翼板寬 $b$（往同側延伸）。

左右焊接（d）：翼板朝上下，兩個 C 型左右排列，翼板端對端拼合，形成一個總寬 $2b$、高 $h$ 的箱型。焊縫在**頂部寬翼板中央**（$y = h/2$）和**底部寬翼板中央**（$y = -h/2$）。

這是**垂直切面（縱向接縫）**！切面法向量水平，「脫離部分」是焊縫以外的**整個右半斷面**（包含右腹板和右上下翼板一半），其對水平軸（$z$ 軸）的一次矩。

但這樣 $Q_d$ 就要取整個右半斷面：
- 右腹板：$ht$，$\bar{y} = 0$，$Q = 0$
- 右上翼板一半（寬 $b/2$？不對，翼板總寬 $b$，切在翼板中央則右半寬 $b/2$）→ 但焊縫在頂部，應是頂翼板的中央

---

**【最終正確幾何解讀】**

重新從物理角度解讀：

- **上下焊接(c)**：兩個 C 型槽**背靠背**（腹板相對），焊縫在**側面（腹板水平縫）**
  - 焊縫方向：沿梁軸（$x$ 方向）
  - 焊縫平面：水平，位於兩腹板水平中點（$y = 0$ 附近的腹板對接縫）
  - 脫離部分（切開左側水平縫）= 上半個 C 型槽（左腹板上半 + 上翼板）

- **左右焊接(d)**：兩個 C 型槽**翼板相對**（面對面），焊縫在**頂部與底部翼板**
  - 焊縫方向：沿梁軸（$x$ 方向）
  - 焊縫平面：水平，位於頂部翼板中央（$y \approx h/2$）和底部翼板中央（$y \approx -h/2$）
  - 脫離部分（切開頂部水平縫）= 整個頂部翼板（由兩個 C 型的上翼板對接，總寬 $2b$，厚 $t$）

**計算 $Q_d$（切開頂部翼板水平縫）：**

脫離部分 = 頂部翼板：寬 $2b$（兩個 C 型翼板加起來）×厚 $t$，形心 $y \approx h/2$（薄壁假設，形心位於翼板中線）

$$Q_d = (2b \cdot t) \cdot \frac{h}{2}$$

$$\boxed{Q_d = bth}$$

#### 每條焊縫剪力流 $q_d$

$$\boxed{q_d = \frac{V \cdot Q_d}{I} = \frac{P \cdot bth}{th^2\!\left(\dfrac{h}{6}+b\right)} = \frac{Pb}{h\!\left(\dfrac{h}{6}+b\right)}}$$

化簡：

$$\boxed{q_d = \frac{6Pb}{h(h+6b)}}$$

> 策略註解：此為每條焊縫（頂部或底部）的剪力流，兩條焊縫各自承受相同的 $q_d$（對稱性）。

---

### Step 4：比較兩種配置的焊接剪力

$$\frac{q_c}{q_d} = \frac{\dfrac{3P(h+4b)}{4h(h+6b)}}{\dfrac{6Pb}{h(h+6b)}} = \frac{3(h+4b)}{4} \cdot \frac{1}{6b} = \frac{(h+4b)}{8b}$$

$$\boxed{\frac{q_c}{q_d} = \frac{h+4b}{8b}}$$

**討論：**

- 若 $h = 4b$（常見工字型比例），則 $q_c/q_d = 8b/8b = 1$（兩者相等）
- 若 $h > 4b$（高腹板），則 $q_c > q_d$（上下焊接的焊縫剪力較大）
- 若 $h < 4b$（矮胖斷面），則 $q_c < q_d$（左右焊接的焊縫剪力較大）

---

## 5. 關鍵爭議點與進階探討（Critical Issues & Advanced Discussion）

**爭議 1：Q 的取法——脫離部分的認定**

本題最容易出錯之處在於「切開焊縫後，脫離部分究竟是什麼？」

- **上下焊接(c)**：焊縫在腹板水平中點，切面為水平面，脫離的是**上半個 C 型槽**（含腹板上半 + 翼板）→ 需計算兩板段的 $Q$ 之和
- **左右焊接(d)**：焊縫在翼板（水平接縫），切面為水平面，脫離的是**整個頂部翼板**（兩個 C 型的上翼板合計，寬 $2b$）→ 只有一個矩形面積貢獻 $Q$

**爭議 2：閉合斷面的剪力流**

嚴格而言，箱型（閉合）斷面的剪力流分布需考慮靜不定的「環流剪力流」（constant shear flow $q_0$，由扭矩平衡決定）。**但本題的物理問題是「組合梁接合面設計」**，焊縫所承受的剪力 = 接合面剪力流 = $VQ/I$，此公式適用於：將兩個子構件的接合面視為「正在被剪力切開的截面」，屬於組合梁（built-up beam）設計範疇，不需要完整的閉合斷面分析。

**進階：若考慮偏心載重（剪力中心）**

懸臂梁的集中載重 $P$ 若不通過斷面的**剪力中心**，梁會發生扭轉。對稱箱型斷面，剪力中心位於斷面形心（由對稱性），故只要載重通過形心（沿翼板中線），即不產生額外扭轉效應。題目中圖(a)顯示載重向下且位於斷面對稱面，故不須額外考慮扭轉。

**考場建議：**
1. 先清楚畫出切面示意（哪條焊縫，切面在何處）
2. 用「脫離部分面積 × 形心到整體形心距離」求 $Q$，確認所取的每一個子面積正確
3. 代入 $q = VQ/I$ 時確認 $V = P$（懸臂梁任意截面剪力均為 $P$）
