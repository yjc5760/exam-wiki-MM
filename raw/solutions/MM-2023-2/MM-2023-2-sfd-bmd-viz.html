MM-2023-2 箱型斷面剪力流分布（組合梁焊縫設計）

懸臂梁受端部集中載重 P | 薄壁近似 | 剪力流 q = VQ/I

(c) 上下焊接 — 焊縫在左右腹板

焊縫位置：左右腹板水平中線（y = 0）
脫離部分：上半個 C 型槽（腹板上半 + 翼板）
Qc = th²/8 + bth/2
    = th(h + 4b) / 8
qc = 3P(h+4b) / [4h(h+6b)]

(d) 左右焊接 — 焊縫在上下翼板

焊縫位置：頂部翼板中央（y ≈ +h/2）
                底部翼板中央（y ≈ −h/2）
脫離部分：整個頂部翼板（寬 2b）
Qd = (2b · t) · h/2 = bth
qd = 6Pb / [h(h+6b)]

兩種配置比較（令 h = αb，計算 q 比值）

qc / qd = (h + 4b) / (8b)
h = 4b 時比值 = 1（兩者相等）
h > 4b → qc > qd（上下焊接焊縫剪力較大）
h < 4b → qc < qd（左右焊接焊縫剪力較大）
