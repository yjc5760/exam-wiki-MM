# MM-2020-4｜四支點連續梁最大撓度（三彎矩方程）

**題號：** MM-2020-4  
**年份：** 2020  
**主題：** 連續梁、三彎矩方程、撓度計算  
**primaryTopicId：** MM-U3-2  
**analysisMethod：** 彈性分析

---

## 題目

梁長 3L，EI 均勻，由左至右等分為三段（每段 L），以 **1 鉸接＋3 滾輪** 等間距支撐（支點位於 x = 0, L, 2L, 3L）。

集中力 **P** 作用於 **第二跨中點**（x = 3L/2，即全梁中央「中間點」）。

求：
- **A 點**（第一段位移最大點）之上下位移量
- **B 點**（第二段位移最大點）之上下位移量

支點命名：1(x=0)、2(x=L)、3(x=2L)、4(x=3L)

---

## 靜不定分析：三彎矩方程

外端支點為鉸/滾輪，端彎矩為零：$M_1 = M_4 = 0$

未知量：內部支點彎矩 $M_2$、$M_3$

各跨自由彎矩圖（簡支梁）：
- 跨 1（x=0 to L）：無荷載 → $m_1^0 = 0$
- 跨 2（x=L to 2L）：P 於跨中 → 三角形，峰值 $PL/4$
- 跨 3（x=2L to 3L）：無荷載 → $m_3^0 = 0$

**荷載積分項**（P 作用於跨 2 中點，$a = b = L/2$）：

$$\frac{6}{L}A'\bar{b}' = \frac{6}{L} \cdot A' \cdot \frac{L}{2} = \frac{6}{L}\cdot\frac{PL^2}{8}\cdot\frac{L}{2} = \frac{3PL^2}{8}$$

（由對稱性，對左端與右端積分值相同）

### 三彎矩方程 @ 支點 2（$M_1=0$，跨 1 無載）

$$M_1 L + 2M_2(L+L) + M_3 L = -\frac{3PL^2}{8}$$

$$4M_2 + M_3 = -\frac{3PL}{8} \qquad\cdots(I)$$

### 三彎矩方程 @ 支點 3（$M_4=0$，跨 3 無載）

$$M_2 L + 2M_3(L+L) + M_4 L = -\frac{3PL^2}{8}$$

$$M_2 + 4M_3 = -\frac{3PL}{8} \qquad\cdots(II)$$

### 求解

由對稱性（I）=（II）→ $M_2 = M_3$：

$$5M_2 = -\frac{3PL}{8} \implies \boxed{M_2 = M_3 = -\frac{3PL}{40}}$$

（負號 = 支點處為負彎矩，即梁於內支點向上彎）

---

## 支承反力

由全梁靜力平衡，對稱結構：$R_1 = R_4$，$R_2 = R_3$

各跨剪力（以跨 1 為例，無橫向荷載，彎矩線性）：
$$V_1 = \frac{M_2 - M_1}{L} = \frac{-3PL/40}{L} = -\frac{3P}{40}$$

$$R_1 = -V_1\Big|_{left} = -\frac{3P}{40} \quad\text{（向下）}$$

$$R_2 = P/2 - R_1 = \frac{P}{2} + \frac{3P}{40} = \frac{23P}{40} \quad\text{（向上）}$$

**驗算：** $2R_1 + 2R_2 = -6P/40 + 46P/40 = 40P/40 = P$ ✓

---

## 彎矩分佈

### 跨 1（$0 \leq x \leq L$，無荷載）：

$$M(x) = -\frac{3P}{40}x \quad \text{（線性，全跨負彎矩）}$$

### 跨 2（$0 \leq \xi \leq L/2$，$\xi$ 自支點 2 量起）：

$$M(\xi) = -\frac{3PL}{40} + \frac{P\xi}{2}$$

跨中（$\xi = L/2$）：$M = 7PL/40$（正彎矩最大）

---

## 撓度計算

### 跨 1 撓度（彎矩方程積分法）

$$EIy'' = -\frac{3P}{40}x$$

積分 + 邊界條件 $y(0)=0,\;y(L)=0$：

$$\boxed{EIy_1(x) = \frac{P}{80}\,x\!\left(L^2 - x^2\right)}$$

最大撓度位置（令 $dy/dx = 0$）：

$$\frac{d}{dx}\left[x(L^2-x^2)\right] = L^2 - 3x^2 = 0 \implies x_A = \frac{L}{\sqrt{3}} = \frac{\sqrt{3}}{3}L$$

**A 點最大撓度：**

$$EIy_A = \frac{P}{80}\cdot\frac{L}{\sqrt{3}}\cdot\left(L^2 - \frac{L^2}{3}\right) = \frac{P}{80}\cdot\frac{L}{\sqrt{3}}\cdot\frac{2L^2}{3} = \frac{PL^3}{120\sqrt{3}} = \frac{\sqrt{3}\,PL^3}{360}$$

$$\boxed{\delta_A = \frac{\sqrt{3}\,PL^3}{360\,EI} \approx \frac{0.00481\,PL^3}{EI} \quad \textbf{朝上}}$$

---

### 跨 2 撓度（分段積分）

對跨 2 兩段分別積分，利用邊界條件 $y(0)=y(L)=0$ 及 $\xi=L/2$ 處連續性：

常數求解：$C_3 = -PL^2/40$，$C_5 = -3PL^2/20$，$C_6 = PL^3/48$

跨中（$\xi = L/2$，即 B 點）：

$$EIy_B = -\frac{3PL}{80}\cdot\frac{L^2}{4} + \frac{P}{12}\cdot\frac{L^3}{8} - \frac{PL^2}{40}\cdot\frac{L}{2}$$
$$= PL^3\left(-\frac{3}{320} + \frac{1}{96} - \frac{1}{80}\right) = PL^3\cdot\frac{-9+10-12}{960} = -\frac{11PL^3}{960}$$

（負號 = 向下）

$$\boxed{\delta_B = \frac{11\,PL^3}{960\,EI} \approx \frac{0.01146\,PL^3}{EI} \quad \textbf{朝下}}$$

**驗算（由右段）：** 同樣計算得 $-11PL^3/960$ ✓

**B 點即為 P 施加點（跨 2 中點），確為該跨最大撓度。**

---

## 最終答案

| 點 | 位置 | 位移量 | 方向 |
|----|------|--------|------|
| A（跨 1 最大） | $x_A = \dfrac{\sqrt{3}}{3}L$ 距左端鉸 | $\dfrac{\sqrt{3}\,PL^3}{360\,EI}$ | **朝上** |
| B（跨 2 最大） | 跨 2 中點（P 施加處） | $\dfrac{11\,PL^3}{960\,EI}$ | **朝下** |

---

## 物理意義與解題關鍵

1. **三彎矩方程**：外端 $M=0$，由對稱性 $M_2=M_3=-3PL/40$；支點彎矩為負（外端被提起）。
2. **跨 1 反力為負**：$R_1 = -3P/40$（向下），物理上 P 作用在跨 2，內支點 2、3 承受大部分載重，外端支點受向下拉力（需能承拉之支點）。
3. **A 點在 $x = L/\sqrt{3}$（非跨中）**：跨 1 無橫向荷載，撓度呈三次曲線，最大撓度不在跨中，而在 $L/\sqrt{3} \approx 0.577L$ 處。
4. **B 點撓度 vs 單跨簡支**：單跨簡支 P 於跨中之撓度 $PL^3/48EI$，本連續梁為 $11PL^3/960EI \approx 0.55 \times PL^3/48EI$，因連續性約束使撓度減小。
