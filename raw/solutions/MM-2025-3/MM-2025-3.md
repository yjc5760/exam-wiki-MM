## 1. 原始題目重述 (Problem Restatement)

兩斷面相同之橫梁（桿）相互對向，以懸臂方式固定於兩側剛性牆上（圖 3a）：

- 原長各 $L = 500$ mm，斷面積 $A = 50$ mm²；
- 環境溫度 $T_0 = 20°C$ 時，兩桿自由端之間有間隙 $\delta = 0.1$ mm；
- 左桿為材料 a、右桿為材料 b，皆為**雙線性彈塑性材料**（圖 3b）：第一段斜率 $E_{1a} = 200$ GPa、$E_{1b} = 100$ GPa；第二段斜率 $E_{2a} = 20$ GPa、$E_{2b} = 10$ GPa；降伏點 $\sigma_{ya} = 200$ MPa、$\sigma_{yb} = 100$ MPa；
- 熱膨脹係數為常數：$\alpha_a = 1.0\times10^{-5}\ (1/°C)$、$\alpha_b = 1.6\times10^{-5}\ (1/°C)$。

試繪出**溫度升高 $\Delta T$ 與桿內軸力 $N$ 之關係曲線**。梁自重與端點效應可忽略不計。（25 分）

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

本題是「**平衡—變形諧和—組成律**」三大支柱的進階應用：組成律從線彈性升級為**雙線性彈塑性**，且加入**間隙**與**溫度**兩個經典變因。出題者要測驗：

- **間隙問題的階段劃分**：接觸前（$N=0$，自由膨脹）與接觸後（靜不定）是兩個不同的結構系統；
- **逐段降伏的追蹤能力**：兩材料降伏點不同，誰先降伏？降伏後系統勁度如何改變？——答案是一條**四段折線**，每個轉折點都有明確的物理事件（接觸、b 降伏、a 降伏）；
- **應變硬化材料的組成律**：降伏後 $\varepsilon = \sigma_y/E_1 + (\sigma-\sigma_y)/E_2$，不是理想塑性的「應力鎖定」——這是本題與傳統理想彈塑性題的最大差異；
- **物理直觀**：每段斜率 $dN/d\Delta T = k_{eq}\,(\alpha_a+\alpha_b)L$，其中 $k_{eq}$ 為兩桿（以切線模數計）的串聯勁度——降伏使系統變軟，曲線斜率逐段下降。

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

1. **階段 0（未接觸）**：自由膨脹，$N = 0$。求間隙閉合溫度 $\Delta T_1$：$(\alpha_a + \alpha_b)L\,\Delta T_1 = \delta$。
2. **階段 1（接觸，雙彈性）**：諧和條件「熱膨脹 − 彈性壓縮 = $\delta$」，解 $N(\Delta T)$ 直線；以 $\sigma = N/A$ 檢查何者先降伏（b 先，因 $\sigma_{yb}$ 較低）→ 求 $\Delta T_2$。
3. **階段 2（b 塑性、a 彈性）**：b 之組成律改用雙線性式，重解 $N(\Delta T)$；檢查 a 降伏 → 求 $\Delta T_3$。
4. **階段 3（雙塑性）**：兩桿皆用雙線性式，得最終直線段。
5. 整理四段折線，標出三個轉折點座標，繪 $N$–$\Delta T$ 圖。

**關鍵陷阱：**

| # | 陷阱 | 應對策略 |
|---|------|---------|
| 1 | **忘記間隙段**：直接從 $\Delta T = 0$ 寫諧和條件 | 先算 $\Delta T_1 = \delta / [(\alpha_a+\alpha_b)L]$，此前 $N \equiv 0$ |
| 2 | **諧和條件漏掉 $\delta$**：接觸後總膨脹量必須「多走」一個間隙 | 接觸後恆有：$(\alpha_a+\alpha_b)L\Delta T - \varepsilon_a L - \varepsilon_b L = \delta$ |
| 3 | **誤判降伏順序**：以為 E 大者先降伏 | 兩桿軸力相同、面積相同 ⇒ 應力相同，**降伏應力低者（b）先降伏** |
| 4 | **塑性段組成律寫錯**：寫成 $\varepsilon = \sigma/E_2$ | 雙線性：$\varepsilon = \sigma_y/E_1 + (\sigma - \sigma_y)/E_2$（過降伏點後沿 $E_2$ 延伸） |
| 5 | **單位混用**：GPa、MPa、mm 混算 | 全程 N、mm、MPa（1 GPa = 1000 MPa） |

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`繪出溫度升高 ΔT 與桿內軸力 N 的四段折線關係曲線（標明各轉折點）`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: } \Delta T_1 = \frac{\delta}{(\alpha_a + \alpha_b)L}\quad\text{（間隙閉合）}$$

$$\text{Step 2: } (\alpha_a+\alpha_b)L\,\Delta T - \frac{NL}{A}\left(\frac{1}{E_{1a}} + \frac{1}{E_{1b}}\right) = \delta\quad\text{（雙彈性段）}$$

$$\text{Step 3: } \Delta T_2:\ \boxed{N} = \sigma_{yb}A\quad\text{（b 先降伏）}$$

$$\text{Step 4: } (\alpha_a+\alpha_b)L\,\Delta T - L\left[\frac{\sigma}{E_{1a}} + \frac{\sigma_{yb}}{E_{1b}} + \frac{\sigma-\sigma_{yb}}{E_{2b}}\right] = \delta,\quad \sigma = \boxed{N}/A$$

$$\text{Step 5: } \Delta T_3:\ \boxed{N} = \sigma_{ya}A\quad\text{（a 降伏）}$$

$$\text{Step 6: } (\alpha_a+\alpha_b)L\,\Delta T - L\left[\frac{\sigma_{ya}}{E_{1a}} + \frac{\sigma-\sigma_{ya}}{E_{2a}} + \frac{\sigma_{yb}}{E_{1b}} + \frac{\sigma-\sigma_{yb}}{E_{2b}}\right] = \delta\quad\text{（雙塑性段）}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

> 📊 互動圖：`MM-2025-3-stress-strain-viz.html`（雙線性材料曲線 + N–ΔT 四段折線）

**單位：N、mm、MPa。令 $N$ = 兩桿共同之壓縮軸力（接觸後串聯系統，兩桿軸力相同），$\sigma = N/A$。**

便利常數：$(\alpha_a + \alpha_b)L = (1.0+1.6)\times10^{-5} \times 500 = 0.013$ mm/°C。

### 階段 0：自由膨脹（未接觸），$N = 0$

兩桿自由端相向伸長，間隙閉合條件：

$$(\alpha_a + \alpha_b)L\,\Delta T_1 = \delta \;\Rightarrow\; 0.013\,\Delta T_1 = 0.1$$

$$\boxed{\Delta T_1 = 7.69°C}\qquad (0 \le \Delta T \le 7.69°C:\ N = 0)$$

### 階段 1：接觸後，雙桿皆彈性（7.69°C ≤ ΔT ≤ 65.38°C）

諧和條件（兩桿端點貼合後，總熱伸長 − 總彈性壓縮 = 間隙）：

$$0.013\,\Delta T - \frac{NL}{A}\left(\frac{1}{E_{1a}} + \frac{1}{E_{1b}}\right) = 0.1$$

$$0.013\,\Delta T - N \times \frac{500}{50}\left(\frac{1}{200000} + \frac{1}{100000}\right) = 0.1 \;\Rightarrow\; 0.013\,\Delta T - 1.5\times10^{-4} N = 0.1$$

$$\boxed{N = 86.67\,\Delta T - 666.7\ \text{(N)}}\qquad \frac{dN}{d\Delta T} = 86.7\ \text{N/°C}$$

> **策略註解（勁度檢核）：** $k_a = E_{1a}A/L = 20000$、$k_b = 10000$ N/mm，串聯 $k_{eq} = 6667$ N/mm，斜率 $= k_{eq} \times 0.013 = 86.7$ ✓。

**降伏檢查**：兩桿應力相同 $\sigma = N/50$。$\sigma_{yb} = 100 < \sigma_{ya} = 200$ ⇒ **桿 b 先降伏**，當 $N = \sigma_{yb}A = 5000$ N：

$$5000 = 86.67\,\Delta T_2 - 666.7 \;\Rightarrow\; \boxed{\Delta T_2 = 65.38°C\ (N = 5000\ \text{N},\ \sigma = 100\ \text{MPa})}$$

### 階段 2：桿 b 塑性（E₂b）、桿 a 彈性（65.38°C ≤ ΔT ≤ 469.2°C）

桿 b 組成律改為 $\varepsilon_b = \dfrac{\sigma_{yb}}{E_{1b}} + \dfrac{\sigma - \sigma_{yb}}{E_{2b}}$：

$$0.013\,\Delta T - 500\left[\frac{\sigma}{200000} + \frac{100}{100000} + \frac{\sigma - 100}{10000}\right] = 0.1$$

$$0.013\,\Delta T - (0.0025\sigma + 0.5 + 0.05\sigma - 5) = 0.1 \;\Rightarrow\; 0.0525\,\sigma = 0.013\,\Delta T + 4.4$$

$$\sigma = 0.2476\,\Delta T + 83.81 \;\Rightarrow\; \boxed{N = 12.38\,\Delta T + 4190\ \text{(N)}}\qquad \frac{dN}{d\Delta T} = 12.4\ \text{N/°C}$$

（驗算連續性：$\Delta T = 65.38$ → $N = 12.38(65.38)+4190 = 5000$ ✓）

> **策略註解：** 桿 b 切線勁度降為 $E_{2b}A/L = 1000$ N/mm，串聯 $k_{eq} = (1/20000 + 1/1000)^{-1} = 952$ N/mm，斜率 $952\times0.013 = 12.4$ ✓——系統明顯變軟。

**降伏檢查**：桿 a 降伏當 $N = \sigma_{ya}A = 10000$ N：

$$10000 = 12.38\,\Delta T_3 + 4190 \;\Rightarrow\; \boxed{\Delta T_3 = 469.2°C\ (N = 10000\ \text{N},\ \sigma = 200\ \text{MPa})}$$

### 階段 3：雙桿皆塑性（ΔT ≥ 469.2°C）

$$0.013\,\Delta T - 500\left[\frac{200}{200000} + \frac{\sigma-200}{20000} + \frac{100}{100000} + \frac{\sigma-100}{10000}\right] = 0.1$$

$$0.013\,\Delta T - (0.075\sigma - 9) = 0.1 \;\Rightarrow\; \sigma = 0.1733\,\Delta T + 118.67$$

$$\boxed{N = 8.67\,\Delta T + 5933\ \text{(N)}}\qquad \frac{dN}{d\Delta T} = 8.7\ \text{N/°C}$$

（驗算連續性：$\Delta T = 469.2$ → $N = 8.67(469.2)+5933 = 10000$ ✓；勁度檢核：$k_{eq} = (1/2000+1/1000)^{-1} = 667$ N/mm，斜率 $667\times0.013 = 8.7$ ✓）

### N–ΔT 關係曲線總結（壓力為正）

| 階段 | $\Delta T$ 範圍 (°C) | $N(\Delta T)$ (N) | 斜率 (N/°C) | 物理事件 |
|------|---------------------|--------------------|:-----------:|---------|
| 0 自由膨脹 | $0 \to 7.69$ | $N = 0$ | 0 | 間隙未閉 |
| 1 雙彈性 | $7.69 \to 65.38$ | $86.67\,\Delta T - 666.7$ | 86.7 | ΔT₁=7.69 接觸 |
| 2 b 塑性 | $65.38 \to 469.2$ | $12.38\,\Delta T + 4190$ | 12.4 | ΔT₂=65.38，b 降伏（N=5000） |
| 3 雙塑性 | $> 469.2$ | $8.67\,\Delta T + 5933$ | 8.7 | ΔT₃=469.2，a 降伏（N=10000） |

**曲線形狀**：自原點沿 $\Delta T$ 軸平移至 (7.69, 0)，接著三段斜率遞減（86.7 → 12.4 → 8.7 N/°C）的折線上升，轉折點 (65.38, 5000) 與 (469.2, 10000)。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

1. **ΔT₃ = 469°C 的合理性**：數值偏高（實際鋼材在此溫度 E、σy 已大幅折減），但題目明示「熱膨脹係數為常數」且材料參數固定，依題意作答即可。考場上可加一句「實際材料性質隨溫度變化，此為題設簡化」展現工程判斷，但**不可**擅自修改題設。

2. **軸力的正負號表達**：本解以「壓力」為正繪圖。若以張力為正，曲線在第四象限（$N$ 為負）。建議圖上明確標注「$N$ 為壓縮軸力」，避免閱卷歧義。

3. **圖 3b 的「解壓」虛線是伏筆**：本題只問升溫曲線，未問卸載。若延伸問「升溫至某點後降回 $T_0$」，則兩桿沿 $E_1$ 彈性卸載，因塑性應變不可回復，可能出現**殘留應力**（桿件互推的殘留壓力或重新開隙）——這是 MM-U4-2 殘留應力的標準延伸題型，複習時建議自行演練「升至 ΔT=100°C 後降回」的情境。

4. **為何兩桿軸力恆相等？** 接觸後兩桿透過接觸面互推，接觸面上力平衡 ⇒ 串聯系統內力單一。若題目改為「兩桿並聯（共同夾於兩剛板間）」則為等變形系統，降伏判據變成比較 $\varepsilon_y = \sigma_y/E$，結論完全不同——區分「等力（串聯）」與「等變形（並聯）」是此類題的核心分水嶺。
