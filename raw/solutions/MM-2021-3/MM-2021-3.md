# 考題編號：MM-2021-3

**主分類：** `MM-U2-2` 梁桿件斷面應力計算
**副分類：** 無
**分析法：** 彈性分析
**標籤：** `非均質梁` `楊氏模數漸變` `純彎曲` `平面截面假設` `積分求彎曲剛度` `應力分布`

---

## 1. 原始題目重述 (Problem Restatement)

**幾何條件：**
- 矩形截面懸臂梁，自由端承受彎矩 $M_0$（如圖三(a)）。
- 截面：寬 $b$，總高 $2h$；$y$ 軸以截面中心（中性軸候選位置）為原點，向上為正，$y \in [-h, +h]$（如圖三(b)）。

**材料條件：**
- 楊氏模數沿 $y$ 軸函數變化：

$$E(y) = E_0\left(\frac{y}{h}\right)^n, \quad n = 0,\ 2,\ 4,\ \ldots$$

- 應力-應變關係：$\sigma(y) = E(y)\,\varepsilon$

**求：** 梁之最大應力 $\sigma_{\max}$（以 $M_0, b, h, n$ 表示）。

![MM-2021-3 題目附圖：懸臂梁整體圖（圖三a）及矩形截面（圖三b），y 軸從 -h 至 +h，寬 b，E(y) 在截面內漸變](MM-2021-3-fig-1.png)

*圖說：懸臂梁長 L，右端自由端施加純彎矩 M₀；截面寬 b，高 2h，y=0 為截面幾何中心，y 向上正，範圍 [-h, +h]。楊氏模數 E(y)=E₀(y/h)ⁿ，n 為偶數（0, 2, 4…），E 在 y=0 為 E₀（n=0）或 0（n=2,4…），在 y=±h 均等於 E₀。*

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**「平面截面假設」（Bernoulli-Euler 假設）依然成立 → $\varepsilon = \kappa y$；但材料模數隨 $y$ 變化，彎曲剛度不能直接用 $EI$，必須**積分求有效彎曲剛度** $\int E(y)\,y^2\,dA$，再由力矩平衡反推曲率 $\kappa$，最後計算應力。

**出題者意圖：**
1. 測試非均質梁的基本推導流程（不能直接套 $\sigma = My/I$）。
2. 考查積分技巧（$\int_{-h}^{h} y^{n+2}\,dy$ 利用偶函數性質化簡）。
3. 結果以 $n$ 為參數表示，驗算 $n=0$ 是否還原均質矩形梁結果。

**陷阱提示：**
- 切勿直接用 $\sigma = My/I$（只適用均質材料，$I$ 為純幾何量）。
- $n$ 為偶數 → $E(y)$ 是偶函數，$E(y)\cdot y$ 是奇函數 → 力平衡積分自動為零，**中性軸在 $y=0$**，無需另外求。
- 最大應力位置：需確認 $\sigma(y)$ 的單調性，確認極值確在 $y=\pm h$。

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

1. **確認中性軸位置**：力平衡 $\int E(y)\,\varepsilon\,dA = 0$，利用奇函數積分=0，確認 $y=0$ 為中性軸。
2. **平面截面假設**：$\varepsilon(y) = \kappa y$（$\kappa$ 為待求曲率）。
3. **力矩平衡**：$M_0 = \int E(y)\,\kappa\,y \cdot y\,b\,dy$，積分出有效彎曲剛度，求 $\kappa$。
4. **應力公式**：$\sigma(y) = E(y)\,\kappa\,y$，化簡，找最大值位置，代入求 $\sigma_{\max}$。

**關鍵陷阱：**

| # | 陷阱 | 應對方法 |
|---|------|---------|
| 1 | 直接套均質梁公式 $\sigma=My/I$ | 此梁 E 非均勻，必須用 $\int E(y)y^2 b\,dy$ 作為有效剛度 |
| 2 | 誤以為中性軸需要重新求解 | $n$ 為偶數 → $E(y)$ 偶函數，$E(y)y$ 奇函數，$\int_{-h}^h E(y)y\,b\,dy = 0$，中性軸自動在 $y=0$ |
| 3 | 積分 $\int_{-h}^h y^{n+2}dy$ 沒用偶函數化簡 | $y^{n+2}$ 偶函數（$n+2$ 偶數）→ $= 2\int_0^h y^{n+2}dy$，簡化計算 |
| 4 | 誤認最大應力在 $y=0$ 或中間某處 | $d\sigma/dy \propto y^n \geq 0$ → $\sigma(y)$ 對 $y>0$ 單調遞增，最大在 $y=h$ |

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`以 M₀, b, h, n 表示非均質矩形梁在純彎矩下的最大正向應力 σ_max`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1（平面截面）：} \varepsilon(y) = \kappa\, y$$

$$\text{Step 2（應力）：} \sigma(y) = E(y)\,\varepsilon = E_0\!\left(\frac{y}{h}\right)^{\!n}\kappa\, y = \frac{E_0\kappa}{h^n}\,y^{n+1}$$

$$\text{Step 3（彎矩平衡）：} M_0 = \int_{-h}^{h} \sigma(y)\cdot y\cdot b\,dy = \frac{E_0\kappa b}{h^n}\int_{-h}^{h} y^{n+2}\,dy = \frac{E_0\kappa b}{h^n}\cdot\frac{2h^{n+3}}{n+3}$$

$$\text{Step 4（求曲率）：} \boxed{\kappa} = \frac{M_0(n+3)}{2E_0 b h^3}$$

$$\text{Step 5（最大應力）：} \sigma_{\max} = \sigma(y=h) = \frac{E_0\boxed{\kappa}}{h^n}\cdot h^{n+1} = E_0\boxed{\kappa}\,h = \frac{M_0(n+3)}{2bh^2}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

> 📊 互動圖（截面應力分布）：`MM-2021-3-section-viz.html`

### Step 1：確認中性軸位置

設 $y = 0$ 為中性軸（截面幾何中心）。驗證力平衡：

$$N = \int_{-h}^{h} \sigma(y)\,b\,dy = \int_{-h}^{h} E(y)\,\kappa\,y\,b\,dy = \frac{E_0\kappa b}{h^n}\int_{-h}^{h} y^{n+1}\,dy$$

因 $n$ 為偶數，$n+1$ 為**奇數**，$y^{n+1}$ 是奇函數：

$$\int_{-h}^{h} y^{n+1}\,dy = 0$$

$$\therefore N = 0 \quad \checkmark$$

> *策略註解：奇偶性論證直接排除中性軸偏移，無需建立額外方程式求解中性軸。*

### Step 2：建立應力表達式

由平面截面假設，截面應變：

$$\varepsilon(y) = \kappa\,y$$

（$\kappa$ 為彎曲曲率，正值表示上緣受壓、下緣受拉的方向，或反之，依 $M_0$ 方向定義）

結合 $\sigma(y) = E(y)\,\varepsilon$：

$$\sigma(y) = E_0\!\left(\frac{y}{h}\right)^{\!n}\cdot\kappa\,y = \frac{E_0\kappa}{h^n}\,y^{n+1}$$

### Step 3：力矩平衡求曲率 κ

$$M_0 = \int_{-h}^{h}\sigma(y)\cdot y\cdot b\,dy = \frac{E_0\kappa b}{h^n}\int_{-h}^{h}y^{n+2}\,dy$$

因 $n+2$ 為偶數，$y^{n+2}$ 是偶函數：

$$\int_{-h}^{h}y^{n+2}\,dy = 2\int_{0}^{h}y^{n+2}\,dy = 2\cdot\frac{h^{n+3}}{n+3}$$

代入：

$$M_0 = \frac{E_0\kappa b}{h^n}\cdot\frac{2h^{n+3}}{n+3} = E_0\kappa b\cdot\frac{2h^3}{n+3}$$

解出曲率：

$$\boxed{\kappa = \frac{M_0(n+3)}{2E_0 b h^3}}$$

> *這就是非均質梁的等效彎曲剛度：$EI_{eff} = E_0 b \cdot 2h^3/(n+3)$，對均質梁（n=0）退化為 $E_0 \cdot b(2h)^3/12 = E_0 I$。*

### Step 4：找最大應力位置

$$\sigma(y) = \frac{E_0\kappa}{h^n}\,y^{n+1}$$

對 $y$ 微分：

$$\frac{d\sigma}{dy} = \frac{E_0\kappa}{h^n}\,(n+1)\,y^{n}$$

因 $n$ 為偶數，$y^n \geq 0$ 恆成立（$\kappa > 0$ 且 $E_0 > 0$），所以：

$$\frac{d\sigma}{dy} \geq 0 \quad \text{對所有 } y$$

$\sigma(y)$ 關於 $y$ **單調遞增**。最大正應力在 $y = +h$（頂緣），最大負應力在 $y = -h$（底緣）。

### Step 5：計算最大應力

$$\sigma_{\max} = \sigma(y = h) = \frac{E_0\kappa}{h^n}\,h^{n+1} = E_0\,\kappa\,h$$

代入 $\kappa$：

$$\sigma_{\max} = E_0 \cdot \frac{M_0(n+3)}{2E_0 b h^3} \cdot h$$

$$\boxed{\sigma_{\max} = \frac{(n+3)\,M_0}{2\,b\,h^2}}$$

（最大壓應力 $\sigma_{\min} = -\sigma_{\max}$，發生在 $y = -h$）

### 驗算：n = 0（均質矩形梁）

$$\sigma_{\max}\big|_{n=0} = \frac{3M_0}{2bh^2}$$

均質矩形梁（高 $2h$，寬 $b$）標準公式：

$$\sigma_{\max} = \frac{M_0}{Z},\quad Z = \frac{b(2h)^2}{6} = \frac{2bh^2}{3}$$

$$\sigma_{\max} = \frac{M_0}{\frac{2bh^2}{3}} = \frac{3M_0}{2bh^2} \quad \checkmark$$

完全一致，驗算通過。

### 各 n 值結果比較

| $n$ | $\sigma_{\max}$ |
|-----|----------------|
| 0（均質） | $3M_0/(2bh^2)$ |
| 2 | $5M_0/(2bh^2)$ |
| 4 | $7M_0/(2bh^2)$ |
| $n$（偶數） | $(n+3)M_0/(2bh^2)$ |

隨 $n$ 增大，最大應力增大（中性軸附近材料剛度趨近於零，外緣必須承擔更高應力）。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 E(y)=0 在 y=0 的問題（n≥2 時）

當 $n \geq 2$，$E(0) = 0$，中性軸材料模數為零。物理上代表中性軸附近材料極軟（如多孔泡棉）、幾乎不傳遞應力。但此時應力公式 $\sigma(0) = 0$ 也成立（中性軸處應力為零），所以沒有矛盾。

### 5.2 有效彎曲剛度的物理意義

$$EI_{eff} = \int_{-h}^{h} E(y)\,y^2\,b\,dy = \frac{2E_0 b h^3}{n+3}$$

對均質梁（n=0）：$EI_{eff} = \frac{2E_0 b h^3}{3} = E_0 \cdot \frac{b(2h)^3}{12} = E_0 I$ ✓

$n$ 越大，$EI_{eff}$ 越小，梁越「軟」（在相同 $M_0$ 下曲率 $\kappa$ 越大），但最外緣恆有 $E = E_0$，所以應力反而越大。

### 5.3 考場建議

- 重點：整題推導唯一需要數學技巧的地方是積分 $\int_{-h}^h y^{n+2} dy$，善用偶函數立即折半，不要暴力展開。
- 答案 $\sigma_{\max} = (n+3)M_0/(2bh^2)$ 格式簡潔，記得用 $n=0$ 驗算均質梁結果。
- 若題目改問「中性軸位置」，直接用奇函數積分論證，無需計算，即可得 $\bar{y} = 0$。
