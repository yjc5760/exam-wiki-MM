# 考題編號：MM-2012-4

**主分類：** `MM-U4-2` 梁之彈塑性分析  
**副分類：**（無）  
**分析法：** 塑性分析（極限分析，上界+下界雙重驗證）  
**標籤：** `完全彈塑性` `固定-鉸接梁` `均布載重極限值` `塑性鉸` `機構法` `虛功原理` `靜力法` `塑性彎矩Mp` `矩形斷面`

---

## 1. 原始題目重述 (Problem Restatement)

有一均質梁承受均佈載重，**一端為固定支承（A），另一端為鉸接支承（B）**。該梁長度為 **L**；斷面為矩形，寬度為 **b**，深度為 **h**；材料性質為**完全彈塑性**，降伏應力為 **σᵧ**。

試求**均佈載重之極限值**（無需附圖，幾何條件由題目文字完整給定）。（25 分）

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
1. **靜不定度**：固定-鉸接梁（propped cantilever）靜不定度 = 1，需 **2 個塑性鉸**形成機構
2. **塑性彎矩 Mₚ**：矩形斷面全截面降伏時的彎矩上限，$M_p = \sigma_y \cdot bh^2/4$
3. **極限分析（上界定理）**：假設機構模式，由外虛功 = 內虛功求極限載重
4. **最不利機構**：在各可能的塑性鉸位置中，使極限載重最小的那個為真實機構

**出題者意圖：**
- 考驗「機構法（虛功法）」的完整流程，包含最小化步驟
- 考驗矩形斷面塑性截面模數 $Z_p = bh^2/4$ 的計算
- 禁止計算器 → 結果含 $\sqrt{2}$，考代數整理與有理化能力

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**步驟化作戰計畫：**
1. 計算矩形斷面塑性彎矩 Mₚ
2. 由彈性解確認固定端先達到 Mₚ（第一個塑性鉸位置）
3. 假設第二個塑性鉸在距A端 c 處（c 待定）
4. 用虛功原理建立 wu 與 c 的關係式
5. 對 c 求極值（$dw_u/dc = 0$），找最不利機構
6. 代回求 wu，有理化整理
7. 用靜力法（下界）交叉驗算

**關鍵陷阱：**

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ⚠ Zp vs Ze | 矩形斷面：$Z_e=bh^2/6$（彈性），$Z_p=bh^2/4$（塑性），兩者差形狀因子 f=1.5 | 極限分析用 $M_p = \sigma_y \cdot Z_p$ |
| ⚠ 靜不定度 1 → 需 2 個塑性鉸 | 一度超靜定梁：原有 1 個鉸（B端）不計，需再形成 2 個塑性鉸 | |
| ⚠ 第二個塑性鉸位置 c 不在跨中 | 因梁不對稱（固定-鉸），最不利機構的 c 需由最小化決定 | $c = (2-\sqrt{2})L$ |
| ⚠ A端塑性鉸的旋轉量 | A端「固定端」的塑性鉸旋轉量 = $\delta/c$（非 $\delta/c + \delta/(L-c)$） | 仔細區分各鉸的轉角 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
求均布載重的極限值 $w_u$（以 $\sigma_y, b, h, L$ 表示）

### 本題關鍵公式

$$M_p = \sigma_y \cdot \frac{bh^2}{4}$$

$$\text{外虛功} = \frac{wL\delta}{2}, \quad \text{內虛功} = M_p\left(\frac{\delta}{c} + \frac{\delta}{c} + \frac{\delta}{L-c}\right) = M_p\delta\cdot\frac{2L-c}{c(L-c)}$$

$$w = \frac{2M_p(2L-c)}{L\cdot c(L-c)} \xrightarrow{\min_c} c^* = (2-\sqrt{2})L$$

$$\boxed{w_u = 2(1+\sqrt{2})^2\frac{M_p}{L^2} = (6+4\sqrt{2})\frac{M_p}{L^2}}$$

### L1：題目直接給定

|------|------|
| $L$ | 梁長 |
| $b$ | 矩形斷面寬 |

| $\sigma_y$ | 降伏應力 |
| 材料 | 完全彈塑性（$E_2=0$）|
| 支承 | A 端固定，B 端鉸接 |
| 載重 | 均布載重 $w$（向下）|

### L2：需知識點推導

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $Z_p$（塑性截面模數）| $bh^2/4$（矩形斷面）| |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 步驟 1：矩形斷面的塑性彎矩 Mₚ

矩形斷面（寬 b，高 h）在完全彈塑性材料下，全截面降伏時的彎矩（塑性彎矩）：

$$M_p = \sigma_y \cdot Z_p = \sigma_y \cdot \frac{bh^2}{4}$$

> 由定義：$Z_p = \int |y|\,dA = 2\cdot\int_0^{h/2}y\cdot b\,dy = 2b\cdot\frac{(h/2)^2}{2} = \frac{bh^2}{4}$

### 步驟 2：確認固定端為第一個塑性鉸（彈性解分析）

固定-鉸接梁（A固 B鉸）受均布載重 $w$ 的彈性解（以疊加法，去掉B支承求超靜定力）：

$$V_B = \frac{3wL}{8}, \quad V_A = \frac{5wL}{8}, \quad M_A = -\frac{wL^2}{8}$$

> **諧合條件**：去掉 B 支承，A 端懸臂梁自由端（B 端）撓度為零：
> $$\frac{wL^4}{8EI} = \frac{V_B L^3}{3EI} \implies V_B = \frac{3wL}{8}$$

**跨中最大正彎矩（剪力為零處）：**

$$M(x) = -\frac{wL^2}{8} + \frac{5wL}{8}x - \frac{wx^2}{2}$$

$$\frac{dM}{dx} = 0 \implies x^* = \frac{5L}{8}, \quad M_{max}^+ = \frac{9wL^2}{128}$$

**彎矩絕對值比較：**

$$|M_A| = \frac{wL^2}{8} = \frac{16wL^2}{128} > M_{max}^+ = \frac{9wL^2}{128}$$

→ **固定端 A（x=0）先達到 Mₚ，形成第一個塑性鉸。**

### 步驟 3：假設機構模式（虛功原理）

超靜定度 = 1，需 **2 個塑性鉸**形成機構：

- 第 1 個塑性鉸：**A 端（固定端）**，x = 0
- 第 2 個塑性鉸：**位置 x = c**（待定，$0 < c < L$）

**機構示意：**

```
A(塑性鉸)         c(塑性鉸)               B(鉸接端)
├─────────────────┼───────────────────────┤
|← ─ ─ ─ c ─ ─ ─→|← ─ ─ (L-c) ─ ─ ─ →|
```

施加虛位移：令 x=c 處虛位移 $\delta$（向下）

**幾何關係（小變形）：**

- 左段（A~c，長 c）：繞 A 端旋轉，轉角 $\theta_1 = \delta/c$
- 右段（c~B，長 L-c）：繞 B 端旋轉，轉角 $\theta_2 = \delta/(L-c)$
- A 端塑性鉸旋轉量 = $\theta_1 = \delta/c$
- c 處塑性鉸旋轉量 = $\theta_1 + \theta_2 = \dfrac{\delta}{c} + \dfrac{\delta}{L-c}$

### 步驟 4：建立虛功方程

**外虛功（均布載重 $w$ 做的功）：**

$$W_{ext} = w\cdot\underbrace{\frac{\delta}{2}\cdot c}_{\text{左段梯形虛位移面積}} + w\cdot\underbrace{\frac{\delta}{2}\cdot(L-c)}_{\text{右段梯形虛位移面積}} = \frac{wL\delta}{2}$$

**內虛功（兩個塑性鉸消耗的能量）：**

$$W_{int} = M_p\cdot\theta_1 + M_p\cdot(\theta_1+\theta_2) = M_p\left(\frac{\delta}{c} + \frac{\delta}{c} + \frac{\delta}{L-c}\right) = M_p\delta\cdot\frac{2L-c}{c(L-c)}$$

**令 $W_{ext} = W_{int}$（消去 $\delta$）：**

$$\frac{wL}{2} = M_p\cdot\frac{2L-c}{c(L-c)}$$

$$w(c) = \frac{2M_p(2L-c)}{L\cdot c(L-c)}$$

### 步驟 5：最小化求最不利機構位置

$w(c)$ 為 $c$ 的函數，取最小值（上界定理：最小的上界等於真實極限值）：

$$\frac{dw}{dc} = 0$$

對 $f(c) = \dfrac{2L-c}{c(L-c)}$ 求導：

令 $u = 2L-c$，$v = c(L-c) = cL - c^2$：

$$f' = \frac{(-1)\cdot v - u\cdot(L-2c)}{v^2} = 0 \implies -(cL-c^2) = (2L-c)(L-2c)$$

$$-cL + c^2 = 2L^2 - 4Lc - cL + 2c^2 \implies c^2 - 2L^2 + 4Lc - 2c^2 = 0$$

$$-c^2 + 4Lc - 2L^2 = 0 \implies c^2 - 4Lc + 2L^2 = 0$$

$$c = \frac{4L \pm \sqrt{16L^2 - 8L^2}}{2} = \frac{4L \pm 2\sqrt{2}L}{2} = L(2 \pm \sqrt{2})$$

取 $c < L$（有效解）：

$$\boxed{c^* = (2-\sqrt{2})L \approx 0.586L}$$

### 步驟 6：代回求極限載重 $w_u$

代入 $c^* = (2-\sqrt{2})L$：

$$2L - c^* = 2L - (2-\sqrt{2})L = \sqrt{2}\,L$$

$$L - c^* = L - (2-\sqrt{2})L = (\sqrt{2}-1)L$$

$$w_u = \frac{2M_p \cdot \sqrt{2}\,L}{L\cdot(2-\sqrt{2})L\cdot(\sqrt{2}-1)L} = \frac{2\sqrt{2}\,M_p}{L^2(2-\sqrt{2})(\sqrt{2}-1)}$$

**化簡分母：**

$$(2-\sqrt{2})(\sqrt{2}-1) = 2\sqrt{2} - 2 - 2 + \sqrt{2} = 3\sqrt{2} - 4$$

**有理化（乘以 $\dfrac{3\sqrt{2}+4}{3\sqrt{2}+4}$）：**

$$\text{分母：}\;(3\sqrt{2}-4)(3\sqrt{2}+4) = 18 - 16 = 2$$

$$\text{分子：}\;2\sqrt{2}(3\sqrt{2}+4) = 12 + 8\sqrt{2}$$

$$w_u = \frac{12 + 8\sqrt{2}}{2L^2}M_p = (6+4\sqrt{2})\frac{M_p}{L^2}$$

$$\boxed{w_u = (6+4\sqrt{2})\frac{M_p}{L^2} = 2(1+\sqrt{2})^2\frac{M_p}{L^2} \approx 11.66\frac{M_p}{L^2}}$$

代入 $M_p = \sigma_y bh^2/4$：

$$\boxed{w_u = \frac{(6+4\sqrt{2})\,\sigma_y bh^2}{4L^2}}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 靜力法交叉驗算（下界定理）

A 端形成塑性鉸後（彎矩 = $M_p$，固定），結構退化為兩端鉸接梁，繼續加載：

$$V_A^{new} = \frac{M_p}{L} + \frac{wL}{2}, \quad V_B = \frac{wL}{2} - \frac{M_p}{L}$$

剪力為零處（第二個塑性鉸位置）：

$$x^* = \frac{V_A^{new}}{w} = \frac{M_p}{wL} + \frac{L}{2}$$

在 $x^*$ 處令 $M(x^*) = M_p$（正彎矩達到極限）：

$$M_p = -M_p + V_A^{new} \cdot x^* - \frac{w(x^*)^2}{2}$$

解此方程得：

$$w_u = 2(2\sqrt{2}+3)\frac{M_p}{L^2} = (6+4\sqrt{2})\frac{M_p}{L^2} \checkmark$$

與虛功法結果完全一致，證明此為真實極限載重（上界 = 下界）。

### 5.2 塑性形狀因子

矩形斷面的形狀因子（shape factor）：

$$f = \frac{M_p}{M_y} = \frac{Z_p}{Z_e} = \frac{bh^2/4}{bh^2/6} = \frac{3}{2} = 1.5$$

即彈性極限載重 $w_y$（最外緣剛好降伏）：

$$|M_A| = w_y L^2/8 = M_y = M_p/1.5 \implies w_y = \frac{8M_p}{1.5L^2} = \frac{16M_p}{3L^2} \approx \frac{5.33M_p}{L^2}$$

超載係數：$w_u/w_y = (6+4\sqrt{2})/(16/3) = 3(6+4\sqrt{2})/16 \approx 2.19$

### 5.3 機構分析的物理意義

```
彎矩圖（極限狀態）：

A ─────────── c* ─────────────── B
│     Mₚ(負)  │     Mₚ(正)      │
│←─(2-√2)L──→│←── (√2-1)L ──→│

M(0) = -Mₚ（A端負彎矩，使上側受拉）
M(c*)= +Mₚ（跨中正彎矩，使下側受拉）
M(L) = 0（B端鉸接）
```

### 5.4 常見錯誤：第二個塑性鉸在 L/2

若誤以為對稱條件在 L/2，代入：

$$w_u^{wrong}(c=L/2) = \frac{2M_p(2L-L/2)}{L\cdot(L/2)\cdot(L/2)} = \frac{2M_p\cdot 3L/2}{L^3/4} = \frac{12M_p}{L^2} > w_u^{exact}$$

此為**高估**（上界定理的另一個上界），不是最小的上界，故不是真實答案。
