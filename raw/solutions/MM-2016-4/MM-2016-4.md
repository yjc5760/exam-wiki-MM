**分析法：** 彈性分析
**標籤：** `三軸應力` `廣義虎克定律` `彈性模數` `柏松比` `主應力` `最大剪應力` `Mohr圓` `平面應力`

## 1. 原始題目重述 (Problem Restatement)

一**均勻厚度之正方形板**受到三軸應力狀態：

$$\sigma_x = 40\ \text{MPa},\quad \sigma_y = -20\ \text{MPa},\quad \sigma_z = 0\ \text{MPa}$$

所產生之應變為：

$$\varepsilon_x = 23 \times 10^{-5},\quad \varepsilon_y = -16 \times 10^{-5}$$

**求：**
1. 材料之彈性係數 $E$（彈性模數）
2. 材料之柏松比 $\nu$（Poisson's ratio）
3. $xy$ 面之最大剪應力 $(\tau_{xy})_{max}$
4. $yz$ 面之最大剪應力 $(\tau_{yz})_{max}$
5. $xz$ 面之最大剪應力 $(\tau_{xz})_{max}$

![MM-2016-4 題目附圖：正方形板受雙軸應力，σ_x=40MPa向右（外側），σ_y=-20MPa向下壓（合力向下）；x、y軸標示於板面中央](MM-2016-4-fig-1.png)

*圖說：均勻厚度正方形板，受雙向正向應力，$\sigma_x = 40\ \text{MPa}$（拉）沿 x 方向，$\sigma_y = -20\ \text{MPa}$（壓）沿 y 方向，$\sigma_z = 0$（板厚方向無應力）；給定對應應變 $\varepsilon_x$ 和 $\varepsilon_y$，求材料常數及各面最大剪應力。*

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

### 核心觀念

**廣義虎克定律（三軸）：**

$$\varepsilon_x = \frac{1}{E}[\sigma_x - \nu(\sigma_y + \sigma_z)]$$

$$\varepsilon_y = \frac{1}{E}[\sigma_y - \nu(\sigma_x + \sigma_z)]$$

$$\varepsilon_z = \frac{1}{E}[\sigma_z - \nu(\sigma_x + \sigma_y)]$$

**求 $E$ 和 $\nu$：** 代入已知 $\sigma_x, \sigma_y, \sigma_z, \varepsilon_x, \varepsilon_y$，解兩個方程的兩個未知數。

**最大剪應力（各面）：**

各面的最大剪應力由**主應力**決定：

$$(\tau_{xy})_{max} = \frac{|\sigma_x - \sigma_y|}{2}\ \text{（在 xy 平面內，需用 Mohr 圓）}$$

等等，更正確的說法：

- $xy$ 面的最大剪應力（在 xy 平面內旋轉截面）= $\frac{1}{2}\sqrt{(\sigma_x - \sigma_y)^2 + 4\tau_{xy}^2}$（本題 $\tau_{xy} = 0$）
- $yz$ 面最大剪應力（在 yz 平面內）= $\frac{|\sigma_y - \sigma_z|}{2}$
- $xz$ 面最大剪應力（在 xz 平面內）= $\frac{|\sigma_x - \sigma_z|}{2}$

**重要澄清：** 題目問的是各「面」的最大剪應力，即：
- $(\tau_{xy})_{max}$：在含 z 軸的平面上旋轉（在 xy 平面上的最大剪應力），$= |\sigma_1 - \sigma_2|/2$（其中 $\sigma_1, \sigma_2$ 為 xy 平面的主應力）
- $(\tau_{yz})_{max}$：在含 x 軸的平面上旋轉，$= |\sigma_2' - \sigma_3'|/2$（$\sigma_2', \sigma_3'$ 為 yz 平面的主應力）
- $(\tau_{xz})_{max}$：在含 y 軸的平面上旋轉，$= |\sigma_1'' - \sigma_3''|/2$

實際上，由於 $\sigma_x, \sigma_y, \sigma_z$ 就是主應力（無剪應力），各面最大剪應力為：

$$(\tau_{xy})_{max} = \frac{|\sigma_x - \sigma_y|}{2}$$

$$(\tau_{yz})_{max} = \frac{|\sigma_y - \sigma_z|}{2}$$

$$(\tau_{xz})_{max} = \frac{|\sigma_x - \sigma_z|}{2}$$

### 出題者意圖
- 測驗**廣義虎克定律**（聯立方程求 $E$ 和 $\nu$）
- 測驗三軸應力狀態下**各面最大剪應力**的計算（用主應力差/2）
- 測驗 $\sigma_z = 0$（平面應力）的特殊情況

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

### 作戰計畫
```
Step 1：寫出廣義虎克定律（x, y 兩方程）
Step 2：聯立求解 E 和 ν
Step 3：確認 σ_x, σ_y, σ_z 就是三個主應力（無剪切分量）
Step 4：各面最大剪應力 = 兩個主應力差的絕對值 / 2
```

### 關鍵陷阱

**陷阱 1：三個「面」的最大剪應力各不同**
> - $(\tau_{xy})_{max}$：在 xy 平面旋轉，用 $\sigma_x$ 和 $\sigma_y$（$\sigma_z$ 不參與）
> - $(\tau_{yz})_{max}$：在 yz 平面旋轉，用 $\sigma_y$ 和 $\sigma_z$
> - $(\tau_{xz})_{max}$：在 xz 平面旋轉，用 $\sigma_x$ 和 $\sigma_z$

**陷阱 2：$\sigma_z = 0$ 但 $\varepsilon_z \ne 0$**
> $\sigma_z = 0$ 是應力條件，但由於 Poisson 效應，$\varepsilon_z = -\nu(\sigma_x + \sigma_y)/E \ne 0$（需計算但題目未要求）

**陷阱 3：已知 $\sigma_x, \sigma_y, \sigma_z$ 為主應力（無剪應力給定）**
> 本題沒有剪應力 $\tau_{xy}$ 等，故 $x, y, z$ 方向即為主應力方向，主應力 $\sigma_1 = \sigma_x = 40$，$\sigma_2 = \sigma_y = -20$，$\sigma_3 = \sigma_z = 0$（MPa）。

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標
`求 E、ν，以及三個面的最大剪應力`

### 本題關鍵公式

$$\varepsilon_x = \frac{\sigma_x - \nu\sigma_y}{E} = \frac{40 - \nu(-20)}{E} = \frac{40 + 20\nu}{E} = 23 \times 10^{-5}\ \cdots\ (1)$$

$$\varepsilon_y = \frac{\sigma_y - \nu\sigma_x}{E} = \frac{-20 - \nu(40)}{E} = \frac{-20 - 40\nu}{E} = -16 \times 10^{-5}\ \cdots\ (2)$$

$$(\tau_{xy})_{max} = \frac{|\sigma_x - \sigma_y|}{2} = \frac{|40-(-20)|}{2} = 30\ \text{MPa}$$

$$(\tau_{yz})_{max} = \frac{|\sigma_y - \sigma_z|}{2} = \frac{|-20-0|}{2} = 10\ \text{MPa}$$

$$(\tau_{xz})_{max} = \frac{|\sigma_x - \sigma_z|}{2} = \frac{|40-0|}{2} = 20\ \text{MPa}$$

### L1：題目直接給定

| 符號 | 數值 |
|------|------|
| $\sigma_x$ | $40\ \text{MPa}$ |
| $\sigma_y$ | $-20\ \text{MPa}$ |
| $\sigma_z$ | $0\ \text{MPa}$ |
| $\varepsilon_x$ | $23 \times 10^{-5}$ |
| $\varepsilon_y$ | $-16 \times 10^{-5}$ |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：廣義虎克定律建立方程式

代入 $\sigma_z = 0$（平面應力）：

$$\varepsilon_x = \frac{\sigma_x - \nu\sigma_y}{E} \Rightarrow \frac{40 + 20\nu}{E} = 23 \times 10^{-5} \quad \cdots (1)$$

$$\varepsilon_y = \frac{\sigma_y - \nu\sigma_x}{E} \Rightarrow \frac{-20 - 40\nu}{E} = -16 \times 10^{-5} \quad \cdots (2)$$

---

### Step 2：聯立求解 $E$ 和 $\nu$

由 (1)：$(40 + 20\nu) = 23 \times 10^{-5} \times E$

由 (2)：$(-20 - 40\nu) = -16 \times 10^{-5} \times E$，即 $(20 + 40\nu) = 16 \times 10^{-5} \times E$

**消去 $E$（兩式相除）：**

$$\frac{40 + 20\nu}{20 + 40\nu} = \frac{23 \times 10^{-5}}{16 \times 10^{-5}} = \frac{23}{16}$$

交叉相乘：

$$16(40 + 20\nu) = 23(20 + 40\nu)$$

$$640 + 320\nu = 460 + 920\nu$$

$$640 - 460 = 920\nu - 320\nu$$

$$180 = 600\nu$$

$$\boxed{\nu = \frac{180}{600} = 0.3}$$

代回方程 (1)：

$$\frac{40 + 20 \times 0.3}{E} = 23 \times 10^{-5}$$

$$\frac{40 + 6}{E} = 23 \times 10^{-5}$$

$$\frac{46}{E} = 23 \times 10^{-5}$$

$$E = \frac{46}{23 \times 10^{-5}} = \frac{46}{23} \times 10^5 = 2 \times 10^5\ \text{MPa}$$

$$\boxed{E = 200{,}000\ \text{MPa} = 200\ \text{GPa}}$$

**驗算（代入方程 (2)）：**

$$\frac{-20 - 40 \times 0.3}{200{,}000} = \frac{-20 - 12}{200{,}000} = \frac{-32}{200{,}000} = -16 \times 10^{-5}\ ✓$$

---

### Step 3：確認主應力

由於 $\sigma_x, \sigma_y, \sigma_z$ 已知，且無剪應力分量（$\tau_{xy} = \tau_{yz} = \tau_{xz} = 0$），故三個主應力為：

$$\sigma_1 = \sigma_x = +40\ \text{MPa},\quad \sigma_2 = \sigma_z = 0\ \text{MPa},\quad \sigma_3 = \sigma_y = -20\ \text{MPa}$$

（按大小排列：$\sigma_1 > \sigma_2 > \sigma_3$）

---

### Step 4：各面最大剪應力

各「面」的最大剪應力 = 該平面兩個主應力的差的絕對值 / 2：

**$xy$ 面最大剪應力（在 xy 平面旋轉，涉及 $\sigma_x$ 和 $\sigma_y$）：**

$$(\tau_{xy})_{max} = \frac{|\sigma_x - \sigma_y|}{2} = \frac{|40 - (-20)|}{2} = \frac{60}{2}$$

$$\boxed{(\tau_{xy})_{max} = 30\ \text{MPa}}$$

**$yz$ 面最大剪應力（在 yz 平面旋轉，涉及 $\sigma_y$ 和 $\sigma_z$）：**

$$(\tau_{yz})_{max} = \frac{|\sigma_y - \sigma_z|}{2} = \frac{|-20 - 0|}{2} = \frac{20}{2}$$

$$\boxed{(\tau_{yz})_{max} = 10\ \text{MPa}}$$

**$xz$ 面最大剪應力（在 xz 平面旋轉，涉及 $\sigma_x$ 和 $\sigma_z$）：**

$$(\tau_{xz})_{max} = \frac{|\sigma_x - \sigma_z|}{2} = \frac{|40 - 0|}{2} = \frac{40}{2}$$

$$\boxed{(\tau_{xz})_{max} = 20\ \text{MPa}}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 「面」最大剪應力 vs 「絕對」最大剪應力

本題問三個「面」的最大剪應力，各為特定平面內旋轉找到的最大值。

**絕對最大剪應力**（在三維空間中所有方向的最大值）：

$$\tau_{abs,max} = \frac{\sigma_1 - \sigma_3}{2} = \frac{40 - (-20)}{2} = 30\ \text{MPa}$$

這恰好等於 $(\tau_{xy})_{max}$，因為 $\sigma_x$ 和 $\sigma_y$ 是最大和最小主應力。

### 5.2 $E$ 和 $\nu$ 的結果驗算

$E = 200\ \text{GPa}$，$\nu = 0.3$ 正是 **A-36 鋼**的典型材料常數！這是出題者刻意設計的（結果整齊且有實際意義）。

### 5.3 $\varepsilon_z$ 的計算（雖未問）

$$\varepsilon_z = \frac{\sigma_z - \nu(\sigma_x + \sigma_y)}{E} = \frac{0 - 0.3 \times (40 - 20)}{200{,}000} = \frac{-6}{200{,}000} = -3 \times 10^{-5}$$

板厚方向收縮（Poisson 效應），雖然 $\sigma_z = 0$，板厚仍有微量變化。

### 5.4 廣義虎克定律的矩陣形式

三軸應力狀態的廣義虎克定律可以矩陣形式表達：

$$\begin{pmatrix} \varepsilon_x \\ \varepsilon_y \\ \varepsilon_z \end{pmatrix} = \frac{1}{E}\begin{pmatrix} 1 & -\nu & -\nu \\ -\nu & 1 & -\nu \\ -\nu & -\nu & 1 \end{pmatrix} \begin{pmatrix} \sigma_x \\ \sigma_y \\ \sigma_z \end{pmatrix}$$

本題為平面應力（$\sigma_z = 0$），矩陣第三列可忽略；但 $\varepsilon_z$ 不為零（Poisson 效應）。
