## 1. 原始題目重述 (Problem Restatement)

一打入土壤內的木樁如圖二所示，貫入土壤之長度為 **l**。沿樁軸單位長度的摩擦力為：

$$f = kx^2 \quad (k \text{ 為常數})$$

其中 $x$ 為自**底部（A 端）**向上量的距離。樁頂 B 點受垂直向下集中力 $F$；C 為土壤表面（$x = l$）。假設**樁自重及底部（A 端）的反力可忽略**。

**試繪出整根木樁的軸力分布圖。**（25 分）

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
1. **截面法**：取截面以下（或以上）為自由體，由靜力平衡求截面軸力
2. **整體靜力平衡**：確認 F 與總摩擦力的關係（$F = kl^3/3$）
3. **軸力圖形狀**：$N(x) \propto x^3$，從 $x=0$（A端，N=0）到 $x=l$（C端，N=-F），三次曲線

**出題者意圖：**
- 測驗「軸力等於截面以下（或以上）所有外力的代數和」的截面法
- 測驗非均勻分布外力（$f=kx^2$）的積分
- 繪製非線性（三次曲線）軸力圖的能力

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**步驟化作戰計畫：**
1. 整體靜力平衡，建立 F 與 k、l 的關係
2. 取貫入段任意截面 x（取截面以下自由體），求 N(x)
3. 確認邊界條件（x=0 和 x=l）
4. 繪製軸力圖（三次曲線，全段壓縮）

**關鍵陷阱：**

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
繪出軸力分布圖 $N(x)$（$x \in [0, l]$）

### 本題關鍵公式

$$\text{整體平衡：}\quad F = \int_0^l kx^2\,dx = \frac{kl^3}{3} \implies k = \frac{3F}{l^3}$$

$$\text{截面法（取截面以下）：}\quad N(x) = -\int_0^x kx'^2\,dx' = -\frac{kx^3}{3}$$

$$\boxed{N(x) = -F\!\left(\frac{x}{l}\right)^3}, \quad x \in [0, l]$$

**軸力圖：**

```
N
0 ─────────────────────────── x=0 (底部A，N=0)
│
│（三次曲線，單調遞減）
│
-F ────────────────────────── x=l (土面C，N=-F)
```

### L1：題目直接給定

| 符號 | 說明 |
|------|------|
| $l$ | 貫入深度 |
| $F$ | 頂部集中力（向下）|
| $f = kx^2$ | 單位長度摩擦力（向上）|
| 底部反力 | 忽略（=0）|
| 樁自重 | 忽略 |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 步驟 1：整體靜力平衡（確認 F 與 k 的關係）

對整根木樁（貫入段）取自由體，只有 F（向下）和摩擦力（向上）：

$$F = \int_0^l f\,dx = \int_0^l kx^2\,dx = \frac{kl^3}{3}$$

$$\boxed{k = \frac{3F}{l^3}}$$

> **策略說明：** 此關係式確認 k 的物理意義，且後續可用 $F$ 替代 $k$ 使答案更直觀。

### 步驟 2：截面法求軸力 N(x)

在貫入段距底部 $x$ 處取截面，對**截面以下**（長度 $x$，底部 A 端）部分取自由體：

- 底部反力 = 0（題目條件）
- 土壤摩擦力（向上）= $\int_0^x kx'^2\,dx' = kx^3/3$（向上）
- 截面軸力 $N(x)$（正=拉，使截面以下受拉向下）

**截面以下自由體的平衡（向上為正）：**

$$N(x) + \int_0^x kx'^2\,dx' = 0$$

$$N(x) + \frac{kx^3}{3} = 0 \implies N(x) = -\frac{kx^3}{3}$$

代入 $k = 3F/l^3$：

$$\boxed{N(x) = -\frac{F x^3}{l^3} = -F\!\left(\frac{x}{l}\right)^3}$$

### 步驟 3：邊界條件驗算

| 位置 | x | N(x) | 物理說明 |
|------|---|------|---------|
| 底部 A（自由端）| 0 | 0 | 底部反力=0 ✓ |
| 土面 C | l | $-F$ | 承受頂部集中力 F 的壓縮 ✓ |

**截面以上自由體交叉驗算（取 C→截面x 部分，含 F）：**

- F（向下）、截面以上摩擦力（向上）$= \int_x^l kx'^2 dx' = k(l^3-x^3)/3$
- 截面內力（對截面以上，向下為正）$= -N(x)$

$$-N(x) + k(l^3-x^3)/3 - F = 0 \implies -N(x) = F - k(l^3-x^3)/3 = kl^3/3 - k(l^3-x^3)/3 = kx^3/3$$

$$N(x) = -kx^3/3 \checkmark$$

### 步驟 4：軸力分布圖

$$N(x) = -F\!\left(\frac{x}{l}\right)^3, \quad x \in [0,\,l]$$

**圖形特徵：**
- 全段為**壓縮**（N < 0）
- $x=0$（底部 A）：$N = 0$，自由端
- $x=l$（土面 C）：$N = -F$，最大壓縮力
- 曲線形狀：三次曲線，在 $x=0$ 附近幾乎水平（斜率 $\propto x^2$），在 $x=l$ 附近斜率最陡

```
  x
  l ─── C ─────────────────────────●  N = -F
       │                         ╱
       │                       ╱
       │                     ╱  （三次曲線）
       │                  ╱
       │              ╱
       │         ╱
  0 ─── A ───●───────────────────────  N = 0
              0                    -F → N
```

**斜率（dN/dx = 分布載重的平衡）：**

$$\frac{dN}{dx} = -kx^2 = -f(x)$$

此為軸力微分方程 $dN/dx = -q(x)$（q 為分布力）的直接驗算 ✓

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 軸力微分關係

軸力分布的微分關係（對軸力桿件）：

$$\frac{dN}{dx} = -q(x)$$

其中 $q(x)$ 為單位長度分布力（向上為正）。本題 $q(x) = f(x) = kx^2$（向上），故：

$$\frac{dN}{dx} = -kx^2 \implies N(x) = -\frac{kx^3}{3} + C$$

邊界條件 $N(0) = 0$（底部自由）→ $C = 0$，得 $N(x) = -kx^3/3$ ✓

### 5.2 不同坐標原點的換算

若改以 x 從頂部 B 向下量（令 $\xi = l - x$，從 C 土面到 A 底部），則：

$$f(\xi) = k(l-\xi)^2$$

$$N(\xi) = -F + \int_0^\xi k(l-\xi')^2\,d\xi' = -F + k\left[l^2\xi - l\xi^2 + \xi^3/3\right]$$

代入 $k = 3F/l^3$ 後化簡：$N(0) = -F$（C 處），$N(l) = 0$（A 處）✓

### 5.3 物理意義：為何底部反力可忽略？

若樁底有硬層，底部反力 $R_A \neq 0$，則 $N(0) = -R_A$（底部壓縮）。  
本題假設底部為軟土（或純摩擦樁），所有的 F 由土壤側摩擦力承擔，底部支承 = 0。
