## §1 題目重述

懸臂梁 AB，A 端固定（左），B 端自由（右），長度 $L = 12\ \text{m}$。梁受三角形均布載重：固定端 A 處強度為零，自由端 B 處強度為 $q_0$（如圖四(a)）。

斷面為對稱 I 形（如圖四(b)）：

| 元件 | 尺寸 |
|------|------|
| 翼板（上、下各一） | 寬 $b_f = 130\ \text{mm}$，厚 $t_f = 20\ \text{mm}$ |
| 腹板 | 淨高 $h_w = 200\ \text{mm}$，厚 $t_w = 15\ \text{mm}$ |
| 斷面總高 | $H = h_w + 2t_f = 240\ \text{mm}$ |

已知容許應力：$\sigma_w = 80\ \text{MPa}$，$\tau_w = 60\ \text{MPa}$。

**求：**梁能承受之最大 $q_0$。

---

## §2 核心概念

- **三角形載重懸臂梁**：最大剪力與最大彎矩均發生在固定端 A（$x = 0$）。
- **I 形斷面**：最大彎曲應力在翼板外緣（$y = \pm c$），最大剪應力在腹板中性軸處。
- **容許應力設計**：分別對彎曲應力與剪應力設限，取兩者中較嚴格的 $q_0$。

---

## §3 解題策略

1. 計算斷面慣性矩 $I$、外緣距 $c$、中性軸一次矩 $Q_{\max}$
2. 推導固定端 A 的 $V_{\max}$ 與 $M_{\max}$（由積分法）
3. 彎曲應力檢核：$\sigma = M_{\max} c / I \leq \sigma_w$
4. 剪應力檢核：$\tau = V_{\max} Q_{\max} / (I\, t_w) \leq \tau_w$
5. 取最小許可 $q_0$

---

## §3.5 變數層次分析（VHA）

### L1 — 直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $L$ | $12\ \text{m} = 12{,}000\ \text{mm}$ | 梁長 |
| $q_0$ | 待求 | 三角形載重最大強度 |
| $b_f$ | $130\ \text{mm}$ | 翼板寬度 |
| $t_f$ | $20\ \text{mm}$ | 翼板厚度 |
| $h_w$ | $200\ \text{mm}$ | 腹板淨高 |
| $t_w$ | $15\ \text{mm}$ | 腹板厚度 |
| $\sigma_w$ | $80\ \text{MPa}$ | 容許彎曲應力 |
| $\tau_w$ | $60\ \text{MPa}$ | 容許剪應力 |

### L2 — 由公式推導

| 符號 | 推導來源 | 說明 |
|------|----------|------|
| $H = h_w + 2t_f$ | 幾何疊加 | 斷面總高 = 240 mm |
| $c = H/2$ | 對稱斷面中性軸 | 外緣距 = 120 mm |
| $I$ | 矩形扣除法 | 斷面慣性矩 |
| $Q_{\max}$ | 中性軸一次矩 | $= Q_\text{翼板} + Q_\text{腹板半高}$ |
| $V_{\max} = q_0 L/2$ | 三角形載重積分 | 固定端剪力 |
| $M_{\max} = q_0 L^2/3$ | 三角形載重積分 | 固定端彎矩 |

### L3 — 深層知識

- 三角形載重（自固定端 0 至自由端 $q_0$）的合力作用點在自由端 B 起算 $L/3$ 處，即距 A 端 $2L/3$ ——由此得 $M_A = (q_0 L/2) \times (2L/3) = q_0 L^2/3$。
- I 形斷面剪力流在腹板-翼板交界處不連續（廣度突變），最大剪應力一定在腹板中性軸（廣度最小、一次矩最大）。

## §4 詳細計算

### 4.1 斷面性質

**慣性矩（矩形扣除法）：**

$$I = \frac{b_f H^3 - (b_f - t_w)\,h_w^3}{12}$$

$$= \frac{130 \times 240^3 - (130 - 15) \times 200^3}{12}$$

$$= \frac{130 \times 13{,}824{,}000 - 115 \times 8{,}000{,}000}{12}$$

$$= \frac{1{,}797{,}120{,}000 - 920{,}000{,}000}{12} = \frac{877{,}120{,}000}{12}$$

$$\boxed{I = 73{,}093{,}333\ \text{mm}^4 \approx 73.09 \times 10^6\ \text{mm}^4}$$

**外緣距：**

$$c = \frac{H}{2} = \frac{240}{2} = 120\ \text{mm}$$

**中性軸一次矩 $Q_{\max}$（中性軸以上面積對中性軸之一次矩）：**

$$Q_\text{翼板} = b_f\, t_f \!\left(\frac{h_w}{2} + \frac{t_f}{2}\right) = 130 \times 20 \times \left(100 + 10\right) = 286{,}000\ \text{mm}^3$$

$$Q_\text{腹板半高} = t_w \cdot \frac{h_w}{2} \cdot \frac{h_w}{4} = 15 \times 100 \times 50 = 75{,}000\ \text{mm}^3$$

$$\boxed{Q_{\max} = 286{,}000 + 75{,}000 = 361{,}000\ \text{mm}^3}$$

### 4.2 固定端內力（$x = 0$）

設 $x$ 從 A 量起，載重強度 $w(x) = q_0\, x/L$。

**最大剪力：**

$$V_{\max} = \int_0^L \frac{q_0\, x}{L}\,dx = \frac{q_0 L}{2}$$

$$V_{\max} = \frac{q_0 \times 12{,}000}{2} = 6{,}000\, q_0\ \text{N} \quad (q_0 \text{ 單位 N/mm})$$

**最大彎矩（合力矩法）：**

合力 $= q_0 L/2$，合力作用點距 A 端 $\bar{x} = 2L/3$（三角形形心）：

$$M_{\max} = \frac{q_0 L}{2} \times \frac{2L}{3} = \frac{q_0 L^2}{3}$$

$$M_{\max} = \frac{q_0 \times 12{,}000^2}{3} = 48{,}000{,}000\, q_0\ \text{N·mm}$$

### 4.3 彎曲應力檢核

$$\sigma = \frac{M_{\max}\, c}{I} \leq \sigma_w$$

$$\frac{48{,}000{,}000\, q_0 \times 120}{73{,}093{,}333} \leq 80$$

$$78.80\, q_0 \leq 80 \implies \boxed{q_0 \leq 1.015\ \text{N/mm} = 1.015\ \text{kN/m}}$$

### 4.4 剪應力檢核

最大剪應力發生於中性軸處（腹板）：

$$\tau = \frac{V_{\max}\, Q_{\max}}{I\, t_w} \leq \tau_w$$

$$\frac{6{,}000\, q_0 \times 361{,}000}{73{,}093{,}333 \times 15} \leq 60$$

$$\frac{2{,}166{,}000{,}000\, q_0}{1{,}096{,}400{,}000} \leq 60$$

$$1.975\, q_0 \leq 60 \implies q_0 \leq 30.38\ \text{kN/m}$$

### 4.5 控制條件

| 檢核項 | $q_0$ 上限 |
|--------|-----------|
| 彎曲應力 $\sigma \leq 80\ \text{MPa}$ | **1.015 kN/m** ← 控制 |
| 剪應力 $\tau \leq 60\ \text{MPa}$ | 30.38 kN/m |

$$\boxed{q_{0,\max} \approx 1.015\ \text{kN/m}}$$

彎曲應力控制；剪應力遠未達上限（$\tau_{\max} = 1.975 \times 1.015 \approx 2.0\ \text{MPa} \ll 60\ \text{MPa}$）。

## §5 討論

**為何剪應力不控制？** 12 m 懸臂梁臂長極長，三角形載重（合力 $q_0 L/2$ 作用於距 A 端 $2L/3$ 處）造成的固定端彎矩 $M_A = q_0 L^2/3$ 對載重量值高度敏感：$L$ 加倍則 $M$ 增四倍，而 $V$ 僅增一倍。因此在長梁設計中，彎曲應力通常遠比剪應力先達極限。

**三角形載重與 UDL 比較：**

| 載重 | $M_A$ | $V_A$ | $M_A$ 相對 UDL |
|------|-------|-------|----------------|
| UDL（強度 $q_0$） | $q_0 L^2/2$ | $q_0 L$ | 1.0 |
| 三角形（自由端最大） | $q_0 L^2/3$ | $q_0 L/2$ | 2/3 |
| 三角形（固定端最大） | $q_0 L^2/6$ | $q_0 L/2$ | 1/3 |

本題採自由端最大（$M_A = q_0 L^2/3$），比固定端最大（$q_0 L^2/6$）嚴格一倍。

**斷面效率：** I 形斷面截面模數 $S = I/c = 609{,}111\ \text{mm}^3$，在 $\sigma_w = 80\ \text{MPa}$ 下可承受彎矩 $M_{\max} = 80 \times 609{,}111 = 48.73\ \text{kN·m}$，與 $q_0 = 1.015\ \text{kN/m}$ 的計算結果一致。

**考場陷阱：**
1. $h_w = 200\ \text{mm}$ 為腹板**淨高**，斷面總高 $H = 240\ \text{mm}$，$c = 120\ \text{mm}$（非 100 mm）。
2. 三角形載重的合力形心在距**較大端** $L/3$ 處；固定端 A 的力臂為 $2L/3$，非 $L/2$（UDL 才是 $L/2$）。
3. $Q_{\max}$ 計算需包含翼板與腹板半高兩部分，漏算翼板將嚴重低估 $Q$。
