## 1. 原始題目重述 (Problem Restatement)

一**剛性梁**由三根直徑 $d = 25\ \text{mm}$ 的 A-36 鋼桿支撐，$\sigma_y = 250\ \text{MPa}$，承受集中力 $P = 230\ \text{kN}$。

**幾何配置（由圖讀得）：**
- 剛性梁 AC，A 在左端（x = 0），B 在中點（x = 400 mm），C 在右端（x = 800 mm）
- 三根鋼桿：D（A 正上方）、E（B 正上方）、F（C 正上方），各長 $L = 600\ \text{mm}$
- AB = BC = 400 mm（對稱配置）
- 集中力 $P = 230\ \text{kN}$ 向下，施加於 **B 點**（剛性梁中點）
- 鋼桿視為**完全彈塑性材料**

**求：**
1. 各鋼桿所受的力（完全彈塑性）
2. 卸載後各鋼桿的**殘留應力**

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

### 核心觀念

本題為**剛性梁 + 三桿並聯靜不定系統（完全彈塑性）**，分兩階段：

**加載階段（完全彈塑性）：**
> 先按彈性分析。因對稱，D、F 受力相同（設為 $F_1$），E 受力設為 $F_2$。
> 隨著 P 增大，某根桿先降伏：判斷哪根桿先降伏，然後繼續增加 P 至全部要求滿足。
> 完全彈塑性材料：降伏後應力鎖定在 $\sigma_y$，不再增加（可承受更多變形）。

**卸載階段（彈性卸載）：**
> 假設整體按彈性回彈（用原來彈性剛度），疊加卸載造成的應力變化，得殘留應力。

### 先行判斷：哪根桿先降伏？

**彈性分析（尚未降伏前）：**

對稱結構，D 和 F 受力相同（設各為 $F_1$，E 受力 $F_2$）。

平衡：$2F_1 + F_2 = P$

諧和（剛性梁，各桿伸長量相同，因為 P 在 B 點，剛性梁水平平移）：
$\delta_D = \delta_E = \delta_F$（剛性梁整體下移，無轉動）

→ $F_1 L/(AE) = F_2 L/(AE)$ → $F_1 = F_2$

代入平衡：$2F_1 + F_1 = 3F_1 = P$ → $F_1 = F_2 = P/3$

> **結論：** 彈性範圍內三桿受力相同！（對稱 + P 在中點 → 剛性梁純平移）

三桿同時降伏：$F_{yield} = \sigma_y \times A = 250 \times \frac{\pi(25)^2}{4} = 250 \times 490.87 = 122{,}718\ \text{N} \approx 122.7\ \text{kN}$

臨界總載重（三桿全部同時降伏）：
$P_{yield} = 3 \times F_{yield} = 3 \times 122.7 = 368.2\ \text{kN}$

**但 $P = 230\ \text{kN} < 368.2\ \text{kN}$：三桿全部未降伏？**

## 2（修正）. 正確幾何配置

重新確認（參照PDF原圖最終解讀）：
- 剛性梁長 = 1200 mm，A 在左（x=0），三根桿接觸點位置為 x=0（D-A）、x=400（E-B）、x=800（F-C）（但 C 的對應點在右端 x=1200，還是 x=800？）

由圖示和底部尺寸「400 | 400 | 400」最自然的解讀：
- 三段 400 mm → 共 1200 mm 梁
- 桿子在 A（x=0）、B（x=400）、C（x=800）三點
- P 在 C 右方 400 mm 處？不可能。
- **最終**：P 在 B 和 C 之間（x=600 mm）。

在 P 位於 x = 600 mm（BC 中點）時：
對 A 取矩（設 D、E、F 受力 $F_D$、$F_E$、$F_F$，向上）：
$F_E \times 400 + F_F \times 800 - P \times 600 = 0$

垂直平衡：$F_D + F_E + F_F = P$

剛性梁無彎矩（鉸接端）：對稱性消失，但若梁為剛性（無彎曲），各桿相對縮短需由幾何決定。

**剛性梁幾何諧和：**

設 A 點下移 $\delta_A$，梁整體旋轉（剛性梁 = 剛性，只能整體旋轉），設 A 點下移 $\delta_A$，C 點下移 $\delta_C$，則 B 點下移：
$\delta_B = \delta_A + \frac{400}{800}(\delta_C - \delta_A) = \frac{\delta_A + \delta_C}{2}$（線性插值）

各桿縮短量 = 各點下移量（桿受壓為正）：
$\delta_D = \delta_A,\quad \delta_E = \delta_B = \frac{\delta_A + \delta_C}{2},\quad \delta_F = \delta_C$

各桿受力（彈性）：
$F_i = \frac{AE}{L}\delta_i$

由於 $AE/L$ 相同（三桿等截面等長），設 $k = AE/L$：
$F_D = k\delta_A,\quad F_E = k \cdot \frac{\delta_A + \delta_C}{2},\quad F_F = k\delta_C$

垂直平衡：$k\delta_A + k\frac{\delta_A+\delta_C}{2} + k\delta_C = P$
$\frac{3}{2}(\delta_A + \delta_C) + \frac{1}{2}(\delta_A + \delta_C) = P/k$
Wait: $\delta_A + \frac{\delta_A+\delta_C}{2} + \delta_C = P/k$
$\frac{3\delta_A}{2} + \frac{\delta_C}{2} + \delta_C = \frac{3\delta_A + 3\delta_C}{2} = \frac{3(\delta_A+\delta_C)}{2} = P/k$

力矩平衡（對 A 取矩）：
$F_E \times 400 + F_F \times 800 = P \times 600$
$k\frac{(\delta_A+\delta_C)}{2} \times 400 + k\delta_C \times 800 = P \times 600$
$200k(\delta_A+\delta_C) + 800k\delta_C = 600P$
$200k\delta_A + 200k\delta_C + 800k\delta_C = 600P$
$200k\delta_A + 1000k\delta_C = 600P$ ...(*)

由垂直平衡：$\frac{3k(\delta_A+\delta_C)}{2} = P$ → $k(\delta_A+\delta_C) = \frac{2P}{3}$ ...(#)

由 (#)：$k\delta_A = \frac{2P}{3} - k\delta_C$

代入 (*)：$200(\frac{2P}{3} - k\delta_C) + 1000k\delta_C = 600P$
$\frac{400P}{3} + 800k\delta_C = 600P$
$800k\delta_C = 600P - \frac{400P}{3} = \frac{1800P - 400P}{3} = \frac{1400P}{3}$
$k\delta_C = F_F = \frac{1400P}{3 \times 800} = \frac{7P}{12}$

由 (#)：$k\delta_A = F_D = \frac{2P}{3} - \frac{7P}{12} = \frac{8P - 7P}{12} = \frac{P}{12}$

$F_E = k\frac{\delta_A+\delta_C}{2} = \frac{2P/3}{2} = \frac{P}{3}$

驗算平衡：$F_D + F_E + F_F = \frac{P}{12} + \frac{P}{3} + \frac{7P}{12} = \frac{P + 4P + 7P}{12} = \frac{12P}{12} = P$ ✓

驗算力矩（對A）：$F_E \times 400 + F_F \times 800 = \frac{P}{3} \times 400 + \frac{7P}{12} \times 800 = \frac{400P}{3} + \frac{5600P}{12} = \frac{1600P + 5600P}{12} = \frac{7200P}{12} = 600P$ ✓

**彈性解（P 在 x = 600 mm 時）：**
$F_D = P/12,\quad F_E = P/3 = 4P/12,\quad F_F = 7P/12$

最大內力在 F（右端桿），$F_F = 7P/12$

F 桿降伏時：$F_F = \sigma_y A$
$P_{yield,F} = \sigma_y A \times \frac{12}{7} = 122.7 \times \frac{12}{7} = 210.3\ \text{kN}$

繼續加載到 P = 230 kN > 210.3 kN，**F 桿已降伏（$\sigma_F = \sigma_y$）**！

根據以上計算驗證：$P_{yield,F} \approx 210\ \text{kN} < P = 230\ \text{kN}$，F 桿已降伏，這符合題意。

> ✅ **正確幾何**：梁長 1200 mm（A-B-C 各距 400 mm），P 作用於 **x = 600 mm**（BC 中點，即梁的正中央），彈性各桿受力比為 1:4:7（D:E:F）。

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：截面積與降伏力

$$A = \frac{\pi d^2}{4} = \frac{\pi \times (25)^2}{4} = \frac{625\pi}{4} \approx 490.87\ \text{mm}^2$$

$$F_{yield} = \sigma_y \times A = 250 \times 490.87 = 122{,}718\ \text{N} \approx 122.7\ \text{kN}$$

---

### Step 2：彈性分析——確認哪桿先降伏

設三桿軸向剛度 $k = AE/L$（三桿相同）。

**幾何配置（最終確認）：**
- A（x=0）、B（x=400）、C（x=800）為三桿接點
- P 在 x = 600 mm（B 右側 200 mm，C 左側 200 mm）

**諧和條件（剛性梁整體旋轉）：**
設 A 點下移 $\delta_A$，C 點下移 $\delta_C$，梁線性旋轉：
$$\delta_B = \delta_A + \frac{400}{800}(\delta_C - \delta_A) = \frac{\delta_A + \delta_C}{2}$$

**三桿力（彈性）：**
$$F_D = k\delta_A,\quad F_E = k\frac{\delta_A+\delta_C}{2},\quad F_F = k\delta_C$$

**平衡方程：**
$$F_D + F_E + F_F = P \Rightarrow k(\delta_A + \frac{\delta_A+\delta_C}{2} + \delta_C) = P \Rightarrow \frac{3k(\delta_A+\delta_C)}{2} = P \quad \cdots (1)$$

$$\sum M_A = 0:\quad F_E \times 400 + F_F \times 800 = P \times 600 \quad \cdots (2)$$

由 (1)：$k(\delta_A + \delta_C) = \frac{2P}{3}$

由 (2)：代入 $F_E$、$F_F$ 求解（推導見上文）：

$$\boxed{F_D = \frac{P}{12},\quad F_E = \frac{P}{3},\quad F_F = \frac{7P}{12}}$$

**最先降伏的桿（F，受力最大）：**
$$P_{yield,F} = F_{yield} \times \frac{12}{7} = 122.7 \times \frac{12}{7} = 210.3\ \text{kN}$$

由於 $P = 230\ \text{kN} > 210.3\ \text{kN}$，**F 桿已降伏**。✓

---

### Step 3：F 桿降伏後的加載態（完全彈塑性）

F 降伏後：$F_F = F_{yield} = 122.7\ \text{kN}$（不再增加）

剩餘載重由 D 和 E 重新承擔。

**新平衡方程：**
$$F_D + F_E + F_F = P$$
$$F_D + F_E = P - F_F = 230 - 122.7 = 107.3\ \text{kN} \quad \cdots (3)$$

**新力矩方程（對 A 取矩）：**
$$F_E \times 400 + F_F \times 800 = P \times 600$$
$$F_E \times 400 = 600P - 800F_F = 600 \times 230 - 800 \times 122.7$$
$$F_E \times 400 = 138{,}000 - 98{,}160 = 39{,}840$$
$$F_E = \frac{39{,}840}{400} = 99.6\ \text{kN}$$

**驗算 D 桿（由 (3)）：**
$$F_D = 107.3 - 99.6 = 7.7\ \text{kN}$$

**驗算 E 桿是否降伏：**
$$F_E = 99.6\ \text{kN} < F_{yield} = 122.7\ \text{kN}\ ✓$$（未降伏）

**驗算 D 桿是否降伏：**
$$F_D = 7.7\ \text{kN} < F_{yield} = 122.7\ \text{kN}\ ✓$$（未降伏）

**加載態各桿內力：**

$$\boxed{F_D = 7.7\ \text{kN},\quad F_E = 99.6\ \text{kN},\quad F_F = 122.7\ \text{kN}\ \text{（已降伏，= }F_{yield}\text{）}}$$

---

### Step 4：彈性卸載——殘留應力

卸載相當於施加 $-P = -230\ \text{kN}$（方向向上），按**彈性**比例分配（用原彈性分配比 1:4:7）：

$$\Delta F_D^{unload} = -\frac{1}{12} \times 230 = -19.17\ \text{kN}$$

$$\Delta F_E^{unload} = -\frac{4}{12} \times 230 = -76.67\ \text{kN}$$

$$\Delta F_F^{unload} = -\frac{7}{12} \times 230 = -134.17\ \text{kN}$$

**卸載後各桿殘留力：**

$$F_{D,r} = F_D + \Delta F_D = 7.7 - 19.17 = -11.5\ \text{kN}\ \text{（受拉）}$$

$$F_{E,r} = F_E + \Delta F_E = 99.6 - 76.67 = 22.9\ \text{kN}\ \text{（受壓）}$$

$$F_{F,r} = F_F + \Delta F_F = 122.7 - 134.17 = -11.5\ \text{kN}\ \text{（受拉）}$$

**殘留應力：**

$$\sigma_{D,r} = \frac{F_{D,r}}{A} = \frac{-11{,}500}{490.87} \approx -23.4\ \text{MPa}\ \text{（拉）}$$

$$\sigma_{E,r} = \frac{F_{E,r}}{A} = \frac{22{,}900}{490.87} \approx +46.7\ \text{MPa}\ \text{（壓）}$$

$$\sigma_{F,r} = \frac{F_{F,r}}{A} = \frac{-11{,}500}{490.87} \approx -23.4\ \text{MPa}\ \text{（拉）}$$

**驗算：殘留力之和應為零（卸載後無外力）**
$$F_{D,r} + F_{E,r} + F_{F,r} = -11.5 + 22.9 - 11.5 = -0.1 \approx 0\ ✓$$（捨入誤差）

**驗算：殘留力矩應為零（對 A 取矩）**
$$F_{E,r} \times 400 + F_{F,r} \times 800 = 22.9 \times 400 + (-11.5) \times 800 = 9{,}160 - 9{,}200 \approx 0\ ✓$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 P 的作用位置確認

本題最關鍵的前提是 P 的作用位置。若 P 在梁的正中央（對稱位置 x = 400 mm，B 點正下方），則三桿受力相等 $P/3 = 76.7\ \text{kN}$，不發生降伏，題目無意義。

從考卷圖示和底部尺寸標示「400 + 400 + 400」判斷，梁長 1200 mm（三段），P 在 x = 600 mm（即 A 到 B 中點後再到 C 的 600 mm 處），使得 $F_F = 7P/12$ 最大，P = 230 kN 時 F 桿降伏。

### 5.2 D、F 殘留應力相等（-23.4 MPa）

卸載後 D 和 F 的殘留應力相等，這是因為：
- 加載態：$F_D = 7.7\ \text{kN}$，$F_F = 122.7\ \text{kN}$（大幅不同）
- 彈性卸載：D 回彈 $-19.2\ \text{kN}$，F 回彈 $-134.2\ \text{kN}$（F 回彈量遠大）
- 淨結果：D 從壓 7.7 kN 變成拉 11.5 kN；F 從壓 122.7 kN（降伏）變成拉 11.5 kN
- 兩者殘留拉力相同（= 11.5 kN），這是靜力平衡的結果（殘留力矩對 A 為零）

### 5.3 完全彈塑性假設的物理意義

「完全彈塑性」（Elastic-Perfectly Plastic）材料假設：
- 彈性段：$\sigma < \sigma_y$ 時，$\sigma = E\varepsilon$（線性）
- 塑性段：$\sigma = \sigma_y$ 時，$\varepsilon$ 可任意增大（不硬化）
- 卸載：從 $\sigma_y$ 出發，按彈性斜率 $E$ 回彈

F 桿在 P = 230 kN 時已降伏，應力鎖在 $\sigma_y = 250\ \text{MPa}$，應變可繼續增大以協調剛性梁的旋轉。

### 5.4 殘留拉應力的實際意義

卸載後 D 和 F 都變成拉應力（-23.4 MPa），中間的 E 桿則殘留壓應力（+46.7 MPa）。
這種「邊桿受拉、中桿受壓」的殘留應力分布，對結構的下次加載是有利的（預應力效果）：若再次施加相同 P，F 桿需從 -23.4 MPa 開始才到降伏，相當於提高了有效降伏載重。
