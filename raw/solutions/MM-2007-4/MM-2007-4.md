# 考題編號：MM-2007-4

**主分類：** `MM-U3-2` 梁桿件變位及內力分析  
**副分類：** —  
**分析法：** 彈性分析  
**標籤：** `靜不定梁` `彈簧` `間隙` `撓度` `疊加法` `接觸問題` `簡支樑` `變形諧和` `彈性支承`

---

## 1. 原始題目重述 (Problem Restatement)

二簡支樑 AB 與 CD 有相同長度 L 及相同彎矩勁度 EI。AB 樑在上，CD 樑在下，兩者中點之間有一線性彈簧（彈簧係數 k）。**彈簧上端與 AB 樑中點下緣之間有一間隙 Δ**。

AB 樑中點受集中力 P 向下作用，且該處向下位移已超過 Δ（即彈簧已接觸並受壓）。

**要求：** 求彈簧中之力 F。

![MM-2007-4 題目附圖：AB 簡支樑（上，跨度 L），CD 簡支樑（下，跨度 L），兩樑中點間有彈簧 k，彈簧上端與 AB 中點有間隙 Δ，AB 中點受 P 向下](MM-2007-4-fig-1.png)

*圖說：上樑 AB，A 端銷接，B 端滾接，跨度 L，中點受 P 向下；下樑 CD，C 端銷接，D 端滾接，跨度 L；兩樑等 EI；彈簧 k 位於兩樑中點之間，彈簧上端與 AB 下緣間隙為 Δ；給定 AB 中點位移超過 Δ（彈簧已接觸）。*

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 接觸問題（含間隙）的變形諧和分析——AB 中點位移、彈簧壓縮量、CD 中點位移三者的幾何相容關係。

**考驗能力：**
1. 正確建立間隙接觸後的幾何相容式（最關鍵步驟）
2. 簡支樑中點撓度公式 $\delta = WL^3/(48EI)$ 的應用
3. 疊加法：AB 受到 P（下）與 F（上），分開計算再疊加
4. 聯立方程解出彈簧力 F

**出題者預設陷阱：**
- 遺漏間隙 Δ，直接令 $\delta_{AB} = \delta_{CD}$（忽略接觸條件）
- 不知道 AB 受 P 的同時，彈簧力 F 對 AB 是向上的（忘記疊加符號）
- 誤以為 AB 和 CD 的中點位移相等（實際上差了彈簧壓縮量加間隙）

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫（4步）：**
1. 設彈簧力 F，確定 F 對 AB（向上）與 CD（向下）的作用
2. 用疊加法寫出 $\delta_{AB}$（P 與 F 共同作用）
3. 用疊加法寫出 $\delta_{CD}$（F 單獨作用）
4. 建立幾何相容式：$\delta_{AB} = \Delta + F/k + \delta_{CD}$，解 F

**陷阱 1：相容式的建立（最多人失分）**

接觸後，從幾何上：AB 中點下移量 = 間隙 + 彈簧壓縮 + CD 中點下移量

$$\delta_{AB} = \Delta + \frac{F}{k} + \delta_{CD}$$

錯誤寫法：$\delta_{AB} = \delta_{CD}$（未含 Δ 和彈簧壓縮）

**陷阱 2：AB 的等效荷重**

AB 受 P 向下，同時受彈簧反力 F 向上；中點合力為 $(P - F)$ 向下：

$$\delta_{AB} = \frac{(P - F)L^3}{48EI}$$

**陷阱 3：CD 受力方向**

彈簧對 CD 施力向下（力傳遞方向），故：

$$\delta_{CD} = \frac{FL^3}{48EI}$$

**陷阱 4：有效性檢核**

題目已說「位移超過 Δ」，代表接觸已發生，F > 0，無需另行判斷接觸條件。若計算結果 F < 0，表示前提矛盾。

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
接觸後彈簧中之壓力 F

### 本題關鍵公式（依計算順序）

$$\text{Step 1 AB 撓度（疊加）} \quad \delta_{AB} = \frac{(P - F)\,L^3}{48EI}$$

$$\text{Step 2 CD 撓度} \quad \delta_{CD} = \frac{F\,L^3}{48EI}$$

$$\text{Step 3 幾何相容式} \quad \boxed{\delta_{AB}} = \Delta + \frac{F}{k} + \boxed{\delta_{CD}}$$

$$\text{Step 4 代入整理} \quad \frac{PL^3}{48EI} - \Delta = F\!\left(\frac{L^3}{24EI} + \frac{1}{k}\right)$$

$$\text{Step 5 解 F} \quad F = \frac{k\!\left(PL^3 - 48EI\Delta\right)}{2kL^3 + 48EI}$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $L$ | 給定 | 兩樑跨度（相同） |
| $EI$ | 給定 | 兩樑彎矩勁度（相同） |
| $k$ | 給定 | 彈簧係數 |
| $\Delta$ | 給定 | 彈簧上端與 AB 下緣初始間隙 |
| $P$ | 給定 | AB 中點集中載重 |

### L2：需知識點推導

**簡支樑中點撓度**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|------|
| $\delta_{AB}$ | $(P-F)L^3/(48EI)$，疊加 P 與 −F | |
| $\delta_{CD}$ | $FL^3/(48EI)$，F 單獨作用 | |
| $F/k$ | 彈簧壓縮量（虎克定律） | |

**幾何相容**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|------|
| 相容式 | $\delta_{AB} - \delta_{CD} - F/k = \Delta$ | |
| 整理後 | $F = k(PL^3 - 48EI\Delta)/(2kL^3 + 48EI)$ | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 4.1 設定未知量與受力分析

設彈簧力 $F$（壓力，正值）。

- 彈簧對 AB 中點：向上 $F$
- 彈簧對 CD 中點：向下 $F$

### 4.2 AB 樑中點撓度（疊加法）

AB 中點受 $P$（下）與 $F$（上），等效中點荷重 = $(P-F)$ 向下：

$$\delta_{AB} = \frac{(P-F)\,L^3}{48EI} \tag{1}$$

### 4.3 CD 樑中點撓度

CD 中點僅受 $F$ 向下：

$$\delta_{CD} = \frac{F\,L^3}{48EI} \tag{2}$$

### 4.4 建立幾何相容式

接觸後，AB 中點向下位移須等於「間隙 + 彈簧壓縮 + CD 中點向下位移」：

$$\delta_{AB} = \Delta + \frac{F}{k} + \delta_{CD} \tag{3}$$

### 4.5 代入並整理

將 (1)(2) 代入 (3)：

$$\frac{(P-F)L^3}{48EI} = \Delta + \frac{F}{k} + \frac{FL^3}{48EI}$$

$$\frac{PL^3}{48EI} - \frac{FL^3}{48EI} = \Delta + \frac{F}{k} + \frac{FL^3}{48EI}$$

移項：

$$\frac{PL^3}{48EI} - \Delta = \frac{FL^3}{48EI} + \frac{FL^3}{48EI} + \frac{F}{k} = \frac{2FL^3}{48EI} + \frac{F}{k}$$

提出 $F$：

$$\frac{PL^3}{48EI} - \Delta = F\!\left(\frac{L^3}{24EI} + \frac{1}{k}\right) \tag{4}$$

通分：

$$\frac{PL^3}{48EI} - \Delta = F\cdot\frac{kL^3 + 24EI}{24EIk}$$

解 $F$：

$$F = \left(\frac{PL^3}{48EI} - \Delta\right)\cdot\frac{24EIk}{kL^3 + 24EI}$$

$$F = \frac{(PL^3 - 48EI\Delta)\cdot 24EIk}{48EI\,(kL^3 + 24EI)}$$

$$\boxed{F = \frac{k\!\left(PL^3 - 48EI\Delta\right)}{2kL^3 + 48EI}}$$

### 4.6 物理意義驗證

令 $\delta_0 = \frac{PL^3}{48EI}$（AB 無彈簧時的自由中點撓度），$k_{beam} = \frac{48EI}{L^3}$（簡支樑中點等效彈簧勁度），則：

$$F = \frac{\delta_0 - \Delta}{\dfrac{2}{k_{beam}} + \dfrac{1}{k}}$$

**解讀：** 「超額位移 $(\delta_0 - \Delta)$」被 3 個串聯柔度（AB 梁柔度 + 彈簧柔度 + CD 梁柔度）共同分配，物理上完全合理。

**邊界檢核：**
- 若 $k \to \infty$（剛性桿）：$F \to \frac{k(PL^3 - 48EI\Delta)}{2kL^3} = \frac{P}{2} - \frac{24EI\Delta}{L^3}$；兩樑各承一半荷重，合理。
- 若 $\Delta \to 0$（無間隙）：$F = \frac{kPL^3}{2kL^3 + 48EI}$，等同於直接接觸的標準結果。
- 若 $k \to 0$（無彈簧）：$F \to 0$，AB 自由撓曲，合理。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**5.1 接觸條件（本題已給定，但考場需說明）**

接觸發生的前提：AB 中點自由撓度 $\delta_0 > \Delta$，即：

$$\frac{PL^3}{48EI} > \Delta \quad \Leftrightarrow \quad P > \frac{48EI\Delta}{L^3}$$

題目已說明此條件成立，計算出的 $F$ 應為正值；若代入數值後 $F < 0$，代表假設矛盾（接觸未發生）。

**5.2 靜不定度分析**

- 接觸發生後，系統含 AB、彈簧、CD 三個串聯元件，多出 1 個多餘未知量（彈簧力 F）
- 使用 1 個幾何相容式（式3）消去多餘未知量 → 1次靜不定
- 此題是「接觸型靜不定」，和溫度型、沉陷型靜不定同屬相容法範疇

**5.3 考場最安全寫法**

1. 畫出接觸後的受力圖（AB 受 P 與 F，CD 受 F）
2. 明確標示間隙 Δ 在相容式中的位置
3. 最終答案寫成封閉式，並說明 $F > 0$ 代表彈簧受壓（物理意義正確）
