# 考題編號：MM-2009-4

**主分類：** `MM-U3-2` 梁桿件變位及內力分析
**副分類：** `MM-U3-4` 柱之挫屈載重分析
**分析法：** 混合
**標籤：** `梁柱問題` `懸臂梁` `均布載重` `軸向壓力` `二階效應` `挫屈` `歐拉公式` `固定自由端`

---

## 1. 原始題目重述 (Problem Restatement)

懸臂梁（cantilever beam），長度 L，左端固定（x=0），右端自由（x=L）。梁的上方作用均布荷重 q（向下），梁的右端自由端同時受一**軸向壓力 P**（向左）。假設 EI 為常數。試求：

**(a) 該梁之變形。（15 分）**

**(b) 挫屈臨界載重 $P_{cr}$。（10 分）**

![MM-2009-4 題目附圖：懸臂梁，左端固定，右端自由端受軸向壓力 P，全梁受均布荷重 q，梁長 L](MM-2009-4-fig-1.png)

*圖說：懸臂梁，x=0 為固定端（左），x=L 為自由端（右）；梁上作用均布荷重 q（向下，N/m）；自由端施加軸向壓力 P（沿梁軸向左）；梁的彎曲剛度 EI 為常數。*

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：梁柱分析（beam-column）— 橫向荷重與軸向壓力的二階效應**

- **(a)** 軸向壓力 P 在梁產生橫向撓度後，產生額外彎矩（P-δ 效應），使問題成為二階 ODE（非純彎曲）
- **(b)** 當 P 達到臨界值 $P_{cr}$ 時，梁在無橫向荷重 q 下也能維持微彎平衡（Euler 挫屈）

**出題者意圖：**
1. 測試能否建立含 P-δ 效應的微分方程
2. 考查二階線性 ODE 的特解與通解求解能力
3. 驗證從挫屈條件推導 $P_{cr}$ 的方法（與標準 Euler 公式一致）

**核心陷阱：**
- 以為 P 不影響變形，只用 EI y'' = -q(L-x)²/2 求解（遺漏 P-δ 效應）
- 混淆挫屈條件：應為 cos(kL)=0，不是 sin(kL)=0
- 部分 (b) 的 Pcr 需確認有效長度係數 K=2（固定-自由），不可用 K=1

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫（a 部分）：**
1. 取任意截面 x，由右側自由體計算總彎矩（含 P-δ 效應）
2. 代入 EI y'' = M(x)，整理成 y'' + k²y = RHS 形式
3. 寫通解（齊次 + 特解）
4. 應用 BC：x=0 固定端（y=0, y'=0）、x=L 自由端（M=0, 即 y''=0）
5. 求解全部常數，得到 y(x) 與端點撓度 δ

**作戰計畫（b 部分）：**
1. 令 q=0（純挫屈模式）
2. 求 y'' + k²y = k²δ 的解 y = δ(1-cos(kx))
3. 應用 M(L)=0 → cos(kL)=0 → kL = π/2 → Pcr

**關鍵陷阱：**

| 陷阱 | 說明 | 應對策略 |
|------|------|---------|
| ⚠ 陷阱①：忘記 P-δ 效應 | 純彎曲解忽略 P 對彎矩的貢獻，當 P 不小時誤差大 | 自由體彎矩 M(x) 必須包含 P·(δ-y) 項 |
| ⚠ 陷阱②：M 正負號 | 懸臂梁 q 引起的彎矩為負（hogging），P 引起的二階彎矩也是負 | 用右側自由體統一計算，符號自然對應 |
| ⚠ 陷阱③：BCs 混淆 | 固定端：y=0, y'=0；自由端：M=0（y''=0），不是 y=0 | 自由端有 P 但 P 沿軸向，不產生末端彎矩 |
| ⚠ 陷阱④：挫屈條件 | cos(kL)=0 不是 tan(kL)=0；Pcr 對應最小正根 kL=π/2 | Euler 固定-自由：有效長度 KL=2L，K=2 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
(a) 求懸臂梁柱的撓度曲線 y(x) 及端部撓度 δ；(b) 求挫屈臨界載重 Pcr

### 本題關鍵公式（依計算順序）

$$
\text{Step 1: } M(x) = -\frac{q(L-x)^2}{2} - P(\delta - y(x))
$$

$$
\text{Step 2: } -EI\,y'' = M(x) \Rightarrow y'' + k^2 y = \frac{q(L-x)^2}{2EI} + k^2\delta, \quad k^2 = \frac{P}{EI}
$$

$$
\text{Step 3: 通解 } y = A\cos(kx) + B\sin(kx) + \underbrace{\frac{q(L-x)^2}{2P} + \delta - \frac{q}{k^2 P}}_{\text{特解}}
$$

$$
\text{Step 4: BC } y(0)=0,\; y'(0)=0 \Rightarrow A = \frac{qEI}{P^2}\left(1-\frac{k^2L^2}{2}\right)-\delta,\quad B = \frac{qL}{kP}
$$

$$
\text{Step 5: BC } y''(L)=0 \Rightarrow \delta = \frac{qEI}{P^2}\left[kL\tan(kL)+1-\sec(kL)-\frac{k^2L^2}{2}\right]
$$

$$
\text{(b) 挫屈條件: } \cos(kL) = 0 \Rightarrow kL = \frac{\pi}{2} \Rightarrow P_{cr} = \frac{\pi^2 EI}{4L^2}
$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $q$ | 均布荷重強度（N/m） | 向下作用於全梁 |
| $P$ | 軸向壓力（N） | 自由端向左（壓縮） |
| $L$ | 梁長（m） | 固定端到自由端 |
| $EI$ | 彎曲剛度（N·m²） | 常數 |

### L2：需知識點推導

**P-δ 效應的彎矩計算**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $\delta = y(L)$ | 自由端撓度（未知，由 BC 決定） | |
| $M(x)$ | $-q(L-x)^2/2 - P(\delta - y(x))$（右側自由體，含二階效應） | |

**ODE 建立與求解**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $k$ | $\sqrt{P/EI}$（特徵頻率） | |
| 特解 $y_p$ | 試解 $ax^2+bx+c$，比較係數 → $y_p = q(L-x)^2/(2P)+\delta-qEI/P^2$ | |
| A | 由 BC $y(0)=0$ 決定 | |
| B | 由 BC $y'(0)=0$ 決定：$B=qL/(kP)$ | |

**決定 δ（端部撓度）**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $y''(L)=0$ | 自由端無彎矩，額外一個方程求 $\delta$ | |
| $\delta$ | 代入 $y''(L)=0$，解出端部撓度公式 | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| 二階效應（P-δ 效應） | 橫向撓度使軸力產生額外彎矩，只有在梁有變形後才出現，使 ODE 為 $y''+k^2y=f(x)$ | |
| ODE 特解的試設 | 非齊次項含 $(L-x)^2$（二次多項式），特解試 $y_p=ax^2+bx+c$ 即可 | |
| 端部撓度 δ 是未知量 | δ 出現在 ODE 右側，作為常數，由第三個 BC（$y''(L)=0$）決定 | |
| 挫屈 vs 梁柱 | 挫屈：q=0 時找非零解的條件（中性平衡）；梁柱：q≠0 時求有限撓度 | |
| 固定-自由挫屈有效長度 | K=2（有效長度 KL=2L），對應 kL=π/2，Pcr=π²EI/(4L²) | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Part (a)：梁之變形

#### Step 1：建立彎矩方程（含 P-δ 二階效應）

取截面 x 右側自由體（x 從固定端量起）：
- q 引起的彎矩（hogging 為負）：$-q(L-x)^2/2$
- P 引起的二階彎矩（hogging 為負，因梁向下彎時 P 使截面 x 右側有偏心）：

設 $\delta = y(L)$（自由端向下撓度，待定）

$$
M(x) = -\frac{q(L-x)^2}{2} - P\bigl[\delta - y(x)\bigr]
$$

#### Step 2：代入樑方程

以向下為正，使用 $M = -EI\,y''$（下凸時 $y''>0$，上凸 hogging 時 $M<0$）：

$$
-EI\,y'' = -\frac{q(L-x)^2}{2} - P\bigl[\delta - y(x)\bigr]
$$

整理：

$$
EI\,y'' - Py = -\frac{q(L-x)^2}{2} - P\delta
$$

$$
\boxed{y'' + k^2 y = \frac{q(L-x)^2}{2EI} + k^2\delta}, \qquad k^2 \equiv \frac{P}{EI}
$$

> **策略注意：** 將 $-P\,y$ 移項得 $+k^2 y$（注意符號：壓縮 P 使 ODE 右端出現 $+k^2y$，對應 **振盪型通解** $\cos/\sin$，而非雙曲 $\cosh/\sinh$）

#### Step 3：求通解（齊次 + 特解）

**齊次通解：**
$$
y_h = A\cos(kx) + B\sin(kx)
$$

**特解：** 試 $y_p = ax^2 + bx + c$，代入 ODE：

$$
2a + k^2(ax^2 + bx + c) = \frac{q(L-x)^2}{2EI} + k^2\delta
$$

比較各次方係數：

| 次方 | 左側 | 右側 | 結果 |
|------|------|------|------|
| $x^2$ | $k^2 a$ | $q/(2EI)$ | $a = \dfrac{q}{2EIk^2} = \dfrac{q}{2P}$ |
| $x^1$ | $k^2 b$ | $-qL/EI$ | $b = -\dfrac{qL}{EIk^2} = -\dfrac{qL}{P}$ |
| $x^0$ | $2a + k^2c$ | $\dfrac{qL^2}{2EI} + k^2\delta$ | $c = \dfrac{qL^2}{2P} + \delta - \dfrac{q}{k^2 P} = \dfrac{qL^2}{2P} + \delta - \dfrac{qEI}{P^2}$ |

特解化簡（$q/(2P)\cdot x^2 - (qL/P)\cdot x + qL^2/(2P) = q(L-x)^2/(2P)$）：

$$
y_p = \frac{q(L-x)^2}{2P} + \delta - \frac{qEI}{P^2}
$$

**通解：**
$$
y(x) = A\cos(kx) + B\sin(kx) + \frac{q(L-x)^2}{2P} + \delta - \frac{qEI}{P^2}
$$

#### Step 4：應用邊界條件

**BC1：$y(0) = 0$（固定端無撓度）**

$$
A + 0 + \frac{qL^2}{2P} + \delta - \frac{qEI}{P^2} = 0 \quad\Rightarrow\quad A = \frac{qEI}{P^2} - \frac{qL^2}{2P} - \delta
$$

**BC2：$y'(0) = 0$（固定端無轉角）**

$$
y'(x) = -Ak\sin(kx) + Bk\cos(kx) - \frac{q(L-x)}{P}
$$

$$
y'(0) = Bk - \frac{qL}{P} = 0 \quad\Rightarrow\quad B = \frac{qL}{kP}
$$

**BC3：$y''(L) = 0$（自由端無彎矩）**

$$
y''(x) = -Ak^2\cos(kx) - Bk^2\sin(kx) + \frac{q}{P}
$$

$$
y''(L) = -k^2 A\cos(kL) - k^2 B\sin(kL) + \frac{q}{P} = 0
$$

代入 A、B：

$$
-k^2\!\left(\frac{qEI}{P^2} - \frac{qL^2}{2P} - \delta\right)\!\cos(kL) - k^2\cdot\frac{qL}{kP}\sin(kL) + \frac{q}{P} = 0
$$

求 δ：

$$
k^2\delta\cos(kL) = k^2\!\left(\frac{qEI}{P^2} - \frac{qL^2}{2P}\right)\!\cos(kL) - \frac{qLk}{P}\sin(kL) + \frac{q}{P}
$$

利用 $q/P = k^2 \cdot qEI/P^2$，整理：

$$
\boxed{\delta = \frac{qEI}{P^2}\!\left[kL\tan(kL) + 1 - \sec(kL) - \frac{k^2L^2}{2}\right]}
$$

（此式在 $\cos(kL)\neq 0$ 時，即 $P < P_{cr}$ 時成立）

**驗算（P→0 極限）：** 利用 Taylor 展開 $kL\to 0$：
$$
kL\tan(kL) + 1 - \sec(kL) - \frac{k^2L^2}{2} \approx \frac{k^4L^4}{8}
$$
$$
\delta \approx \frac{qEI}{P^2}\cdot\frac{k^4L^4}{8} = \frac{qEI}{P^2}\cdot\frac{P^2L^4}{8EI^2}\cdot EI = \frac{qL^4}{8EI} \checkmark
$$

#### Step 5：撓度曲線全式

$$
y(x) = \frac{qEI}{P^2}\!\left\{(1-\cos kx)\!\left[kL\tan(kL) - \sec(kL)\right] + kL\sin(kx) - \frac{k^2 x(2L-x)}{2}\right\}
$$

---

### Part (b)：挫屈臨界載重 $P_{cr}$

**方法：令 $q=0$，求非零撓度解的條件（中性平衡）**

令 $q=0$，ODE 退化為：
$$
y'' + k^2 y = k^2\delta
$$

通解：$y = C_1\cos(kx) + C_2\sin(kx) + \delta$

BCs：
- $y(0)=0$：$C_1 + \delta = 0 \Rightarrow C_1 = -\delta$
- $y'(0)=0$：$C_2 k = 0 \Rightarrow C_2 = 0$（$k\neq 0$）

撓度型態：$y = \delta(1 - \cos(kx))$（典型懸臂挫屈模態）

BC：$y''(L) = 0$（自由端 M=0）

$$
y''(L) = -\delta k^2(-\cos(kL))... \quad y''(x) = -\delta k^2 \cdot(-\cos(kx))
$$

Wait: $y = \delta(1-\cos kx)$, so $y'' = \delta k^2\cos(kx)$.

$$
y''(L) = \delta k^2\cos(kL) = 0
$$

非零解條件（$\delta\neq 0$，$k\neq 0$）：

$$
\cos(kL) = 0 \quad\Rightarrow\quad kL = \frac{\pi}{2} + n\pi \quad (n=0,1,2,\ldots)
$$

取最小正根 $kL = \dfrac{\pi}{2}$：

$$
k^2 = \frac{\pi^2}{4L^2} = \frac{P_{cr}}{EI}
$$

$$
\boxed{P_{cr} = \frac{\pi^2 EI}{4L^2}}
$$

**挫屈模態：** $y = \delta\!\left(1 - \cos\dfrac{\pi x}{2L}\right)$

**物理意義：** 有效長度 $K = 2$（固定-自由端），等效自由長度 $KL = 2L$，與 Euler 公式 $P_{cr} = \pi^2EI/(KL)^2$ 一致 ✓

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 Part (a) 端點撓度隨 P 增大的趨勢

當 $P \to P_{cr}$（即 $kL \to \pi/2$）：$\tan(kL) \to \infty$，$\sec(kL) \to \infty$，兩者之差趨於無窮大 → $\delta \to \infty$。

這代表在臨界荷重附近，即使微小的 q 也會引起極大的撓度，與挫屈的物理直覺一致。

### 5.2 Part (b) 考場快捷：直接套 Euler 公式

懸臂梁（固定-自由）挫屈有效長度 K=2：
$$P_{cr} = \frac{\pi^2 EI}{(KL)^2} = \frac{\pi^2 EI}{(2L)^2} = \frac{\pi^2 EI}{4L^2}$$

但若考題要求「由微分方程推導」，則需走 Part (b) 的完整推導。

### 5.3 二階效應的工程意義

當 $P/P_{cr}$ 的比值越大，二階效應越顯著。工程設計中常用**放大係數**近似：
$$\delta_{\text{total}} \approx \frac{\delta_{q\text{ alone}}}{1 - P/P_{cr}}$$

本題精確解中 $kL\tan(kL) - \sec(kL) - k^2L^2/2 + 1$ 在 $kL\to 0$ 時趨近於 $k^4L^4/8$，與一階解的 $qL^4/(8EI)$ 一致，也可在更高階展開中看到 P 的放大效果。
