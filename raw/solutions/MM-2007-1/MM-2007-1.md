# 考題編號：MM-2007-1

**主分類：** `MM-U3-4` 柱之挫屈載重分析  
**副分類：** `MM-U3-1` 軸力桿件變位及內力分析  
**分析法：** 彈性分析  
**標籤：** `桁架` `熱膨脹` `溫度載重` `挫屈` `臨界載重` `靜不定桁架` `歐拉挫屈` `變形諧和`

---

## 1. 原始題目重述 (Problem Restatement)

有一桁架如圖一所示，三根桿件 AD、BD、CD 在下方接點 D 以銷釘相連，上端分別固定於天花板銷釘支承 A、B、C。BD 為垂直桿（B 在正上方），AD 與 CD 對稱斜置，與 BD 之夾角各為 β（在 D 點量測）。

**幾何與材料條件：**
- 所有桿件 E、I、A、α 均相同
- BD 垂直長度為 L
- AD、CD 長度為 $L_{AD} = L_{CD} = L/\cos\beta$
- BD 溫度上升 ΔT，AD 與 CD 溫度不變

**要求：** 為避免 BD 桿發生挫屈，求 ΔT 之最大值。

![MM-2007-1 題目附圖：三桿桁架，A（左上）、B（正上方）、C（右上）固定於天花板，D 為下方接點，BD 垂直長度 L，AD 及 CD 與 BD 夾角各為 β](MM-2007-1-fig-1.png)

*圖說：A、C 為天花板銷釘（左右對稱），B 為天花板銷釘（正上方），D 為接點；BD 垂直長 L；β 為 BD 與斜桿在 D 點之夾角；斜桿長 L/cosβ。所有桿件 E、I、A、α 相同。*

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 溫度載重下的一次靜不定桁架分析，結合歐拉挫屈臨界力判斷。

**考驗能力：**
1. 能否正確識別靜不定度（3 桿 1 接點 → 1 次靜不定）
2. 能否建立溫度效應的變形諧和條件（考場最常失誤處）
3. 能否判斷 BD 的有效長度（兩端銷接 → K = 1）
4. 最終能將 |P_BD| ≤ P_cr 解出 ΔT

**出題者預設陷阱：**
- 混淆熱膨脹量與機械變形量
- 忽略斜桿的長度修正（$L_{AD} = L/\cos\beta$）
- 誤用有效長度（BD 兩端均為銷接，K = 1）

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫（5步）：**
1. 確認靜不定度 → 設 D 點垂直位移 δ_D 為多餘未知量
2. 建立平衡方程（D 點，利用對稱性）
3. 建立各桿件變形諧和條件（含溫差項）
4. 聯立求 P_BD（BD 之壓力）以 ΔT 表示
5. 令 |P_BD| = P_cr（歐拉公式）解 ΔT_max

**陷阱 1：熱膨脹與機械變形的分離**
- BD 桿的總伸長量 = 機械伸長 + 熱伸長：$\delta_{BD,total} = \frac{P_{BD} \cdot L}{AE} + \alpha \Delta T \cdot L$
- 幾何相容要求 $\delta_{BD,total} = \delta_D$，切勿直接令機械伸長等於位移

**陷阱 2：斜桿伸長量需投影**
- D 往下移 $\delta_D$ 時，AD 的伸長量 = $\delta_D \cos\beta$（沿桿件方向投影）
- 斜桿長度 = $L/\cos\beta$，勁度 = $AE\cos^2\beta/L$

**陷阱 3：有效長度係數**
- BD 上端 B 為銷接（固定於天花板），下端 D 為銷接（接點），故 K = 1
- $P_{cr} = \pi^2 EI / L^2$（不是 $4\pi^2 EI/L^2$）

**陷阱 4：對稱性的運用**
- 由對稱性 D 點無水平位移，AD 與 CD 受力相同，$F_{AD} = F_{CD}$

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
求 BD 桿不挫屈之最大溫升 $\Delta T_{max}$

### 本題關鍵公式（依計算順序）

$$\text{Step 1 平衡} \quad P_{BD} = -2F_d \cos\beta$$

$$\text{Step 2 斜桿伸長} \quad \delta_{AD} = \delta_D \cos\beta = F_d \cdot \frac{L/\cos\beta}{AE}$$

$$\text{Step 3 由 Step 2 得} \quad F_d = \frac{AE\cos^2\beta}{L}\,\delta_D$$

$$\text{Step 4 BD 諧和（含熱膨脹）} \quad \frac{\boxed{P_{BD}} \cdot L}{AE} + \alpha\Delta T\cdot L = \delta_D$$

$$\text{Step 5 聯立解} \quad \boxed{\delta_D} = \frac{\alpha\Delta T\cdot L}{1 + 2\cos^3\beta}$$

$$\text{Step 6 壓力} \quad \boxed{P_{BD}} = -\frac{2AE\alpha\Delta T\cos^3\beta}{1+2\cos^3\beta}$$

$$\text{Step 7 挫屈條件} \quad |P_{BD}| = P_{cr} = \frac{\pi^2 EI}{L^2}$$

| 符號 | 數值 | 說明 |
|------|------|------|
| $E$ | 共用 | 楊氏係數 |
| $I$ | 共用 | 斷面慣性矩 |
| $A$ | 共用 | 斷面積 |
| $\alpha$ | 共用 | 熱膨脹係數 |
| $\Delta T$ | 待定 | BD 桿溫升 |
| $L$ | 給定 | BD 垂直長度 |
| $\beta$ | 給定 | 斜桿與 BD 夾角 |

### L2：需知識點推導

**幾何關係**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|------|
| $L_{AD}$ | $= L/\cos\beta$ | |
| $\delta_{AD}$ | $= \delta_D\cos\beta$（D 垂直位移沿 AD 投影） | |

| 符號 | 公式／來源 | 卡關? |
|------|-----------|------|

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 4.1 結構分析與靜不定度

三桿件 AD、BD、CD 匯聚於接點 D（銷接），D 點有 2 個平衡方程，未知桿力 3 個，**靜不定度 = 1**。

利用對稱性：令 $F_d = F_{AD} = F_{CD}$（拉力為正），$P_{BD}$（壓力取負值）。

設 D 點垂直向下位移為 $\delta_D$（正值），水平位移為 0（對稱）。

### 4.2 幾何關係

$$L_{AD} = L_{CD} = \frac{L}{\cos\beta}$$

D 點下移 $\delta_D$ 時，AD（CD）的伸長量：

$$\delta_{AD} = \delta_D \cos\beta$$

### 4.3 各桿組成律（Hooke's Law）

**斜桿 AD：**（無溫差）

$$\delta_{AD} = \frac{F_d \cdot L_{AD}}{AE} = \frac{F_d \cdot L}{AE\cos\beta}$$

與幾何諧和聯立：

$$\frac{F_d \cdot L}{AE\cos\beta} = \delta_D\cos\beta$$

$$\boxed{F_d = \frac{AE\cos^2\beta}{L}\,\delta_D} \tag{1}$$

**BD 桿：**（含熱膨脹）

$$\delta_{BD,total} = \frac{P_{BD}\cdot L}{AE} + \alpha\Delta T\cdot L = \delta_D \tag{2}$$

### 4.4 D 點平衡方程

$$\sum F_y = 0:\quad P_{BD} + 2F_d\cos\beta = 0$$

$$P_{BD} = -2F_d\cos\beta \tag{3}$$

### 4.5 聯立求解 δ_D 與 P_BD

將 (1) 代入 (3)：

$$P_{BD} = -2 \cdot \frac{AE\cos^2\beta}{L}\cdot\delta_D\cdot\cos\beta = -\frac{2AE\cos^3\beta}{L}\,\delta_D$$

代入 (2)：

$$\frac{-2AE\cos^3\beta\cdot\delta_D/L\cdot L}{AE} + \alpha\Delta T\cdot L = \delta_D$$

$$-2\delta_D\cos^3\beta + \alpha\Delta T\cdot L = \delta_D$$

$$\alpha\Delta T\cdot L = \delta_D(1 + 2\cos^3\beta)$$

$$\boxed{\delta_D = \frac{\alpha\Delta T\cdot L}{1 + 2\cos^3\beta}}$$

則 BD 之壓力：

$$|P_{BD}| = \frac{2AE\cos^3\beta}{L}\cdot\frac{\alpha\Delta T\cdot L}{1+2\cos^3\beta} = \frac{2AE\alpha\Delta T\cos^3\beta}{1+2\cos^3\beta}$$

### 4.6 挫屈臨界力

BD 桿兩端均為銷接，有效長度係數 $K=1$：

$$P_{cr} = \frac{\pi^2 EI}{(KL)^2} = \frac{\pi^2 EI}{L^2}$$

### 4.7 不挫屈條件 → ΔT 最大值

$$|P_{BD}| \leq P_{cr}$$

$$\frac{2AE\alpha\Delta T\cos^3\beta}{1+2\cos^3\beta} \leq \frac{\pi^2 EI}{L^2}$$

$$\boxed{\Delta T_{max} = \frac{\pi^2 I\,(1 + 2\cos^3\beta)}{2\,A\,\alpha\,L^2\,\cos^3\beta}}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**5.1 結果的物理意義**

- 分子含 $(1+2\cos^3\beta)$：斜桿對 BD 的約束愈強（β 小 → $\cos\beta$ 大），分子愈大，容許 ΔT 反而愈大？
  - 實際上分母也含 $\cos^3\beta$，整體效應：β 小時，斜桿束制強，分子增加但分母增加更快，故容許 ΔT **減小**，即夾角愈小、斜桿愈有效抵制，BD 受到更大壓力，更容易挫屈。
  - 特例：$\beta \to 90°$（斜桿幾乎水平）：$\cos^3\beta \to 0$，分母 $\to 0$ 快於分子…需用 L'Hôpital 分析，此時斜桿幾乎無垂直束制，BD 自由膨脹，壓力趨近零，ΔT 趨近無窮大，合理。

**5.2 歐拉公式適用性**

- 題目給定桿件有 I（慣性矩），表示非純粹桁架假設（桁架一般假設桿件為二力構件，僅承軸力）。
- 此處 BD 雖為桁架桿件，但題目要求檢核挫屈，故需知道 I 值，用歐拉公式計算臨界荷重。
- 考場使用 $P_{cr} = \pi^2 EI/L^2$ 即可，不需考慮側向支撐。

**5.3 考場最安全寫法**

建議在考卷上明確標示：
1. 靜不定度判斷
2. 設多餘未知量（$\delta_D$）並說明對稱性
3. 分開寫出「熱伸長」與「機械伸長」再相加等於幾何位移
4. 最後清楚寫出 $\Delta T_{max}$ 的最終形式
