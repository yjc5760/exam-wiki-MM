# 考題編號：MM-2005-1

**主分類：** `MM-U1-3` 應力及應變分析原理與應用
**副分類：** `MM-U2-1` 軸力桿件斷面應力計算、`MM-U2-3` 扭力桿件斷面應力計算
**分析法：** 彈性分析
**標籤：** `複合應力` `圓形斷面` `軸力` `扭矩` `最大剪應力` `莫爾圓` `交互曲線` `允許應力設計`

---

## 1. 原始題目重述 (Problem Restatement)

一圓形斷面桿件（半徑為 $r$）受一扭矩 $T$ 與軸力 $P$。

**已知條件：**
- 斷面：實心圓形，半徑 $r$
- 載重：扭矩 $T$（繞軸）＋軸力 $P$（沿軸）
- 材料：線彈性、微小變形
- 允許剪應力：$\bar{\tau}$

**求：** 當桿件達到允許剪應力 $\bar{\tau}$ 時，$T$ 與 $P$ 須滿足的關係式。

> 本題無附圖，幾何與載重完全由文字描述。

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 組合應力下的最大剪應力分析（應用莫爾圓）。

考生必須能夠：
1. 正確寫出圓形斷面在扭矩與軸力下各自產生的應力分量
2. 將兩種應力疊加成平面應力狀態
3. 用最大剪應力公式導出 $T$-$P$ 的交互（interaction）關係式

**出題者意圖：**
- 測驗「應力疊加＋莫爾圓」的整合運用能力
- 最終結果應為橢圓形交互曲線方程式，格式 $(\cdot)^2 + (\cdot)^2 = \bar{\tau}^2$
- 陷阱：軸力只產生正應力（非剪應力），扭矩只產生剪應力，兩者不在同一分量上「相加」

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 計算軸力 $P$ 產生的正應力 $\sigma$（A = πr²）
2. 計算扭矩 $T$ 在外緣（最危險點）產生的剪應力 $\tau_T$（J = πr⁴/2）
3. 建立平面應力元素：$\sigma_x = \sigma$，$\sigma_y = 0$，$\tau_{xy} = \tau_T$
4. 代入最大剪應力公式，令 $\tau_{max} = \bar{\tau}$
5. 整理得 $T$-$P$ 關係式

**關鍵陷阱：**

| # | 陷阱 | 應對 |
|---|------|------|
| ① | 誤以為 $P$ 也會產生剪應力而直接相加 | $P$ 只產生正應力 $\sigma_x$；剪應力僅來自扭矩 |
| ② | 忘記最外緣才是扭矩剪應力最大處（$\rho = r$） | 代入 $\rho = r$ 計算 $\tau_T = Tr/J$ |
| ③ | 遺漏係數：J 圓形 = $\pi r^4/2$（非 $\pi r^4/4$） | $J = \pi r^4/2$；$I = \pi r^4/4$ 是面積慣性矩，兩者不同 |
| ④ | 直接將 $\sigma$ 與 $\tau_T$ 相加 | 需透過莫爾圓公式 $\tau_{max} = \sqrt{(\sigma/2)^2 + \tau_T^2}$ |

---

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
導出允許剪應力條件下 $T$ 與 $P$ 的交互關係方程式。

### 本題關鍵公式（依計算順序）

$$\text{Step 1: } \sigma = \frac{P}{A} = \frac{P}{\pi r^2}$$

$$\text{Step 2: } \tau_T = \frac{T \cdot r}{J} = \frac{2T}{\pi r^3} \quad (J = \tfrac{\pi r^4}{2})$$

$$\text{Step 3（平面應力）: } \sigma_x = \boxed{\sigma},\; \sigma_y = 0,\; \tau_{xy} = \boxed{\tau_T}$$

$$\text{Step 4: } \tau_{max} = \sqrt{\left(\frac{\sigma_x}{2}\right)^2 + \tau_{xy}^2} = \sqrt{\left(\frac{\boxed{\sigma}}{2}\right)^2 + \boxed{\tau_T}^2}$$

$$\text{Step 5（令 } \tau_{max} = \bar{\tau}\text{）: } \left(\frac{P}{2\pi r^2}\right)^2 + \left(\frac{2T}{\pi r^3}\right)^2 = \bar{\tau}^2$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $r$ | — | 圓形斷面半徑 |
| $T$ | — | 施加扭矩 |
| $P$ | — | 施加軸力 |
| $\bar{\tau}$ | — | 允許剪應力 |

### L2：需知識點推導

**斷面幾何性質**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $A$ | $\pi r^2$（圓形面積） | |
| $J$ | $\pi r^4 / 2$（實心圓極慣性矩） | |

**應力分量計算**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $\sigma$ | $P/A = P/(\pi r^2)$（軸力正應力） | |
| $\tau_T$ | $Tr/J = 2T/(\pi r^3)$（扭矩剪應力，外緣最大） | |

**莫爾圓求最大剪應力**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|-------|
| $\tau_{max}$ | $\sqrt{(\sigma_x/2)^2 + \tau_{xy}^2}$（平面應力最大剪應力） | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| 平面應力元素建立 | 軸力→$\sigma_x$，扭矩→$\tau_{xy}$，不能直接相加 | |
| 莫爾圓中心與半徑 | 中心 $= (\sigma_x+\sigma_y)/2$；半徑 $= \tau_{max}$ | |
| $I$ vs $J$ 的區別 | $I = \pi r^4/4$（彎曲用）；$J = \pi r^4/2$（扭轉用），$J = 2I$ | |
| 交互關係的幾何意義 | 結果為 $T$-$P$ 平面上的橢圓，表示設計承載邊界 | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：斷面性質

$$A = \pi r^2$$

$$J = \frac{\pi r^4}{2}$$

### Step 2：各載重在外緣產生的應力分量

**軸力 $P$ → 均勻正應力（全斷面）：**

$$\sigma = \frac{P}{A} = \frac{P}{\pi r^2}$$

**扭矩 $T$ → 在外緣（$\rho = r$）產生最大剪應力：**

$$\tau_T = \frac{T \cdot r}{J} = \frac{T \cdot r}{\pi r^4/2} = \frac{2T}{\pi r^3}$$

### Step 3：建立平面應力狀態

取外緣危險點，建立應力元素：

$$\sigma_x = \frac{P}{\pi r^2}, \quad \sigma_y = 0, \quad \tau_{xy} = \frac{2T}{\pi r^3}$$

### Step 4：最大剪應力（莫爾圓）

$$\tau_{max} = \sqrt{\left(\frac{\sigma_x - \sigma_y}{2}\right)^2 + \tau_{xy}^2} = \sqrt{\left(\frac{P}{2\pi r^2}\right)^2 + \left(\frac{2T}{\pi r^3}\right)^2}$$

### Step 5：令 $\tau_{max} = \bar{\tau}$，整理交互關係

$$\boxed{\left(\frac{P}{2\pi r^2}\right)^2 + \left(\frac{2T}{\pi r^3}\right)^2 = \bar{\tau}^2}$$

此即 $T$ 與 $P$ 須滿足的關係式（橢圓形交互曲線）。

**展開形式：**

$$\frac{P^2}{4\pi^2 r^4} + \frac{4T^2}{\pi^2 r^6} = \bar{\tau}^2$$

$$\frac{P^2 r^2 + 16T^2}{4\pi^2 r^6} = \bar{\tau}^2$$

$$\boxed{P^2 r^2 + 16T^2 = 4\pi^2 r^6 \bar{\tau}^2}$$

---

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**絕對最大剪應力 vs 平面內最大剪應力：**

本題使用的是平面應力（$\sigma_z = 0$）下的 in-plane $\tau_{max}$：

$$\tau_{max,\text{in-plane}} = \sqrt{\left(\frac{\sigma_x}{2}\right)^2 + \tau_{xy}^2}$$

嚴格而言，第三主應力 $\sigma_3 = 0$，絕對最大剪應力需比較三個主應力對：

$$\tau_{abs,max} = \max\left(\frac{|\sigma_1 - \sigma_2|}{2}, \frac{|\sigma_1|}{2}, \frac{|\sigma_2|}{2}\right)$$

若 $\sigma_1 > 0 > \sigma_2$（即正應力與剪應力均不可忽略時），$\tau_{abs,max} = |\sigma_1 - \sigma_2|/2 = \tau_{max,\text{in-plane}}$，兩者一致，本題答案不受影響。

若 $P = 0$（純扭矩），$\sigma_1 = -\sigma_2 = \tau_T$，主應力正好對稱，同樣一致。

**實務意義：** 此橢圓交互關係是機械設計中「複合載重允許應力設計」的基礎形式，與 Tresca（最大剪應力）準則直接對應。
