**執行規則：**
```
【階段一：建立資料夾並等待截圖（觸發語句後立即執行）】

2. 讀取 raw/json/question_index.json，找出尚無解析的題目

   → 解完一題，存檔後停止

   → 直接存入 raw/solutions/MM-XXXX-N/MM-XXXX-N.md
9. 所有數學公式使用 LaTeX 格式
10. 每份解析開頭必須包含標籤區塊
11. 判斷是否需要輸出互動圖（依「圖形觸發條件」）：
    → 直接存入 raw/solutions/MM-XXXX-N/MM-XXXX-N-[內容碼]-viz.html
```

#### 解題前置：三層掃描法（每題解題前先執行）

**核心原則：** 先見林（外在全貌）→ 再見樹（策略路徑）→ 後見土（內在意圖）

**第一層：外在掃描（2分鐘內完成）**
```
[目標] 這題最終要我交出什麼？（最大應力？撓度？臨界載重？主應力？塑性矩？）
[已知] 題目給了哪些數字和條件？（截面尺寸、E/G 值、載重、幾何、邊界條件）
[待算] 從已知到目標，需要算出哪些中間參數？（I、Q、c、V、M、Pe...）
```

**第二層：策略掃描**
```
[邏輯] 適用哪個分析邏輯？（見下方「解析哲學框架」）
[陷阱] 考官可能在哪裡挖了坑？（形心偏移、最大剪應力位置、邊界條件選擇）
[捷徑] 有沒有條件可以跳過某些步驟？（對稱截面→形心在幾何中心、已知彎矩圖→跳過反力）
```

**第三層：內在掃描**
```
[核心] 這題最重要的一個觀念是什麼？（彈性範圍內的線性行為、塑性鉸機構）
[線索] 特殊用字給了什麼暗示？（「完全彈性」→線彈性分析；「剛體」→只算支承反力）
```

#### 解析哲學框架（解題邏輯對照）

**斷面性質計算（MM-U1-1）**
```
核心：所有後續應力/變位計算的基礎，錯一步全盤錯
步驟：① 分割截面為簡單幾何形狀（矩形/圓形/三角形）
      ② 形心：ȳ = ΣAᵢyᵢ / ΣAᵢ（以某固定基準計算）
      ③ 慣性矩（平行軸定理）：I = Σ(Iᵢ₀ + Aᵢdᵢ²)
      ④ 極慣性矩：J = Iᵧ + I_z（對圓形截面：J = πd⁴/32）
      ⑤ 斷面模數：S = I / c_max（c = 形心到最外緣距離）
關鍵：T 形截面形心偏向面積較大側，c_bot ≠ c_top
```

**虎克定律與廣義虎克定律（MM-U1-2）**
```
核心：應力-應變線性關係，是彈性分析的基礎假設
單軸：σ = Eε；τ = Gγ；ε_lateral = -νε_axial
廣義（平面應力）：
  εₓ = (σₓ - νσᵧ) / E
  εᵧ = (σᵧ - νσₓ) / E
  γₓᵧ = τₓᵧ / G
材料常數關係：G = E / (2(1+ν))；K = E / (3(1-2ν))
```

**Mohr's Circle 應力轉換（MM-U1-3）**
```
核心：任意斜截面的正向應力與剪應力，由主應力決定上下界
步驟：① 確認 σₓ, σᵧ, τₓᵧ（繪製基礎狀態）
      ② 圓心：C = (σₓ + σᵧ) / 2
      ③ 半徑：R = √[((σₓ - σᵧ)/2)² + τₓᵧ²]
      ④ 主應力：σ₁,₂ = C ± R
      ⑤ 最大剪應力：τ_max = R
      ⑥ 主應力方向：tan(2θp) = -τₓᵧ / ((σₓ - σᵧ)/2)
陷阱：τ 的方向定義（Mohr's circle 上正 τ 向下）；θ 在圖上是 2θ
```

**軸力桿件應力與變形（MM-U2-1 / MM-U3-1）**
```
核心：軸向應力均勻分布，變形可疊加
應力：σ = P / A
變形：δ = PL / (AE) 或 δ = ∫P dx / (AE)（變截面）
不靜定：相容性方程式 → δ_total = 0 或指定值
熱應力：δ_thermal = αΔTL；與機械變形相容
```

**梁彎曲應力（MM-U2-2）**
```
核心：純彎梁的正向應力線性分布，中性軸 σ = 0
彎曲正向應力：σ = M·y / I（y = 距中性軸距離，與 M 同號規定）
最大正向應力：σ_max = M·c / I（c = 形心到最外緣距離）
複合梁（不同 E 材料）：轉換截面法，以基準材料等效寬度處理
梁剪應力（Jourawski）：τ = V·Q / (I·b)
  Q = ∫y dA（剪力截面以外部分的面積一次矩）
  最大剪應力：矩形截面在形心軸處 τ_max = 1.5 V/A
             圓形截面 τ_max = 4V/(3A)
             T 形截面在形心軸（腹板）處
```

**扭力桿件（MM-U2-3 / MM-U3-3）**
```
核心：圓截面扭矩引起的剪應力與扭轉角
實心圓截面：
  τ_max = T·c / J（c = 半徑；J = πd⁴/32）
  τ_r = T·r / J（r = 任意半徑處）
  φ = TL / (GJ)（扭轉角，弧度）
空心圓截面：J = π(d_o⁴ - d_i⁴) / 32
薄壁封閉截面（剪力流）：
  q = T / (2A_m)（A_m = 中線包圍面積）
  τ = q / t（t = 薄壁厚度）
  φ = TL / (4GA_m²) · ∮(ds/t)
```

**梁撓度（MM-U3-2）**
```
核心：EI·v'' = M(x)，從彎矩圖積兩次得撓度
方法一：雙積分法
  EI·v'' = M(x) → 積分 → EI·v' = ∫M dx + C₁ → 積分 → EI·v = ∬M dx + C₁x + C₂
  邊界條件：固定端 v=0, v'=0；簡支端 v=0
方法二：疊加法（利用已知標準撓度公式）
  均布載重簡支梁：δ_max = 5wL⁴/(384EI)（跨中）
  集中載重簡支梁：δ_max = PL³/(48EI)（跨中）
  懸臂梁端點：δ = PL³/(3EI)（集中）或 wL⁴/(8EI)（均布）
方法三：Mohr's 積分法（彎矩面積法）
  δ_B = ∫M·m̄ dx / EI（m̄ = 單位荷重下虛彎矩圖）
靜不定梁：需額外列相容方程式（釋放多餘拘束，以已知撓度還原）
```

**柱挫屈（MM-U3-4）**
```
核心：Euler 臨界載重 → 壓力超過此值柱件突然側移（失穩）
臨界載重：Pₑ = π²EI / (KL)²
  K = 有效長度係數：
    兩端鉸支：K = 1.0
    兩端固定：K = 0.5
    一端固定一端鉸支：K = 0.7
    一端固定一端自由：K = 2.0
臨界應力：σₑ = Pₑ / A = π²E / (KL/r)²（r = 迴轉半徑 = √(I/A)）
細長比：KL/r（越大越容易挫屈）
陷阱：I 取最小值（決定最弱軸）；K 依邊界條件決定
```

**塑性分析（MM-U4）**
```
核心：超過降伏後鋼材呈完全彈塑性，塑性鉸承受 Mp（不可再增加彎矩）
塑性矩：Mp = σᵧ · Z（Z = 塑性斷面模數）
  矩形截面：Z = bh²/4（= 1.5 × S_elastic = 1.5 × bh²/6）
  形狀因子：f = Mp/My = Z/S（矩形 = 1.5，圓形 ≈ 1.7）
極限載重（機構法）：
  1. 找出可能的塑性鉸位置
  2. 假設破壞機構（保證靜定化且不穩定）
  3. 虛功原理：P_u × δ = Mp × θ（對所有塑性鉸）
殘留應力（MM-U4-2）：
  σ_residual = σ_plastic - σ_elastic_springback
  加載到 σ > σᵧ 後卸載，彈性回彈量 = σ_load/E × 全長
  殘留應力 = 完全塑化應力 - 彈性回彈應力
```

### 【QUERY】查詢知識庫並存回（在 Cowork 執行）

**觸發方式：** 直接向 Cowork 提問，問完後說「把這個答案存進知識庫」

**適用情境：**
```
跨題目分析：「T形截面在歷年考題中如何演變？」
弱點整理：「撓度計算有哪些常見陷阱？」
比較分析：「積分法與疊加法計算撓度的核心差異？」
趨勢判斷：「塑性分析最近三年考什麼？今年可能考什麼？」
考前衝刺：「哪些三星陷阱出現頻率最高？」
```

**執行流程：**
```
1. Cowork 讀取 wiki/index.md 定位相關分類
2. Cowork 讀取相關 concepts/、problems/、traps/、methods/ 頁面
3. Cowork 合成跨頁面的分析答案
4. 你說「把這個答案存進知識庫」
5. Cowork 存成 wiki/queries/[主題]-[日期].md
```
